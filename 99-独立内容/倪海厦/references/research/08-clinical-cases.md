# 08-clinical-cases.md - 倪海厦经典临床思路

## 临床用药核心理念

### 1. 阳气论（临床应用）
**倪师核心观点**：阳气是生命的根本，阳化气，阴成形

**临床意义**：
- 阳气充足 → 身体健康，不易生病
- 阳气不足 → 各种症状：手脚冰凉、容易疲劳、免疫力低下
- 养生就是养阳气

**倪师临床用药原则**：
- 阳虚先扶阳（姜、附子、桂枝）
- 阴虚配合滋阴（麦冬、沙参等）
- 但以温阳为主，滋阴为辅

### 2. 六经辨证临床路径

**太阳病（表证）**：
- 主症：脉浮、头项强痛、恶寒
- 有汗 → 桂枝汤（桂枝、白芍、生姜、大枣、甘草）
- 无汗 → 麻黄汤（麻黄、桂枝、杏仁、甘草）
- 项强 → 葛根汤（桂枝汤+葛根）

**阳明病（里热）**：
- 身大热、大汗出、大渴、脉洪大 → 白虎汤
- 便秘、胃家实 → 承气汤（根据燥屎程度选择调胃/小/大承气）

**少阳病（半表半里）**：
- 口苦、咽干、目眩、往来寒热 → 小柴胡汤

**太阴病（脾虚寒湿）**：
- 腹满、吐利、食不下 → 理中汤

**少阴病（心肾阳虚）**：
- 脉微细、但欲寐、四肢厥冷 → 四逆汤

**厥阴病（寒热错杂）**：
- 上热下寒、消渴、气上撞心 → 乌梅丸

### 3. 常用方剂应用要点

**桂枝汤（群方之首）**：
- 调和营卫，解肌发表
- 适用：外感风寒表虚证（有汗、脉浮缓）
- 禁忌：表实无汗不能用

**小柴胡汤**：
- 和解少阳
- 适用：往来寒热、胸胁苦满、心烦喜呕
- 关键药引：人参（扶正）+ 柴胡（解表）

**四逆汤**：
- 回阳救逆
- 适用：少阴病，四肢厥冷，脉微欲绝
- 关键：附子必须用足量

### 4. 倪师临床经验总结

**倪师看诊时间要求**：
- 普通病：15-20分钟
- 重症：30分钟以上
- 理由：必须充分望闻问切，不能草率

**倪师常用问诊十问歌**（简化版）：
1. 睡眠如何？（入睡/易醒/梦多）
2. 饮食如何？（食欲、口味偏好）
3. 大便如何？（便秘/便溏/正常）
4. 小便如何？（颜色、次数）
5. 手脚温度？（冷/热/正常）
6. 出汗情况？（自汗/盗汗/无汗）
7. 口渴情况？（喜冷/喜热/不渴）
8. 面色如何？（红润/苍白/潮红）
9. 精神状态？（精力充沛/容易疲劳）
10. 疼痛部位？（哪里痛？什么性质？）

**倪师特别强调的体征**：
- 手脚是否冰凉（判断阳气）
- 舌苔颜色（判断寒热）
- 脉象（配合症状，不是独立诊断）
- 面色（实证vs虚证的第一眼判断）

## 二、表达DNA

### 核心口头禅

| 口头禅 | 场景 | 频率 |
|--------|------|------|
| 嚎 | 句首强调 | 极高 |
| 对不对 | 反问确认 | 极高 |
| 很简单嘛 | 表达自信 | 高 |
| 诶 | 惊讶/强调 | 高 |

### 典型句式

**断言+反问**：
> "先辨六经啊！发烧、怕冷、项强 → 太阳病，桂枝汤。对不对？很简单嘛。"

**批评+立论**：
> "那你找的那些中医都是温病派嘛！用经方啊，伤寒论里都有方子。"

### 语气特征
- 自信直接，不绕弯子
- 批评时情绪激动
- 略带台湾口音（"嚎"的使用）

## 三、教学方法论

### 核心教学法

1. **经典原文背诵法**
   - 先背诵，后理解
   - 理解在临床中自然发生
   - 不主张先看后世注解

2. **六经辨证入门**
   - 先辨三阴三阳（哪一经）
   - 再辨表里、虚实、寒热
   - 最后选方

3. **四诊优先顺序**
   - 望诊（第一眼信息量最大）
   - 问诊（出汗、大小便、手脚温度）
   - 脉诊（配合症状，不是独立诊断）
   - 闻诊（声音、气味）

### 纠错式教学

**常见错误**：
- 只看脉诊 → 要先问症状
- 温病派思路 → 发烧要分表里
- 过度依赖药物 → 生活习惯要改

### 辨证流程

```
望诊 → 问诊 → 脉诊 → 六经定位 → 虚实判断 → 选方 → 加减
```

## 四、临床思维

### 六经辨证速查

| 六经 | 主症 | 主方 |
|------|------|------|
| 太阳 | 脉浮、头项强痛、恶寒 | 桂枝汤/麻黄汤 |
| 阳明 | 胃家实 | 白虎汤/承气汤 |
| 少阳 | 口苦咽干目眩 | 小柴胡汤 |
| 太阴 | 腹满吐利 | 理中汤 |
| 少阴 | 脉微细、但欲寐 | 四逆汤 |
| 厥阴 | 寒热错杂 | 乌梅丸 |

### 十问歌（简化版）

1. 睡眠如何？
2. 饮食如何？
3. 大便如何？
4. 小便如何？
5. 手脚温度？
6. 出汗情况？
7. 口渴情况？
8. 面色如何？
9. 精神状态？
10. 疼痛部位？

### 常见病套路

**感冒发烧**：
- 怕冷 → 表证（桂枝汤/麻黄汤）
- 不怕冷 → 里证（白虎汤等）
- 有汗 → 桂枝汤
- 无汗 → 麻黄汤

**失眠**：
- 难以入睡 → 实证（血热/胃热）
- 易醒 → 虚证（血虚/阴虚）
- 梦多 → 肝血不足

## 五、决策风格

### 人生决策
1. 音乐 → 中医（选择终身事业）
2. 台湾 → 美国（移民行医）
3. 录制教学视频（系统传承）

### 临床决策
- 先辨六经，再选经方
- 不看健保，保证看诊质量
- 15-20分钟看诊，重症半小时

### 学术立场
- 明确站队经方派
- 批评温病派和西医
- 坚持师徒制传承

## 六、外界评价

### 正面评价
- 系统整理伤寒论、金匮要略
- 推广经方派中医
- 影响众多学习者

### 争议点
1. 过于强调经方
2. 阳气论的偏向
3. 对温病派和西医的批评

### 客观建议
- 倪海厦观点代表经方派立场，非中医界共识
- 临床应用需结合具体情况
- 不可教条化

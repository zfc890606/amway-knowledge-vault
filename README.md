## 📖 简介 | Introduction

这是一个**真实运转中的知识库**，不是教学 demo。

它服务于一个明确的商业目标——安利（Amway）创业，聚焦两大核心价值：

- 🥦 **营养专业** — 用知识吸引顾客
- 🤖 **AI × 互联网打法** — 用系统吸引合作伙伴

知识库围绕三条流水线运行：**知识加工 → 短视频生产 → 项目管理**，所有内容经「思维操作系统」加工后，自动归入四大板块，最终产出短视频脚本、公众号文章、课件等成品。

> **核心理念**：知识不是用来囤的，是用来「运行」的。

### 4 大板块归类规则

| 板块 | 范围 | 卡片前缀 |
|------|------|---------|
| ① 营养解决方案 | 健康、营养、产品 | `KC-营养-*` |
| ② 创业认知 | 创业、安利、思维模型 | `KC-创业-*` |

所有新知识进入后自动按上述规则归类，形成「入盒 → 加工 → 归类 → 输出」的完整闭环。

## 🚀 快速开始 | Getting Started

### 前提

- 安装 [Obsidian](https://obsidian.md/)（免费）

### 使用步骤

1. **克隆仓库**
   ```bash
   git clone https://github.com/zfc890606/amway-knowledge-vault.git
   ```
2. **用 Obsidian 打开**
   - 启动 Obsidian → "Open folder as vault" → 选择克隆下来的文件夹
   - **建议开启设置中的「严格换行」模式**
3. **（可选）启用社区插件**
   - 本 vault 使用了 Obsidian 核心插件（无需额外安装）
   - 推荐搭配插件：Dataview、Calendar、Templater
4. **开始探索**
   - 从 `00-灵感库（标记灵感）/index.md` 开始了解入盒规则
   - 浏览 `02-Areas（关注的领域）/思维操作系统.md` 了解核心方法论
   - 查看 `知识卡片/` 中的 4K 卡片示例

> **💡 提示**：本知识库是**真实工作系统**，不是演示品。你可以直接 Fork 作为自己的起点，替换其中的板块内容。

## 🧩 4K 知识卡片模板

每条卡片的固定结构：

| 模块 | 作用 |
|------|------|
| **一句话本质** | AI 读完就知道什么时候该用 |
| **🧩 因果/原理** | 解决什么问题？为什么有效？ |
| **🛠 可执行方法** | 最小行动 + 完整执行步骤 |
| **📍 场景与边界** | 适用场景 + 不适用条件 |
| **🔗 关联** | 来源 + 相关卡片 + 对应问题 |

> 模板文件详见 `00-灵感库（标记灵感）/知识卡片/卡片模板.md`

## 🎯 核心流程 v2.0

### 总览：一条主脑 + 三条流水线

```
          ┌──────────────────────────────┐
          │   思维操作系统（大脑加工厂）     │
          │  算子① 捕获 → ② 拆解 →        │
          │  ③ 结构化 → ④ 系统化 → ⑤ 闭环 │
          └──────────────┬───────────────┘
                         ▲
         所有输入都经过5算子加工
                         │
    任何新知识/观点/素材进来
                         │
                         ▼
     ┌──────────────────────────────────┐
     │  灵感入盒（[[00-灵感库]]）          │
     │  标记→用5个问题拷打：拆什么？        │
     │  放哪个系统？跟谁打架？行动是啥？    │
     │ ⭐五环以外？                       │
     └────────────────┬─────────────────┘
                      │
                      ▼
                ╔══════════════╗
                ║  分流判断   ║  ← 新
                ╚══════════════╝
                      │
      ┌───────────────┼───────────────┐
      ▼               ▼               ▼
 ┌──────────┐  ┌──────────┐  ┌──────────┐
 │ 流水线A  │  │ 流水线B  │  │ 流水线C  │
 │ 知识加工 │  │短视频生产 │  │ 项目管理 │
 └──────────┘  └──────────┘  └──────────┘
```

---

### 流水线A：知识加工（一般认知/方法/观点）

```
灵感入盒
   │
   ▼
┌─ 7刃筛选（炼金炉）───────────────────────────┐
│ 7个问题拷打知识点，≥5/7通过 → 进挖矿         │
│ ＜5/7 → 暂存，定期复盘                       │
└─────────────────────┬───────────────────────┘
                      ▼
┌─ 挖矿8条 ────────────────────────────────────┐
│ 溯源→解构→类比→数据→案例→反方→迁移→行动     │
└─────────────────────┬───────────────────────┘
                      ▼
┌─ 成卡（新模板）──────────────────────────────┐
│ 模板：一句话本质 → 因果/原理 → 可执行方法     │
│      → 场景边界 → 关联                       │
│ 存到：知识卡片/KC-*.md                        │
└─────────────────────┬───────────────────────┘
                      ▼
┌─ 自动归类4大板块 + 并行反哺 ──────────────────┐
│ 归类 → 关联痛点榜 → 提取金句 → 更新系统      │
└─────────────────────┬───────────────────────┘
                      ▼
┌─ 积累充分 → 06-输出/对应板块/ ───────────────┐
│ 口播稿 / 长文 / 短视频 / PPT                  │
└──────────────────────────────────────────────┘
```

### 流水线B：短视频生产（选题/痛点/脚本）

```
调研 → 稀缺检查 → 情绪设计 → 嘴替写稿
  → 反差人设检查 → 开头钩子 → 真实感写作
  → 风格自检7道关 → 去AI味 → 风险检测
  → AI痕迹复核 → 交付纯文案
```

> 完整流程详见 [[05-Skills（技能sop）/短视频方法库]]

### 流水线C：项目管理（任务/待办）

```
提炼出"需要做的事"
  → 判断属于哪个板块
  → 01-Projects/待办/板块①-④/
  → 执行 → 完成 → 归档或输出
```

## 🧩 卡片模板（新4K）

当前卡片模板已升级，结构如下：

| 模块 | 内容 |
|------|------|
| **一句话本质** | AI读完就知道什么时候该用 |
| **🧩 因果/原理** | 解决什么问题？为什么有效？ |
| **🛠 可执行方法** | 最小行动 + 完整执行步骤 |
| **📍 场景与边界** | 适用场景 + 不适用条件 + 失败原因 |
| **🔗 关联** | 来源 + 相关卡片 + 对应问题 |

> 详细模板 → [[知识库/00-灵感库（标记灵感）/知识卡片/卡片模板]]

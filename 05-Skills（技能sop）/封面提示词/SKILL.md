# 阿囤囤小红书封面 Skill

读取用户已提供的内容 -> 逐步让用户做选择题 -> 收集参考图/标题/素材 -> 按对应风格模板生成可直接用于图片生成模型的封面提示词。

**重要：一次只问一个问题，等用户回答后再问下一个。**

如果用户已经在当前消息提供了某项信息，直接记录并跳过对应问题。最终只套用用户选择的风格模板；不要输出通用模板。

除用户粘贴文章、脚本、选题或上传图片外，所有封面决策问题都必须做成编号选择题。不要用开放式问题收集风格、人物、素材、背景、字体、颜色或标题偏好。

---

## 问题流程（逐一问，不要一次全问）

**第1问：选风格**

列出以下8种风格，让用户选一种：

1. **爆款大字压顶风** — 顶部超大标题，人物居中，适合强钩子标题
2. **巨字拆分冲击风** — 2-4个核心大词拆开放满画面，适合概念/身份/能力类主题
3. **小白科普问答风** — 黑灰科技海报、模糊界面或真实室内背景，问号和解释感，适合"什么是/怎么理解"
4. **教程清单风** — 绿色勾选清单+步骤感，适合安装、部署、上手教程
5. **产品测评榜单风** — 大数字、箭头、维度标签，适合横评、榜单、测评
6. **种草推荐风** — 人物点赞或推荐姿势，emoji和温暖背景，适合资源/播客/工具推荐
7. **贴纸拼贴入门风** — 人物+小卡片+产品图标+贴纸，适合"入门/搭建/组合"
8. **黑底效率工作流风** — 深色背景+Logo墙+清单，适合工作法、效率流、工具组合

---

**第2问：图1**

要求用户上传人物参考图（图1），不要提供"默认形象"或"通用博主形象"选项。

只问：

请上传图1人物参考图。可以上传1张清晰人脸照，也可以上传2-3张同一人物照片用于保持人物一致性。

如果用户明确说不露脸，也必须要求上传局部出镜参考图，例如手部、半脸边缘或侧脸轮廓。不要凭空生成默认人物。

用户上传图片后，只记录为图1人物参考，不额外追问人物外貌。

---

**第3问：人物表情和动作**

给用户以下选项：

1. **张嘴震惊** — 嘴巴张开，眼睛睁大，适合测评/反差/发现
2. **双手点赞** — 双手竖大拇指，适合推荐/种草/公开私藏
3. **指向标题** — 一只手指向大字或清单，适合教程/认知
4. **托腮疑惑** — 单手托腮，表情认真疑问，适合科普/问题钩子
5. **举拳自信** — 单手举拳或握拳，适合入门/搭建/行动感
6. **双手打开讲解** — 双手在脸旁或胸前展开，适合分享方法
7. **交给模型决定**

---

**第4问：额外素材**

问用户有没有图2、图3等额外素材，只给以下选项：

1. **有1张素材图** — 图2是软件截图、产品图、Logo或工具界面
2. **有2张素材图** — 图2是主素材，图3是辅助素材
3. **有3张及以上素材图** — 适合贴纸拼贴、产品测评、工具墙
4. **没有素材图** — 模型根据主题生成贴纸、图标、清单、箭头和卡片

用户选择1-3时，只要求按顺序上传图2、图3等，不额外追问开放描述。若必须说明素材用途，给选择题：

1. **产品/工具主体**
2. **教程结果证明**
3. **背景氛围素材**
4. **贴纸拼贴元素**

---

**第5问：背景色调**

给用户以下选项：

1. **室内暖光背景** — 像真实房间、书架、桌面，适合推荐/经验分享
2. **黑灰科技海报背景** — 黑/深灰网格、数字标注、科技海报感，适合AI/工具/概念科普
3. **深色工具墙背景** — 深灰/黑底+软件Logo墙，适合效率流/工具组合
4. **模糊软件界面背景** — 让产品截图或工具界面成为氛围
5. **高饱和撞色背景** — 黄/蓝/紫/绿强对比，适合爆款强钩子
6. **交给模型决定**

---

**第6问：字体风格**

给用户以下选项：

1. **综艺超粗黑体** — 默认首选，最像高点击小红书封面
2. **漫画爆款标题字** — 更夸张，适合震惊/测评/反差
3. **圆润粗黑体** — 更亲切，适合种草推荐/播客分享
4. **科技感粗黑体** — 更硬朗，适合AI工具/效率工作流
5. **粗标题+手写涂鸦小字** — 主标题很粗，旁边加少量涂鸦标注
6. **交给模型决定**

---

**第7问：字体颜色效果**

给用户以下选项：

1. **柔和浅黄色 #FDFFA7 填充+粗黑描边** — 默认首选，最强点击感
2. **纯白填充+粗黑描边** — 深色背景更清晰
3. **#FDFFA7/白色混排+粗黑描边** — 关键词用 #FDFFA7，辅助词用白色
4. **#FDFFA7+白色高光+粗黑描边** — 更像短视频爆款标题
5. **深底白字+#FDFFA7关键词** — 适合黑底效率工作流和测评榜单
6. **交给模型决定**

---

**第8问：封面标题**

根据用户已提供的内容提炼3个候选封面标题，让用户只选编号：

1. **候选标题A**
2. **候选标题B**
3. **候选标题C**
4. **重新生成3个标题**

不要默认问"是否修改标题"这种开放问题。如果用户主动给出标题，直接使用用户标题。

标题规则：
- 主标题优先4-10个字
- 可以有副标题，但主标题必须最醒目
- 优先使用口语化强钩子，如"别瞎选了"、"从0搭建"、"全流程公开"、"小白上手"、"到底行不行"
- 测评类必须突出数字，如"5款产品"、"7个维度"
- 教程类必须突出低门槛，如"零基础"、"不用命令行"、"纯小白"

## 通用视觉规范

所有风格都遵守：

- 小红书封面构图必须是3:4竖版比例，不接受其他比例或"接近3:4"
- 真人出镜，人物占画面35%-55%，除非风格另有说明
- 人物边缘有浅黄色（优先 #FDFFA7）、白色或高对比亮色描边，形成明显抠图感
- 主标题是视觉第一优先级，必须超大、清晰、短促
- 主标题优先使用柔和浅黄色 #FDFFA7 或纯白，必须有粗黑描边
- 标题填充、标题关键词、标题背后的底板/色块/横幅只要使用黄色，统一指定为柔和浅黄色 #FDFFA7；不要使用荧光黄、高饱和亮黄、金黄或偏橙黄色
- 字体使用综艺感超粗黑体、漫画标题字、短视频封面大字
- 常用元素：绿色勾选框、柔和浅黄色 #FDFFA7 箭头、问号、感叹号、emoji、贴纸、小卡片、软件Logo
- 背景可以是室内场景、软件界面、模糊产品图、深色Logo墙、黑灰科技海报或参考素材延展背景，但不能抢标题
- 画面高对比、高密度、强情绪，有"一眼能读懂"的点击感
- 所有重要文字、人物脸部、清单都不超出安全区

避免：

- 不要极简留白
- 不要细字体
- 不要冷淡杂志风
- 不要低对比浅灰字
- 不要全息投影、数据流、粒子特效
- 不要让标题遮挡人物眼睛和嘴巴
- 不要用荧光黄、高饱和柠檬黄、金黄或偏橙黄色做标题、关键词或标题底板

## 生成提示词的核心要求

拿到所有答案后，按对应风格模板生成提示词。提示词必须做到：

- **人物姿势具体**：说清楚人物在画面哪里、手指向哪里、表情是什么
- **标题层级清楚**：主标题、副标题、清单文字分别写清楚
- **黄色色号固定**：标题、关键词、标题底板、标题色块、横幅、箭头凡是用黄色，都明确写 #FDFFA7
- **空间关系清晰**：前景/中景/背景各是什么，哪个遮挡哪个
- **素材指代明确**：直接写"参考图1/图2/图3"，不要复述参考图全部细节
- **风格差异明显**：每种风格有自己的构图、背景、文字位置和装饰逻辑
- **比例写死**：每个最终提示词都必须明确写"3:4竖版小红书封面"

## 风格一：爆款大字压顶风

顶部超大标题压住画面，人物居中或中下方，适合强钩子标题、经验公开、教程入口。

### 视觉特征
- 标题：顶部占35%-45%，柔和浅黄色 #FDFFA7 或白色，粗黑描边
- 人物：中下方居中，占45%左右，表情强烈
- 背景：室内或主题相关场景，轻微虚化
- 点缀：少量箭头、emoji、贴纸，强化情绪

### 提示词模板

```
参考图1的人物五官特征，保持人物一致性，真人半身出镜。

[具体表情]，人物居中偏下，占画面约45%，[具体手势和身体姿态]，眼神看向镜头，人物边缘有浅黄色 #FDFFA7 描边，形成清晰抠图感。

画面顶部放置超大中文标题"[封面标题]"，字体为综艺感超粗黑体，柔和浅黄色 #FDFFA7 填充，粗黑描边，轻微白色高光，文字占据画面上方约40%，有强烈压迫感；如果标题背后需要底板或色块，也使用 #FDFFA7，避免荧光黄或金黄。

如果有副标题，将"[副标题]"放在画面底部，白色粗体，黑色描边，字号小于主标题但仍然醒目。

背景是[背景描述]，轻微虚化，降低细节，让人物和标题成为绝对主体。

画面两侧加入3-5个[主题相关贴纸/emoji/箭头/问号]，只做情绪强化，不抢主标题。

3:4竖版小红书封面，高对比，高密度，强点击感，标题清晰可读，人物脸部不被文字遮挡，所有重要元素不超出安全区。
```

## 风格二：巨字拆分冲击风

把标题拆成2-4个巨大词块，人物被文字包围或夹在中间，适合"产品经理软技能"、"录音工作大法"这类核心概念。

### 视觉特征
- 标题：拆成多个词块，几乎铺满画面
- 人物：夹在字块之间，动作夸张
- 背景：深色街景、室内、模糊工作场景
- 整体：像短视频封面，强冲击、强节奏

### 提示词模板

```
参考图1的人物五官特征，保持人物一致性，真人半身出镜。

[具体表情]，人物位于画面中央，占画面约38%，[具体动作：双手指向上方/双手举起/双手打开]，人物边缘有白色或浅黄色 #FDFFA7 描边。

将中文标题"[封面标题]"拆分成[2/3/4]个巨大词块：[词块1]、[词块2]、[词块3]、[词块4]。

词块使用超粗综艺黑体，柔和浅黄色 #FDFFA7 和纯白交替，全部加粗黑描边。词块分别放在画面上方、左侧、右侧和底部，围绕人物排布，部分文字被人物身体轻微遮挡，形成前后层次；如果词块背后有底板或色块，黄色统一使用 #FDFFA7。

背景是[深色或中性背景描述]，轻微虚化，局部有暗角，让 #FDFFA7/白色大字更突出。

加入少量手绘箭头、短横线、感叹号或英文涂鸦小字，数量控制在3个以内，增加街头感和节奏感。

3:4竖版小红书封面，强对比，超大字视觉锤，人物与文字紧密组合但不遮挡眼睛和嘴巴，不超出安全区。
```

## 风格三：小白科普问答风

黑灰科技海报、模糊软件界面或真实室内背景，人物认真或疑惑，问号和大白话标题突出"给小白讲明白"。

### 视觉特征
- 背景：黑灰科技海报、模糊软件界面、真实室内或参考素材延展背景
- 人物：居中偏下，托腮或认真看镜头
- 标题：上方大字，像口语解释
- 点缀：大问号、勾选人群标签

### 提示词模板

```
参考图1的人物五官特征，保持人物一致性，真人半身或大头出镜。

[具体表情：托腮疑惑/认真解释/微微睁大眼睛]，人物位于画面中下方，占画面约42%，[手部动作：单手托腮/手指轻点脸侧/指向上方标题]，人物边缘有白色或浅黄色 #FDFFA7 描边。

画面上方放置超大中文标题"[封面标题]"，字体为超粗黑体，柔和浅黄色 #FDFFA7 填充，粗黑描边，文字分成2-3行，像在用大白话解释概念；标题背后的黄色底板或色块统一使用 #FDFFA7。

人物旁边放置2个巨大的白色问号，黑色描边，轻微投影，问号围绕人物头部但不遮挡脸。

画面左下或右下放置3条绿色勾选标签：[标签1]、[标签2]、[标签3]，每条是绿色勾选框+白色粗体文字+黑色描边。

背景是[黑灰科技海报/模糊软件界面/真实室内/参考素材延展背景]，略微虚化或压暗，保留科技科普氛围，不抢标题和人物。

3:4竖版小红书封面，标题一眼可读，信息清楚，适合小白科普，不超出安全区。
```

## 风格五：产品测评榜单风

大数字、箭头、维度标签和产品名形成测评感，适合"5款产品"、"7个维度"、"横向测评"。

### 视觉特征
- 大数字：占底部或中部，如5款、7个维度
- 人物：震惊或认真，指向产品/榜单
- 元素：箭头、产品卡片、维度标签
- 背景：深色或工具相关背景

### 提示词模板

```
参考图1的人物五官特征，保持人物一致性，真人半身出镜。

[具体表情：张嘴震惊/认真测评/眉毛上扬]，人物位于画面[左下/右下/中央]，占画面约40%，[具体动作：手指指向数字/手掌推向产品卡片/侧身看向榜单]，人物边缘有浅黄色 #FDFFA7 描边。

画面顶部放置中文标题"[封面标题前半]"，字体为柔和浅黄色 #FDFFA7 超粗黑体，粗黑描边。

画面底部放置巨大数字和结果短语"[封面标题后半]"，例如"5款产品"、"7个维度"，数字字号最大，柔和浅黄色 #FDFFA7 填充，黑色描边，占据底部约30%。
```

## 通用写提示词规范

1. **先写画面，再写风格**：人物、标题、素材、背景、装饰、构图顺序要清楚。
2. **标题必须原样出现**：不要让模型改写用户确认过的标题。
3. **清单文字必须短**：每条2-6个字，适合封面阅读。
4. **参考图直接编号**：写"参考图1/图2/图3"，不要长篇复述。
5. **少用负面词**：尽量正向描述画面里应该出现什么。
6. **保持阿囤囤风格**：真人出镜、#FDFFA7/白色大字、粗黑描边、强情绪、绿色勾选、贴纸感、高点击密度。

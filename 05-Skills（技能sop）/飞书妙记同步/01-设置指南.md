## 1. 准备工作

- macOS 电脑（已安装 Python 3）
- 飞书账号（有妙记权限）
- Obsidian vault（本知识库）

### 脚本位置

```
.scripts/feishu-sync/
├── feishu-to-obsidian.py    ← 主同步脚本（OAuth 版 v2）
├── config.example.json      ← 配置模板
├── com.vault.feishu-sync.plist    ← 定时同步（每15分钟）
└── com.vault.feishu-process.plist ← AI 处理队列（每60秒）
```

### 同步目标目录

```
日记（每日日记）/          ← 妙记原文存这里
00-灵感库（标记灵感）/知识卡片/  ← AI 提炼的卡片存这里
金句素材库.md             ← 金句自动追加到此处
```

## 2. 飞书应用配置

1. 打开 [飞书开放平台](https://open.feishu.cn/app)
2. 创建企业自建应用 → 任意命名（如"妙记同步"）
3. 获取 **App ID** 和 **App Secret**
4. 开通权限：
   - `drive:drive:readonly`（云盘读取）
   - `docx:document:readonly`（文档读取）
   - `offline_access`（刷新令牌）
5. 设置重定向 URI：
   ```
   http://localhost:9981/callback
   ```
6. **发布应用**（必须发布权限才能生效）

## 3. 首次授权（OAuth）

```bash
# 复制配置模板
cp .scripts/feishu-sync/config.example.json ~/.config/feishu-obsidian/config.json

# 编辑配置文件，填入 app_id 和 app_secret
vim ~/.config/feishu-obsidian/config.json

# 运行授权（会自动打开浏览器）
python3 .scripts/feishu-sync/feishu-to-obsidian.py --auth
```

授权成功后，`refresh_token` 会自动写入配置，以后无需重复授权。

## 4. 配置文件详解

```json
{
  "feishu": {
    "app_id": "cli_xxxxxxxxxxxxx",
    "app_secret": "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
    "sync_interval_hours": 1
  },
  "vault": {
    "path": "/Users/mac/Documents/知识库/日记（每日日记）",
    "default_type": "讲课",
    "type_keywords": {
      "讲课|课程|讲座|培训|教学": "讲课",
      "对话|聊天|沟通|讨论|咨询": "对话",
      "学习|阅读|笔记|研究|看书": "学习",
      "会议|开会|周会|月会|复盘": "会议",
      "灵感|想法|思考|反思|觉察": "思考"
    }
  },
  "ai": {
    "anthropic_api_key": "sk-ant-xxxxxxxxxxxx"
  },
  "state_file": "/Users/mac/.config/feishu-obsidian/sync_state.json"
}
```

| 字段 | 说明 |
|------|------|
| `feishu.app_id` | 飞书开放平台 App ID |
| `feishu.app_secret` | 飞书开放平台 App Secret |
| `vault.path` | 妙记存入的 vault 目录 |
| `vault.type_keywords` | 根据标题/内容自动识别录音类型 |
| `ai.anthropic_api_key` | Anthropic API Key（AI 提炼用） |
| `state_file` | 同步状态记录文件（防重复同步） |

## 5. 安装 launchd 定时任务

### 5.1 定时同步（每15分钟）

```bash
cp .scripts/feishu-sync/com.vault.feishu-sync.plist ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.vault.feishu-sync.plist
```

### 5.2 AI 处理队列（每60秒检测）

```bash
cp .scripts/feishu-sync/com.vault.feishu-process.plist ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.vault.feishu-process.plist
```

> AI 处理仅在 Obsidian 运行时触发（检测进程 `pgrep -q "Obsidian"`），不会在后台浪费 API 配额。

### 5.3 管理命令

```bash
# 查看状态
launchctl list | grep feishu

# 停止
launchctl unload ~/Library/LaunchAgents/com.vault.feishu-sync.plist
launchctl unload ~/Library/LaunchAgents/com.vault.feishu-process.plist

# 查看日志
tail -f /tmp/feishu-sync.stdout.log
tail -f /tmp/feishu-sync.stderr.log
```

## 6. 手动使用

```bash
# 正常同步（只拉取新内容）
python3 .scripts/feishu-sync/feishu-to-obsidian.py

# 试运行（看看会拉什么，不写入）
python3 .scripts/feishu-sync/feishu-to-obsidian.py --dry-run

# 强制重新同步所有
python3 .scripts/feishu-sync/feishu-to-obsidian.py --force

# 只看状态
python3 .scripts/feishu-sync/feishu-to-obsidian.py --status

# 手动触发 AI 处理队列
python3 .scripts/feishu-sync/feishu-to-obsidian.py --process-queue
```

## 7. 工作流程说明

```
飞书妙记录音
     │
     ▼  (每15分钟自动同步)
日记（每日日记）/*.md    ← 写入原始转写，状态：待处理
     │
     ▼  (每60秒检测，仅Obsidian开启时)
AI 处理（Claude API）
     │
     ├─ 创建知识卡片 → 00-灵感库/知识卡片/KC-xxx.md
     ├─ 提取金句     → 追加到 金句素材库.md
     └─ 更新笔记状态 → status: 已处理
```

### 每条妙记笔记的 frontmatter

```yaml
---
created: 2026-07-14
source: 飞书妙记
feishu_url: "https://..."
type: 讲课        # 自动识别：讲课/对话/学习/会议/思考
status: 待处理     # AI 处理后改为"已处理"
tags:
  - 原始素材/妙记
---
```

## 8. 排错指南

| 问题 | 解决方案 |
|------|---------|
| `Token 无效` | 重新运行 `--auth` 授权 |
| 同步不到妙记 | 确认飞书应用已发布 + 妙记在云空间 |
| AI 处理不触发 | 检查 `ai.anthropic_api_key` 是否配置 |
| launchd 不执行 | 检查 `~/Library/LaunchAgents/` 权限 |
| 日志报错 | 查看 `/tmp/feishu-sync.stderr.log` |

# 飞书妙记 → Obsidian 自动同步 · 设置指南

> 把你的讲课、学习、对话录音，从飞书妙记自动同步到 Obsidian vault。

---

## 📋 前置要求

- [ ] macOS 系统（已满足）
- [ ] Python 3（已满足：`/usr/local/bin/python3`）
- [ ] 飞书账号（已满足）
- [ ] `requests` 库（已满足）

---

## 🔧 步骤一：飞书开放平台创建应用

1. 打开 [飞书开放平台](https://open.feishu.cn) → 点击右上角「开发者登录」
2. 进入 **开发者后台** → 点击「创建应用」→「企业自建应用」
   - 应用名称：`Obsidian 妙记同步`（随意）
   - 应用描述：自动同步妙记笔记到本地 Obsidian
3. 创建完成后，进入 **凭证与基础信息**，记下：
   - **App ID**（格式：`cli_xxxxxxxxxxxxx`）
   - **App Secret**（点击「显示」复制）
4. 进入 **权限管理**，搜索并开启以下权限：
   - `drive:drive:readonly` — 读取云空间文件列表
   - `docx:document:readonly` — 读取文档内容
5. 进入 **安全设置** → 配置 **IP 白名单**（可选，如果你有固定 IP）
6. 进入 **版本管理与发布** → 创建版本 → 填写版本号 `1.0.0` → 保存并发布

> ⏳ 应用发布审核通常几分钟内自动通过。

---

## 🔧 步骤二：配置飞书文件夹授权

飞书应用默认只能访问它被"邀请"进入的文件夹。

1. 打开飞书 → 进入 **云空间** → 创建一个新文件夹，命名为 `妙记同步`
2. 进入这个文件夹 → 点击右上角 **「···」** → **「共享」**
3. 在搜索框中输入你的应用名称（`Obsidian 妙记同步`）
4. 将应用添加为 **成员**（至少有"阅读"权限即可）
5. **获取文件夹 token**：
   - 在浏览器中打开这个文件夹
   - URL 格式：`https://xxx.feishu.cn/drive/folder/xxxxxxxxx`
   - 最后一串 `xxxxxxxxx` 就是 **folder_token**（格式：`fldcnxxxxxxxxxxxx`）

## 🔧 步骤三：配置本地脚本

### 3.1 编辑配置文件

```bash
vim ~/.config/feishu-obsidian/config.json
```

填入步骤一获取的 App ID、App Secret，和步骤二获取的 folder_token：

```json
{
  "feishu": {
    "app_id": "cli_xxxxxxxxxxxxx",
    "app_secret": "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
    "folder_token": "fldcnxxxxxxxxxxxxxxxxxxxx",
    "sync_interval_hours": 1
  },
  "vault": {
    "path": "/Users/mac/Documents/知识库/00-灵感库（标记灵感）/飞书妙记",
    "default_type": "讲课",
    "type_keywords": {
      "讲课|课程|讲座|培训|教学": "讲课",
      "对话|聊天|沟通|讨论|咨询": "对话",
      "学习|阅读|笔记|研究|看书": "学习",
      "会议|开会|周会|月会|复盘": "会议",
      "灵感|想法|思考|反思|觉察": "思考"
    }
  },
  "state_file": "/Users/mac/.config/feishu-obsidian/sync_state.json"
}
```

### 3.2 手动测试

```bash
cd /Users/mac/Documents/.scripts/feishu-sync
python3 feishu-to-obsidian.py --dry-run
```

如果能正确列出文件，说明配置成功。再跑一次正常同步：

```bash
python3 feishu-to-obsidian.py
```

## 🔧 步骤四：设置定时自动同步

### 方式一：launchd（推荐，macOS 原生）

创建定时任务配置：

```bash
cp /Users/mac/Documents/.scripts/feishu-sync/com.vault.feishu-sync.plist ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.vault.feishu-sync.plist
```

### 方式二：crontab（备选）

```bash
crontab -e
```

添加一行（每小时执行一次）：

```
0 * * * * /usr/local/bin/python3 /Users/mac/Documents/.scripts/feishu-sync/feishu-to-obsidian.py >> /tmp/feishu-sync.log 2>&1
```

## 🔧 步骤五：在飞书中使用

### 日常使用流程

```
① 打开飞书 → 妙记 → 开始录音
② 录音结束 → 妙记自动转写 + AI 生成笔记
③ **将妙记文档移入「妙记同步」文件夹**
④ 脚本下次运行时自动拉取到 Obsidian
```

### 设置妙记自动保存位置（推荐）

1. 打开飞书 → 点击左上角头像 → **设置**
2. 找到 **妙记** 相关设置
3. 将 **笔记默认保存位置** 设置为「妙记同步」文件夹
4. 此后所有妙记自动进入同步文件夹，**无需手动移动**

## 🛟 常见问题

| 问题 | 解决方法 |
|------|---------|
| `配置文件不存在` | 先跑 `python3 feishu-to-obsidian.py --init` |
| `获取 token 失败` | 检查 App ID / App Secret 是否正确 |
| `列出文件夹失败` | 确认 folder_token 正确，且应用已被添加到该文件夹的成员中 |
| 同步后没看到文件 | 跑 `python3 feishu-to-obsidian.py --status` 检查状态 |
| 想重新同步所有文件 | `python3 feishu-to-obsidian.py --force` |

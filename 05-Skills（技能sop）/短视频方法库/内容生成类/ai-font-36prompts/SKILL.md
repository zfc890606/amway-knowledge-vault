# 🖌️ 36种AI中文字体 · 出图 Prompt 库

> **一句话用法：** 把你想生成的文字告诉我（如"发财"），我会按你选的风格输出可直接复制到即梦/豆包/MJ的完整提示词。

从下方36种风格中选一款。

我会按你选的风格+你的文字，生成可直接复制到AI绘图工具的完整提示词（格式：`中文字体设计，黑底白字，[文字]，[风格描述]，[字效]，[背景]，[整体氛围]`）。

## 💡 使用技巧

1. **先出黑底白字** — 先用最简单版本确认字体造型，满意再加背景和装饰
2. **字越少效果越好** — 1-4字效果最佳，超过6字容易变形
3. **即梦AI优先** — 中文字体生成推荐 即梦AI（Dreamina）/ 豆包AI，英文AI工具对中文字体支持较差
4. **组合叠加** — 书法+水墨、创意+霓虹、科技+金属 可以叠加使用
5. **负面提示词** — 如果不是黑底白字，可加 `no realistic photo, no 3D render`
6. **即梦AI参数建议** — 精细度 70-80%，宽高比 1:1 或 3:4

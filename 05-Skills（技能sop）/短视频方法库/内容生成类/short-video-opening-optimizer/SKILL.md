# 🎬 短视频开头优化器 (Short Video Opening Optimizer)

> **黄金3秒法则：** 开头决定完播率，完播率决定推流。

## 功能

- **开头诊断** — 分析已有脚本开头，指出吸引力不足的问题
- **钩子生成** — 按不同模式生成高打开率的开头方案（5+种钩子类型）
- **平台适配** — 针对抖音/快手/视频号/TikTok/Reels/Shorts 分别优化
- **完播率预估** — 对开头方案做完播率预判和排序

## 钩子类型库

| 类型 | 公式 | 适用场景 |
|------|------|---------|
| 悬念钩子 | "为什么XX…" / "我发现了…" | 知识分享、揭秘类 |
| 冲突钩子 | "千万别XX，除非…" | 观点争议、反转类 |
| 利益钩子 | "3个方法帮你…" | 干货教程、技巧类 |
| 共鸣钩子 | "你是不是也…" / "90%的人都…" | 情感、生活类 |
| 数据钩子 | "XX人都在用…" | 权威背书、行业洞察 |
| 反常识钩子 | "你以为XX？其实错了" | 认知颠覆类 |
| 紧迫钩子 | "再不看就晚了…" | 时效热点类 |

## 使用方法

```text
帮我优化这段开头：[粘贴你的脚本前15秒]
→ 输出 3-5 个高完播率开头方案 + 推荐理由

给这段脚本配一个抖音黄金3秒开头：[脚本内容]
→ 输出适配抖音的开头方案
```

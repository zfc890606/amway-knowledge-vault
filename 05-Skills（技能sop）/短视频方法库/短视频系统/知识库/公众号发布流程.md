| 对比项 | 🅰️ 路径A：手动粘贴 | 🅱️ 路径B：API 自动上传 |

## 🅰️ 路径A：Markdown → WeChat 转换（零配置，立刻能用）

### 你需要的工具

一个 Python 脚本：`.claude/skills/wechat-publisher/convert.py`
- 将 Markdown 文章自动转换成微信公众号编辑器可粘贴的 HTML
- 不需要安装任何第三方库（仅用 Python 标准库）
- macOS 系统自带 Python 3，开箱即用

### 操作步骤

```
1. 写文章
   └─ 按 [[公众号文章模板]] 在 Obsidian 中完成文章
   └─ 存到 公众号文章/ 目录

2. 转换格式
   └─ 在终端运行：
     python3 .claude/skills/wechat-publisher/convert.py "公众号文章/article-xxx.md"
   └─ 脚本会输出 HTML 到屏幕
   └─ 脚本自动将 HTML 复制到剪贴板（macOS）

3. 粘贴到公众号后台 → 保存草稿
   └─ 打开 https://mp.weixin.qq.com
   └─ 新建图文消息
   └─ 在编辑器中按 Cmd+V（Mac）/ Ctrl+V（Windows）粘贴
   └─ 检查格式是否完整
   └─ 自行补充封面图、摘要等素材
   └─ 点击「保存为草稿」
   └─ 在 Obsidian 中更新文章状态为「🟡 草稿」

4. 自行发布（等你准备好图片后）
   └─ 在公众号后台打开草稿 → 添加图片 → 确认无误后发布
   └─ 发布后更新状态为「✅ 已发布」
```

### 转换效果对照

| Markdown 元素 | → | WeChat 编辑器效果 |
|--------------|---|-----------------|
| `# 标题` | → | 大标题（H1） |
| `## 分段标题` | → | 分段标题（H2） |
| `### 子要点` | → | 子标题（H3） |
| `**加粗**` | → | **加粗** |
| `> 引用` | → | 引用块（金句突出） |
| `- 列表` | → | 无序列表 |
| `1. 列表` | → | 有序列表 |
| `` `代码` `` | → | 等宽代码字体 |
| `---` | → | 分隔线 |
| `[[wikilink]]` | → | 纯文本（去掉方括号） |
| 段落 | → | 自动分段（p 标签） |

## 🅱️ 路径B：API 自动上传草稿箱（推荐）

> 我说"发公众号"，文章自动进入公众号草稿箱，你只需到后台加图片发布。

### 前置配置（一次性）

在公众号后台完成以下配置，然后把凭据给我：

#### Step 1：获取 AppID 和 AppSecret

```
公众号后台 → 设置与开发 → 基本配置
├─ AppID：直接复制
└─ AppSecret：点击「重置」获取（只显示一次，马上记下来）
```

#### Step 2：配置 IP 白名单

```
公众号后台 → 设置与开发 → 基本配置 → IP 白名单
└─ 添加本机公网 IP（运行脚本的电脑）

查看本机 IP：
  curl ifconfig.me
```

#### Step 3：上传封面图获取 media_id

```
公众号后台 → 素材库 → 图片 → 上传一张封面图
└─ 上传后点击图片，URL 中的 `material/` 后面的 ID 就是 media_id
   或者：公众号后台 → 素材管理 → 鼠标悬停图片 → 链接中的 `mmbiz.qpic.cn` 参数
```

#### Step 4：把凭据给我

拿到以下3个值后发给我，我帮你写入配置文件：

```
AppID:        wx...
AppSecret:    xxxxxxxxxx...
thumb_media_id:   xxxxxx...
```

配置写入后存储在 `.claude/skills/wechat-publisher/config.json`，**不会泄露**。

### 操作步骤（配置完成后）

```
你说 "帮我把文章发到公众号"

我自动执行：
  → 读取 Markdown 文章
  → 转换为公众号 HTML
  → 获取 access_token（自动缓存，到期刷新）
  → 调用微信 API 上传到草稿箱
  → 返回草稿 media_id
  → 告诉你：去 mp.weixin.qq.com → 草稿箱 → 补充图片后发布
```

### 命令

配置完成后也可以手动运行：

```bash
# 上传草稿
python3 .claude/skills/wechat-publisher/publish.py "公众号文章/article-xxx.md"

# 指定标题
python3 .claude/skills/wechat-publisher/publish.py "公众号文章/article-xxx.md" --title "自定义标题"

# 预览不上传
python3 .claude/skills/wechat-publisher/publish.py "公众号文章/article-xxx.md" --no-upload
```

### 接口参考

- [微信公众号 · 草稿箱接口文档](https://developers.weixin.qq.com/doc/offiaccount/Draft_Box/Add_draft.html)
- 上传草稿：`POST https://api.weixin.qq.com/cgi-bin/draft/add`
- 获取 access_token：`GET https://api.weixin.qq.com/cgi-bin/token`

## 常见问题

### Q：为什么不用 Obsidian 插件直接发布？

Obsidian 的微信公众号插件大多需要配置 AppID/AppSecret，和路径B的门槛一样高。
路径A的「转换→复制→粘贴」只需3步，比配置插件更快上手。

### Q：图片怎么办？

目前需要手动上传到公众号素材库。后续可以扩展：
- 自动转换本地图片为素材库 URL（需 API 权限）
- 使用图床（如阿里云 OSS / 腾讯云 COS）

### Q：格式乱了怎么办？

1. 检查 Markdown 文件是否使用了模板中规定的格式
2. 在公众号编辑器中，可以选中文字手动调整格式
3. 如果部分样式丢失（如块引用），可以手动重新应用

### Q：如何更新已发布的文章？

微信公众号已发布文章无法修改正文，只能修改最多 20 个字符。
需要更新内容的话：
1. 在 Obsidian 中修改文章（保留修改记录）
2. 重新发布为**新文章**（旧文章删除或保留）
3. 在新文章中注明更新说明

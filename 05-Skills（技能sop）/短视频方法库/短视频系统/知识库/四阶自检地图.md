# 🗺️ 短视频四阶自检地图

> **来源**：博主「姜胡说」方法论 | **核心公式**：长期主义 = 及时止损 × 认知系统 × 短期闭环
>
> 不知道自己卡在哪？拿这张图自检。每个阶段都有明确的信号和下一步动作。

---

## 总览图

```
                    长期主义 = 及时止损 × 认知系统 × 短期闭环
                                   │
      ┌────────────────────────────┼────────────────────────────┐
      ▼                            ▼                            ▼
┌─────────────┐          ┌─────────────────┐          ┌──────────────┐
│  ① 止损     │  ───→   │  ② 构建系统      │  ───→   │  ③ 形成闭环   │  ───→  ④ 飞轮
│  (方向)     │          │  (效率)          │          │  (底气)       │        (复利)
└─────────────┘          └─────────────────┘          └──────────────┘
      │                          │                          │
      ▼                          ▼                          ▼
  活下来                     效率化                     有反馈
```

**三个因子相乘**，任何一个为零结果就是零。

## 🥇 第一阶：止损期 — "我该不该继续？"

### 你在这个阶段的信号

- ❓ 刚起步，发了没几条，效果不好
- ❓ 不知道该坚持还是换方向
- ❓ 拍了又拍，但没想过每条在"测什么"

### 自检清单

| 自检问题 | ✅ / ❌ | 如果 ❌ 怎么办 |
|---------|---------|-------------|
| 我至少拍了 **10-20条** 了吗？ | ☐ | 还没到量，先别下结论，继续拍 |
| 效果不好是**表达问题**还是**方向问题**？ | ☐ | 别人讲同主题能火吗？能→练表达；主题没流量→换方向 |
| 我清楚每一条在**验证什么假设**吗？ | ☐ | 拍之前写一句：这条视频我想验证_____ |
| 最差的那条和最火的那条，区别在哪里？ | ☐ | 对比分析，找到变量 |
| 我有没有因为一两条数据差就放弃了？ | ☐ | 数据样本不够时，结论不可靠 |

### 止损期的核心原则

> **止损不是不做了，是换方向、换选题、换形式，继续试。**

- 早期多拍是对的，但目的是**试错**，不是盲目堆量
- 每拍一条都要清楚：**我在试什么？这条视频出去，结果是什么？**
- 最差的结果不是"没火"，而是"火了你都不知道为什么"

### ✅ 进入下一阶的标准

你清楚知道以下三个问题的答案：
1. 你的目标用户是谁
2. 什么选题/形式他们爱看
3. 你的表达能力够不够（或正在练）

## 🥈 第二阶：系统期 — "效率怎么提？"

### 你在这个阶段的信号

- ✅ 已经找到方向，但每次创作都**从零开始**
- ✅ 感觉输出很吃力，**每次选题→写稿→拍摄都像打仗**
- ✅ 知道该做什么，但**做不快**

### 自检清单

| 自检问题 | ✅ / ❌ | 如果 ❌ 怎么办 |
|---------|---------|-------------|
| 我有稳定的**信息输入**节奏吗？ | ☐ | 见下方「输入系统」 |
| 我有**知识卡片**系统吗？ | ☐ | 见下方「处理系统—知识卡片法」 |
| 我用卡片"讲"视频，而不是写稿背词？ | ☐ | 见下方「输出系统」 |
| 我的卡片四要素对齐短视频结构吗？ | ☐ | 见下方「卡片即脚本」对应关系 |

### 构建系统三环

```
┌─ 输入 ────────────┐   ┌─ 处理 ────────────┐   ┌─ 输出 ────────────┐
│                    │   │                    │   │                    │
│  S级：经典书籍     │   │  知识卡片四要素：  │   │  不写逐字稿        │
│  A级：深度文章     │ → │  一句话要点        │ → │  用知识卡片"讲"   │
│  B级：优质课程     │   │  → 底层逻辑        │   │  一镜到底          │
│  C级：碎片信息     │   │  → 执行步骤        │   │  掐头去尾配字幕    │
│                    │   │  → 边界条件        │   │                    │
└────────────────────┘   └────────────────────┘   └────────────────────┘
```

### 输入系统 — 信息源分级

| 级别 | 类型 | 占比 | 每天花多少 |
|:----:|------|:----:|----------|
| **S级** | 经典书籍 | 50% | 30min 精读 |
| **A级** | 深度文章/论文 | 25% | 15min 速读 |
| **B级** | 优质课程/长视频 | 15% | 10min 扫描 |
| **C级** | 碎片信息 | 10% | 随缘 |

### 处理系统 — 知识卡片法 ⭐

每张卡片统一格式：

```markdown
## [概念名/关键词]

**一句话要点**
（一句话说清楚这是什么）

**背后的机制/逻辑**
（为什么是这样？底层原理是什么？）

**执行步骤/怎么用**
（具体怎么做？可操作的行动指南）

**边界条件/注意事项**
（什么时候不适用？常见错误？）
```

### 🔑 核心洞察：卡片四要素 = 短视频四段结构

| 卡片要素 | 对应视频段落 |
|---------|------------|
| **一句话要点** | 🎬 开头 **Hook** — 前5秒抓住注意力 |
| **背后的机制/逻辑** | 🎬 第2段 **为什么重要** — 建立信任 |
| **执行步骤/怎么用** | 🎬 第3段 **视频主体** — 给干货 |
| **边界条件/注意事项** | 🎬 结尾 **升华/避坑** — 引导互动+CTA |

> **写卡片 = 写脚本。卡片备好了，对着镜头讲就行了。**

### 输出系统

1. 从卡片库挑3-5张相关卡片
2. 按「hook→逻辑→步骤→边界」排列
3. 对着卡片讲（不是背稿）
4. 一镜到底，掐头去尾配字幕

### ✅ 进入下一阶的标准

- 你可以在30分钟内从卡片库里"拼"出一条视频
- 你的输出频率稳定（至少每周2-3条）
- 你不再为"写脚本"头疼

## 🥉 第三阶：闭环期 — "我怎么知道自己做对了？"

### 你在这个阶段的信号

- ✅ 能稳定输出了，但**不知道效果怎么样**
- ✅ 发了就发了，**没有数据反馈**
- ✅ 凭感觉决定下一条拍什么

### 自检清单

| 自检问题 | ✅ / ❌ | 如果 ❌ 怎么办 |
|---------|---------|-------------|
| 每条视频发完后**3天填数据**了吗？ | ☐ | 建立数据追踪习惯，见 [[数据追踪]] |
| 我能说出**哪条最好/哪条最差**和**为什么**吗？ | ☐ | 用数据对比，找变量 |
| 数据在**指导我的下一条选题**吗？ | ☐ | 见下方「数据决策矩阵」 |
| 我有**闭环记录**（哪怕就一条评论/一个私信）？ | ☐ | 每条视频至少记录1条反馈 |

### 数据决策矩阵

发完视频3天后，对照数据做决策：

| 播放量 | 互动率（点赞+评论+收藏+分享）÷播放 | 判断 | 行动 |
|-------|----------------------------------|------|------|
| 📈 高 | 📈 高 | ✅ 选题准、内容好、表达到位 | **保留要素，复制成功** |
| 📈 高 | 📉 低 | ⚠️ 标题/封面吸引人但内容不够好 | **加强内容干货，增加CTA引导** |
| 📉 低 | 📈 高 | ⚠️ 内容好但包装不够 | **优化标题+封面，重发或改版** |
| 📉 低 | 📉 低 | ❌ 选题不够痛 | **换方向/换选题** |

### 三大目标对应数据

| 你的目标 | 看什么数据 |
|---------|-----------|
| 🎯 **要流量** | 播放量、完播率、点赞 |
| 🧠 **要干货验证** | 收藏、关注 |
| 💰 **要变现** | 有没有人买单（哪怕只卖一单） |

> **"有了闭环，才有底气继续。"**

### 迭代流程

```
发布 → 3天后填数据 → 对照决策矩阵 → 决定下一条方向
                                                ↓
                                      更新痛点排行榜 + 更新内容策略
                                                ↓
                                      回到第二阶（系统）快速产出
```

### ✅ 进入下一阶的标准

- 你能对每一条视频说出"它帮我验证了什么"
- 数据在驱动你的选题方向，而不是拍脑门
- 你有了第一批"因为你内容而来"的精准关注/私信

## 🏆 第四阶：飞轮期 — "系统在替你工作"

### 你在这个阶段的信号

- ✅ 前三阶都跑通了
- ✅ 有稳定的内容产出节奏 + 数据反馈循环
- ✅ 内容资产开始**自动积累和放大**

### 飞轮效应

```
止损 → 你活下来了（不纠结方向）
系统 → 你效率化了（30分钟一条）
闭环 → 你有反馈了（数据驱动决策）
             ↓
      ⚡ 飞轮启动 ⚡
             ↓
    ┌───────────────────┐
    │ 每一条视频 →      │
    │   播放量积累       │
    │   搜索排名上升     │
    │   信任资产增长     │
    │   → 下一条更容易  │
    └───────────────────┘
```

### 复利来源

| 资产类型 | 怎么积累 | 多久见效 |
|---------|---------|---------|
| 🔍 **搜索流量** | 每条视频都是SEO资产，持续被搜到 | 3-6个月 |
| 🤝 **信任资产** | 内容越多人看，越多人信任你 | 6-12个月 |
| 🃏 **卡片库** | 卡片越写越多，拼视频越来越快 | 持续累积 |
| 📊 **数据资产** | 数据越多，决策越准 | 3个月+ |

> **"你不能保证每一条都爆，但你可以保证，你的系统会赢。"**
> **"别当买彩票的人，当那个做系统的人。"**

## 📋 快速自检卡（打印版）

> 每次迷茫时拿出来过一遍，定位自己在哪一阶。

```
□ 第一阶（止损）：我清楚我在测什么吗？拍了10-20条了吗？
   → 还没 → 继续试，别下结论

□ 第二阶（系统）：我有知识卡片系统吗？输出靠卡片还是靠硬写？
   → 硬写 → 先建卡片库

□ 第三阶（闭环）：我有数据反馈吗？数据在指导决策吗？
   → 没有 → 每条发完3天后填数据再决定下一条

□ 第四阶（飞轮）：前三阶都跑通了吗？系统在自动运转吗？
   → 还没 → 回到卡住的那一阶
```

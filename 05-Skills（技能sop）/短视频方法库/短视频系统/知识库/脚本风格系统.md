## 〇、一句话总结

> **一个踩过坑的普通人，用三段递进 + 自曝失败 + 土味比喻 + 认知反转，告诉正在踩坑的人：这事没那么玄，顺序对了就行。**

### 1️⃣ 开场铁律 — 痛点问句 + 自曝入局

**这是你所有脚本的共同起点，没有例外。**

```
[痛点场景（1-2句）] + [我以前也……] + [一句自嘲/具体惨状]
```

**真实案例对照：**

| 脚本 | 开头 |
|------|------|
| 001 | "做过安利，没做起来？**我以前也是**。干了半年，一分钱没挣着。" |
| 002 | "三个月了，一个客户没有。你是不是开始怀疑……**我告诉你**一句不太好听的大实话。" |
| 003 | "你是不是也觉得……？**我当初也这么想的**。干了两个月，一个客户没有。" |
| 006 | "想做安利，但就是不敢开口。手机拿起来……删了。**我以前也这样。我太懂了。**" |
| 009 | "我也想试试，但我不行……**我以前也觉得自己不行。**" |
| 019 | "一个月被拒绝了二十一次。最狠的那次，我刚开口，对方直接让我走。" |

**给你自己写的铁律：**
> ❌ 不要上来就给答案
> ❌ 不要说"今天给大家分享"
> ✅ 先交底再交心——你的失败比他惨，他才会听你的方案

### 2️⃣ 核心句式DNA（你的听觉签名）

**这些句式在你的脚本里出现了80-100%，这就是你的标签——观众听三句就知道是你。**

| 句式 | 作用 | 示例 |
|------|------|------|
| **"我以前也……"** | 开场卸防备 | "我以前也是。""我以前也这么想的。" |
| **"说出来不怕你笑"** | 老师式松弛感 | "说出来不怕你笑，我到现在去跟人聊之前还会紧张。" |
| **"我告诉你……"** | 自信输出 | "我告诉你一句不太好听的大实话""我告诉你，真正有用的方法就3步" |
| **"后来我[发现/搞明白/学会]"** | 认知翻转启动 | "后来我发现，不是我人不行……""后来我花了三年才搞明白一件事" |
| **"很多人 [错误做法]"** | 先破后立 | "很多人做安利，上来就干三件事""很多人搞反了" |

**写稿时在这些句式里选3-4个嵌入，脚本立刻有你的味。**

### 3️⃣ 结构铁律 — 三段递进

**你的脚本结构核心：不是并列三点，是递进。第②步站在第①步的肩膀上，第③步是最终的收口。**

```
┌─ 第一段：痛点场景（铺垫"为什么这件事难"）
├─ 第二段：认知翻转（"后来我发现/其实不是这样的"）
├─ 第三段：方案递进（"①××× → ②××× → ③×××"）
│               ↑ 三步是递进，不是并列
└─ 结尾：一句话总结 + 双钩
```

**你的三段命名法则：用白话命名步骤，不要用名词。**
> ✅ "先把自己吃成案例 → 学本事 → 发出去"
> ❌ "建立个人品牌 → 提升专业能力 → 内容输出"

**真实案例验证：**

| 脚本 | 三段递进 |
|------|---------|
| 001 | ①自己吃出效果 ②学真本事 ③把学的发出去 |
| 002 | ①心态关（转念）②方法关（学专业）③坚持关（熬过前三月） |
| 003 | ①吃成案例（算蛋白账）②学本事（每天半小时）③发出去（半懂不懂最值钱） |
| 005 | ①选一个平台 ②分享知识非广告 ③私信不急先问情况 |
| 010 | ①太自由→自律滤网 ②太温和→决定权还给客户 ③太考验耐心→建管道别急着要水 |
| 011 | ①早餐换蛋白粉+蛋 ②午餐米饭减半 ③晚餐少主食 |
| 014 | ①承认不知道 ②去评论区偷听 ③每天学30分钟 |
| 016 | ①找家底（七成把握）②锁死家底 ③多余精力冲增量 |

**检验标准：**
> 这三步能颠倒顺序吗？
> 能 → 改成递进
> 不能 → 过关

### 4️⃣ 语言指纹 — 土味比喻·重复强调·反问逼认同

#### 4.1 土味比喻（你最大的武器）

**比喻素材库（从你的脚本里提取）：**
- 家庭日常：妈妈发朋友圈、爸爸的衬衫
- 生活场景：锁屏叹气、手机拿起来又放下、碳水开会
- 工作常识：建管道要水、盯盘的人、空壳
- 身体感受：脸上挂不住、心里咯噔一下

**每次写稿，至少给自己塞一个土味比喻。**
> ✅ "比我爸的衬衫还过时"
> ✅ "你这不叫吃得清淡，这叫碳水开会"
> ✅ "做内容不是搞创作，是做内容是亮家底"
> ✅ "你把自己活成了一个盯盘的人"

#### 4.2 重复强调（口语节奏感）

**同一个意思，用两三个不同的说法反复锤：**
```
"每天缺，天天缺，一年三百六十五天都在缺。"  ← 001
"有关就做。无关就不做。就这么简单。"          ← 016
"你这不是没结果，是你明明不知道，却假装自己知道。假装久了，连你自己都信了。"  ← 014
```

**写完后读出声——哪个词/哪句话值得重复三遍？加进去。**

#### 4.3 反问逼认同

**在方案讲完之后，用一个反问让观众自己点头：**
```
"换你你买吗？你肯定也烦。"                                   ← 001
"观众看你三条视频就知道你什么水平，信不信？"               ← 008
"你天天在给自己存脂肪，你的肝受得了吗？"                   ← 008
"如果连你自己先倒了，你身后那批人看谁呢？"                 ← 019
"你今天做的事，跟你的目标有关吗？"                          ← 016
```

### 5️⃣ 老师式表达 — 允许废话·自曝其短·有缺口

**这是你和"卖课博主"最大的区别。你不完美，所以你真实。**

| 特征 | 用法 | 你脚本里的例子 |
|------|------|--------------|
| **自曝其短** | 开头先说自己干过蠢事 | "干了半年，一分钱没挣着" |
| **允许废话** | 插入"说出来不怕你笑""怎么说呢" | 006/009/010/016/017/019 都有 |
| **留缺口** | 承认自己也没全懂 | "不是我了不起，是我终于把顺序搞对了" |
| **反鸡汤** | 不说"你一定能行"，说"没那么玄" | "这事我验证过，不是看来的，是自己走过来的" |

**检验标准：**
> 读出声——被朋友看到会尴尬吗？
> 不尴尬 → 太端了，重写
> 有点尴尬 → 对了，这就是你

### 6️⃣ 结尾双钩（铁律）

**每条脚本必须有，缺一不可：**

```
评论钩 + 追更钩
```

**格式：**
```
"[互动问题]？评论区告诉我，我帮你看看/我帮你分析/我展开讲讲。"
"关注我，下一条说——[下期预告]"
```

### 7️⃣ 数字锚点 — 算账法·数据嵌入

**你擅长在讲故事的过程中自然地嵌入数据，让它变成"证据"而不是"说教"。**

| 使用方式 | 你的例子 |
|---------|---------|
| **需求数据** | "你每天需要73克蛋白质，早餐一碗粥一个包子，不到10克。" |
| **时间数据** | "一个月每天半小时，你就跟90%的人不一样了。" |
| **结果数据** | "半年时间，我的团队做到了二十个人。" |
| **对比数据** | "一天七八十克。一年下来，三四万克。三十多斤肝脏脂肪。" |
| **频率数据** | "一个月被拒绝了二十一次。" |

**写稿铁律：**
- 营养类 → 算账法必用（73g蛋白/碳水算账）
- 业绩类 → 结果数字支撑（半年20人）
- 方法类 → 时间量化（每天半小时、一个月）

### 1️⃣ 选题哲学 — "命中正在行动的人"

姜胡说说的："评论区就是选题。"

**和你风格的结合点：**

| 姜胡说做法 | 你的做法 |
|-----------|---------|
| 社群老伙计问的问题，统计频率排榜 | 你的客户/评论区/同行评论区，什么问题被问最多 |
| 付费问题全是好选题 | 别人愿意付钱问的，是真痛点 |
| 命中正在行动的人 | "播放量卡在500" > "怎么营销" |

**你的选题来源优先级：**
1. 你自己客户/伙伴问的问题（最精准）
2. 同行评论区高频问题
3. 痛点排行榜（已统计好）

**选题检验：**
> ❌ "怎么做好安利" — 太大，没人对号入座
> ✅ "做过安利没做起来？" — 正在行动的人卡在那了

### 2️⃣ 结构参考 — 问题→方案→总结

**姜胡说最核心的骨架，和你的三段递进天然兼容：**

```
你是不是有[痛点]？
    ↓
方案（三段递进）
    ↓
[可选] 讲一下原因 → 过滤不精准的人
    ↓
一句话总结
```

**关于"讲原因"的取舍（姜胡说原话）：**
> "我如果不讲道理，播放量会更高。但我为什么还要讲？过滤。我要牺牲播放量，瞄准真正的用户。"
- 你想要播放量？去掉原因，直接给方案（模式B）
- 你想要精准粉丝？保留原因，没耐心听完的人不是你的客户（模式A）

**你在 001/003/005/011 都是保留了原因的，这天然帮你过滤了泛粉。**

### 3️⃣ 流量思维 — "视频是资产，不是烟花"

| 维度 | 你的做法（和姜胡说一致） |
|------|----------------------|
| **流量来源** | 吃搜索流量，用户搜过来的 |
| **生命周期** | 视频持续有人搜，不是推流半个月就没了 |
| **用户质量** | 搜来的用户本身就是有需求的 |
| **关键词** | 用户怎么说你就怎么讲，不用高大上的词 |

**你的脚本天然适合搜索流量：** 标题里全是用户真实的搜索词（"做过安利没做起来""安利是传销吗""脂肪肝怎么吃""安利新手第一步"）。

---

### 4️⃣ 过滤意识 — "敢于筛选，不是讨好所有人"

姜胡说最特别的一点：**他故意赶人。** 你的脚本其实也在做同样的事：

> "你如果不信，咱就不聊了。这事信不信是你的事，我帮不帮是我的事。" — 你虽然没说这么直白，但你一直在**给方案的同时筛人**。

你的 002（三关）、014（承认无知）、010（三个缺点）——都是劝退型内容。敢劝退的人，留下的才死心塌地。

### 营养科普类

**结构：** 具体人物案例开场 → 算账法拆解 → 三招方案 → 结尾钩

**示例骨架：**
```
"我前两年带一个[人物]，叫他[称呼]吧。[背景描述]"

"他拿着报告来找我，第一句话就是——[痛点问题]"

"我说[称呼]，[认知翻转：你的理解是错的]"

"我给你算笔账。[具体数字拆解]"

"改三样就够了。第一……第二……第三……"

"你发现没有。营养调理不是什么深山老林的偏方。"

"[互动问题]？评论区打出来，我帮你看看。"
```

**核心手法：** 案例代入 → 算账法强化 → 具体可操作的三招
**参考编号：** 011 / 008

### 认知颠覆类（观念升级）

**结构：** 反驳常识 → 三段递进解释 → 反转总结 → 钩子

**示例骨架：**
```
"你是不是也觉得[常见认知]？"

"我告诉你，[颠覆性结论]。"

"[原因/故事/数据支撑]"

"所以很多人搞错了什么？以为[错误认知]，其实是[正确认知]。"

"你今天做的事，跟你的目标有关吗？"
```

**核心手法：** 先破后立，每篇必有一个"嗯？"时刻
**参考编号：** 010 / 014 / 016 / 017

### 方法教学类（工具类）

**结构：** 痛点场景 → 给工具 → 工具拆解 → 最小行动 → 钩子

**示例骨架：**
```
"你是不是也[具体困境]？"

"我跟你说个方法。[来源/老师讲的方法]。"

"大多数人对[这个方法]的理解是什么？[错误理解]。"

"核心是什么？[正确理解]。"

"我给你一个最小动作。[今天就能做的事]。"

"[检验反问]？评论区告诉我。"
```

**核心手法：** 先破错误理解 → 再给正确方法 → 给最小行动
**参考编号：** 016 / 005 / 024

- [ ] 有没有重复强调一处关键句？（三遍锤法）
- [ ] 有没有一个反问让观众自己点头？

### ✅ 老师式检验
- [ ] 被朋友看到会尴尬吗？
- [ ] 有没有"说出来不怕你笑"的松弛感？
- [ ] 你有没有留一个缺口？（"不是我了不起，是……"）

### ✅ 结尾关
- [ ] 评论钩（"你现在卡在哪？评论区聊聊"）？
- [ ] 追更钩（"关注我，下一条说……"）？

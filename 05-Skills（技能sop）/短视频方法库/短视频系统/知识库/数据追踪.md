## 分析模型

每次填写数据后，自动对照以下规则：

```
播放量高 + 互动率高  → ✅ 保留要素，复制成功
播放量高 + 互动率低  → ⚠️ 内容好缺引导，加强CTA
播放量低 + 互动率高  → ⚠️ 选题准但标题/封面差，优化包装
播放量低 + 互动率低  → ❌ 选题不够痛，换方向
```

## 💡 使用说明

1. **发布时间**：发布后3天填入数据（此时数据趋于稳定）
2. **互动率计算**：（点赞+评论+收藏+分享）÷ 播放量 × 100%
3. **私信转化**：通过这条视频主动私信你的人数
4. **填写频率**：每次发视频后填写
5. **复盘时使用**：生成迭代报告时，这个表是核心依据

## 通用风格检查点（所有模板适用 — v2版）

开始写之前，问自己：
- **开场铁律**：痛点问句 + "我以前也……" + 自嘲？
- **结构铁律**：三段递进（不是并列），白话命名步骤？
- **反转关**：有没有"后来我发现……"的认知翻转？
- **语言三料**：土比喻 + 重复锤 + 反问逼认同？
- **沙子**：有没有一句"没必要说但说了就真了"的话？
- **结尾双钩**：评论钩 + 追更钩？缺一不可

每条脚本写完之后，再过一遍 [[脚本风格系统（融合版v2）#写稿自检清单（v2）]]。

## 模板一：痛点共鸣型

### 信息
- **时长**：60-90秒
- **对应情绪曲线**：曲线A（扎心→共情→翻转→再扎→给解法→收尾）
- **嘴替重点**：痛点开场必须是观众心里说不出的委屈/挫败
- **自信锚点**：中间"后来我发现了…"这段要笃定

### 结构

```
🎬 【0:00-0:05】 痛点开场
   嘴替法：替他说出心里话
   "你是不是[目标人群]也遇到过这个问题？"
   "你是不是[具体痛点]？"

🎬 【0:05-0:10】 共情
   情绪法：拉近距离，承认自己也踩过坑
   "我以前也是这样的…"
   "其实这不是你的问题…"

🎬 【0:10-0:45】 解决方案（核心干货）
   自信法：用具体案例+对话，不要讲道理
   "后来我用了这个方法…"
   "第一步…第二步…第三步…"
   （确保三点是递进关系，不是并列）

🎬 【0:45-0:55】 [可选] 原因/底层逻辑
   "为什么很多人卡在这？因为…"

🎬 【0:55-1:00】 一句话总结 + CTA
   不要"所以"开头，直接说结论
   "先让自己成为答案，别人自然来找你"
   "你现在卡在哪？评论区告诉我，我帮你分析"
```

## 模板二：方法教学型

### 信息
- **时长**：60-120秒
- **对应情绪曲线**：曲线B（质疑→承认踩坑→发现问题→给方法→收尾）
- **嘴替重点**：开场替他说出错误做法，让他觉得"说的就是我"
- **自信锚点**：全程用"我告诉你""我跟你说个事"开场，自信输出

### 结构

```
🎬 【0:00-0:05】 痛点 + 承诺
   嘴替法：先说出他现在的错误做法
   "你是不是觉得[常见错误做法]？"
   "我告诉你，真正做起来的人根本不是这么干的"

🎬 【0:05-0:10】 快速建立信任
   自信法：用具体结果说话
   "我用了这个方法，[具体成果]"

🎬 【0:10-1:10】 分步教学
   对话法：每一步都用具体场景+对话来呈现，不要干讲
   "第一步（做什么 + 为什么 + 具体例子）"
   "第二步（做什么 + 为什么 + 具体例子）"
   "第三步（做什么 + 为什么 + 具体例子）"

🎬 【1:10-1:20】 总结强调
   "很多人搞错了什么？以为…其实是…"

🎬 【1:20-1:30】 CTA
   "你现在卡在哪一步？评论区告诉我"
   或 "关注我，下条讲…"
```

## 模板三：案例见证型

### 信息
- **时长**：90-120秒
- **对应情绪曲线**：曲线E（背景→转折→结果→方法→邀请）
- **嘴替重点**：中间让观众产生"他跟我一样普通"的代入感
- **自信锚点**：结尾"你也可以"要有力量

### 结构

```
🎬 【0:00-0:05】 结果前置
   "我有个伙伴，[具体结果]，他是怎么做到的？"

🎬 【0:05-0:15】 背景介绍
   嘴替法：让观众觉得"这不就是我吗"
   "他原来是做[什么]的，遇到[什么困难]"

🎬 【0:15-0:45】 转折点
   "后来他做了[什么改变]…"

🎬 【0:45-1:15】 具体方法
   自信法：方法要具体，可复制

🎬 【1:15-1:30】 你的角色
   "我帮他做了[什么]…"

🎬 【1:30-1:40】 总结 + CTA
   "所以你也可以。想跟他一样，私信我"
```

## 模板四：认知颠覆型

### 信息
- **时长**：45-60秒
- **对应情绪曲线**：曲线C（抛出问题→打破认知→深入解释→对号入座→结论）
- **嘴替重点**：开头替他说出那个"最怕被问的问题"
- **自信锚点**：认知翻转那一刻要果断有力

### 结构

```
🎬 【0:00-0:03】 颠覆性观点
   嘴替法：替他说出他最想反驳的那个观点
   "你是不是也觉得安利就是卖产品的？"
   "我告诉你，安利真正赚钱的人，都不是在卖产品"

🎬 【0:03-0:10】 制造好奇心
   情绪法：停顿 + 放慢
   "是不是很意外？我告诉你真相…"

🎬 【0:10-0:35】 解释/论证
   自信法：三段递进，不要并列
   "真正做起来的人都在做这三件事…"

🎬 【0:35-0:45】 结论强化
   "所以不要再[错误做法]了"

🎬 【0:45-0:50】 CTA
   "你被问过这个问题吗？评论区聊聊"
```

## 模板五：营养科普型

### 信息
- **时长**：60-90秒
- **对应情绪曲线**：曲线D（焦虑→病因诊断→认知反差→给方案→温暖收尾）
- **嘴替重点**：替他说出"我到底吃没吃够"的困惑
- **自信锚点**：自媒体教练式算账说理，数据要准

### 结构

```
🎬 【0:00-0:05】 场景切入
   嘴替法：替他说出日常困惑
   "[常见健康问题]的人，你是不是也有这样的困扰？"

🎬 【0:05-0:10】 问题本质
   情绪法：揭示他不知道的真相
   "其实这个问题背后的原因是什么…"

🎬 【0:10-0:40】 营养方案
   自信法：算账式科普，让他对号入座
   "我建议你补充这三样东西…第一…第二…第三…"

🎬 【0:40-0:55】 原理说明
   "为什么这个有用？因为…"

🎬 【0:55-1:00】 总结 + CTA
   "想定制个人营养方案，私信我你的情况"
```

## 模板六：答疑解惑型

### 信息
- **时长**：45-60秒
- **对应情绪曲线**：曲线F（提问→直接回答→展开解释→建议收尾）
- **嘴替重点**：替他说出困扰已久的问题
- **自信锚点**：直接干脆，不绕弯子

### 结构

```
🎬 【0:00-0:05】 提问开场
   嘴替法：替他说出想问但没问出口的问题
   "经常有人问我：'[具体问题]？'"

🎬 【0:05-0:15】 直接回答
   自信法：开头就给出明确答案，不绕弯
   "我的答案是：[核心观点]"

🎬 【0:15-0:40】 展开解释
   情绪法：三段递进解释
   "为什么？因为三个原因…第一…第二…第三…"

🎬 【0:40-0:50】 建议/方案
   "所以我建议你…"

🎬 【0:50-0:55】 CTA
   "还有别的问题？评论区告诉我"
```

## 模板七：博主极简结构法（评论区选题+痛点结构）

> 🆕 来源：[[知识卡片/KC-视频-001-短视频创作方法论——评论区选题法与痛点结构|KC-视频-001]] · 博主方法论吸收
>
> **什么时候用这个模板**：当你有明确的用户痛点+具体解决方案时，3分钟出稿。
> 不需要复杂的情绪曲线，因为"他正好需要"本身就是最大的情绪。

### 信息
- **时长**：30-90秒
- **对应情绪曲线**：简化版（痛点→方案→总结）
- **嘴替重点**：第一句直接命中评论区里出现最高频的那个问题
- **自信锚点**："我告诉你""我是这么做的"

### 结构

```
🎬 【0:00-0:05】 痛点开场（第一句就让观众对号入座）
   直接从评论区拿来的原话
   "你是不是也[评论区原话]？"
   → 例："你是不是也觉得做安利就得天天刷朋友圈？"

🎬 【0:05-0:45】 给方案（12345步，清晰可执行）
   "我告诉你几个方法，我是这么做的"
   1. ______
   2. ______
   3. ______
   📌 用具体数字+场景+对话，别讲道理
   📌 每步之间用"然后你猜怎么着？"过渡

🎬 【0:45-0:55】 [可选] 讲原因（过滤精准用户）
   "为什么你会有这个问题？因为…"
   → 想要播放量 → 删掉这步
   → 想要精准粉丝 → 保留这步

🎬 【0:55-1:00】 一句话总结
   不需要"所以"，直接说结论
   → 例："先让自己成为答案，别人自然来找你"
   CTA："你现在卡在哪？评论区告诉我"
```

### 什么时候用 vs 不用

| 用这个模板 | 用其他模板 |
|-----------|-----------|
| 用户问题非常具体（"播放量卡500""脂肪肝怎么吃"） | 需要共鸣感（"做安利心累"） |
| 快速产出，不想花30分钟设计情绪曲线 | 需要认知翻转（"安利是传销吗"） |
| 你回答的就是他正在搜的问题 | 内容偏品牌/定位 |
| **大部分安利话题都合适** | 需要讲故事带情绪 |

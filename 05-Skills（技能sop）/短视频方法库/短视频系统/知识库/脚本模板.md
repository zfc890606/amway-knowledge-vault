## 通用风格检查点（所有模板适用）

开始写之前，问自己：
- 这条视频的**嘴替点**是什么？——观众心里说不出口的那句话是什么？
- 情绪**曲线**怎么走？——参考风格系统中的六种情绪曲线
- **自信感**怎么体现？——有没有说一句笃定的结论？

每条脚本写完之后，再过一遍 [[脚本风格系统#八、写稿自检清单]]。

## 模板一：痛点共鸣型

### 信息
- **时长**：60-90秒
- **对应情绪曲线**：曲线A（扎心→共情→翻转→再扎→给解法→收尾）
- **嘴替重点**：痛点开场必须是观众心里说不出的委屈/挫败
- **自信锚点**：中间"后来我发现了…"这段要笃定

### 结构

```
🎬 【0:00-0:05】 痛点开场
   嘴替法：替他说出心里话
   "你是不是[目标人群]也遇到过这个问题？"
   "你是不是[具体痛点]？"

🎬 【0:05-0:10】 共情
   情绪法：拉近距离，承认自己也踩过坑
   "我以前也是这样的…"
   "其实这不是你的问题…"

🎬 【0:10-0:45】 解决方案（核心干货）
   自信法：用具体案例+对话，不要讲道理
   "后来我用了这个方法…"
   "第一步…第二步…第三步…"
   （确保三点是递进关系，不是并列）

🎬 【0:45-0:55】 [可选] 原因/底层逻辑
   "为什么很多人卡在这？因为…"

🎬 【0:55-1:00】 一句话总结 + CTA
   不要"所以"开头，直接说结论
   "先让自己成为答案，别人自然来找你"
   "你现在卡在哪？评论区告诉我，我帮你分析"
```

## 模板二：方法教学型

### 信息
- **时长**：60-120秒
- **对应情绪曲线**：曲线B（质疑→承认踩坑→发现问题→给方法→收尾）
- **嘴替重点**：开场替他说出错误做法，让他觉得"说的就是我"
- **自信锚点**：全程用"我告诉你""我跟你说个事"开场，自信输出

### 结构

```
🎬 【0:00-0:05】 痛点 + 承诺
   嘴替法：先说出他现在的错误做法
   "你是不是觉得[常见错误做法]？"
   "我告诉你，真正做起来的人根本不是这么干的"

🎬 【0:05-0:10】 快速建立信任
   自信法：用具体结果说话
   "我用了这个方法，[具体成果]"

🎬 【0:10-1:10】 分步教学
   对话法：每一步都用具体场景+对话来呈现，不要干讲
   "第一步（做什么 + 为什么 + 具体例子）"
   "第二步（做什么 + 为什么 + 具体例子）"
   "第三步（做什么 + 为什么 + 具体例子）"

🎬 【1:10-1:20】 总结强调
   "很多人搞错了什么？以为…其实是…"

🎬 【1:20-1:30】 CTA
   "你现在卡在哪一步？评论区告诉我"
   或 "关注我，下条讲…"
```

## 模板三：案例见证型

### 信息
- **时长**：90-120秒
- **对应情绪曲线**：曲线E（背景→转折→结果→方法→邀请）
- **嘴替重点**：中间让观众产生"他跟我一样普通"的代入感
- **自信锚点**：结尾"你也可以"要有力量

### 结构

```
🎬 【0:00-0:05】 结果前置
   "我有个伙伴，[具体结果]，他是怎么做到的？"

🎬 【0:05-0:15】 背景介绍
   嘴替法：让观众觉得"这不就是我吗"
   "他原来是做[什么]的，遇到[什么困难]"

🎬 【0:15-0:45】 转折点
   "后来他做了[什么改变]…"

🎬 【0:45-1:15】 具体方法
   自信法：方法要具体，可复制

🎬 【1:15-1:30】 你的角色
   "我帮他做了[什么]…"

🎬 【1:30-1:40】 总结 + CTA
   "所以你也可以。想跟他一样，私信我"
```

## 模板四：认知颠覆型

### 信息
- **时长**：45-60秒
- **对应情绪曲线**：曲线C（抛出问题→打破认知→深入解释→对号入座→结论）
- **嘴替重点**：开头替他说出那个"最怕被问的问题"
- **自信锚点**：认知翻转那一刻要果断有力

### 结构

```
🎬 【0:00-0:03】 颠覆性观点
   嘴替法：替他说出他最想反驳的那个观点
   "你是不是也觉得安利就是卖产品的？"
   "我告诉你，安利真正赚钱的人，都不是在卖产品"

🎬 【0:03-0:10】 制造好奇心
   情绪法：停顿 + 放慢
   "是不是很意外？我告诉你真相…"

🎬 【0:10-0:35】 解释/论证
   自信法：三段递进，不要并列
   "真正做起来的人都在做这三件事…"

🎬 【0:35-0:45】 结论强化
   "所以不要再[错误做法]了"

🎬 【0:45-0:50】 CTA
   "你被问过这个问题吗？评论区聊聊"
```

## 模板五：营养科普型

### 信息
- **时长**：60-90秒
- **对应情绪曲线**：曲线D（焦虑→病因诊断→认知反差→给方案→温暖收尾）
- **嘴替重点**：替他说出"我到底吃没吃够"的困惑
- **自信锚点**：自媒体教练式算账说理，数据要准

### 结构

```
🎬 【0:00-0:05】 场景切入
   嘴替法：替他说出日常困惑
   "[常见健康问题]的人，你是不是也有这样的困扰？"

🎬 【0:05-0:10】 问题本质
   情绪法：揭示他不知道的真相
   "其实这个问题背后的原因是什么…"

🎬 【0:10-0:40】 营养方案
   自信法：算账式科普，让他对号入座
   "我建议你补充这三样东西…第一…第二…第三…"

🎬 【0:40-0:55】 原理说明
   "为什么这个有用？因为…"

🎬 【0:55-1:00】 总结 + CTA
   "想定制个人营养方案，私信我你的情况"
```

## 模板六：答疑解惑型

### 信息
- **时长**：45-60秒
- **对应情绪曲线**：曲线F（提问→直接回答→展开解释→建议收尾）
- **嘴替重点**：替他说出困扰已久的问题
- **自信锚点**：直接干脆，不绕弯子

### 结构

```
🎬 【0:00-0:05】 提问开场
   嘴替法：替他说出想问但没问出口的问题
   "经常有人问我：'[具体问题]？'"

🎬 【0:05-0:15】 直接回答
   自信法：开头就给出明确答案，不绕弯
   "我的答案是：[核心观点]"

🎬 【0:15-0:40】 展开解释
   情绪法：三段递进解释
   "为什么？因为三个原因…第一…第二…第三…"

🎬 【0:40-0:50】 建议/方案
   "所以我建议你…"

🎬 【0:50-0:55】 CTA
   "还有别的问题？评论区告诉我"
```

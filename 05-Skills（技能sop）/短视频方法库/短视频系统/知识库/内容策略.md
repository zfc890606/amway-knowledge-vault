## 📚 新增认知框架型选题（2026-07-14）

> 灵感来源：《人类简史》5张知识卡片 → 转化为创业者/安利从业者的认知颠覆型内容

### 核心策略

用"人类学视角"做差异化——自媒体教练讲实操（怎么做），付后坚讲系统（体系是什么），你可以讲**认知底层**（为什么你的大脑决定了你会在这个行业成功或失败）。

你的差异定位：
- 自媒体教练 → 实操教练
- 付后坚 → 系统架构师
- **你 → 认知拆解师**（帮他们理解"为什么我会卡在这里"）

### 新增选题池

| 选题 | 对应概念 | 视频类型 | 核心句子（钩子） |
|------|---------|---------|----------------|
| 「做了3年没结果？不是你不够努力，是你不知道"承认无知"才是第一步」 | [[KC-认知-005-承认无知的力量]] | 认知颠覆型 | "你以为你知道安利怎么做？我知道你不知道" |
| 「你囤货、买课、拼命干——是不是掉进了'奢侈品的陷阱'？」 | [[KC-认知-004-奢侈品的陷阱]] | 认知颠覆型 | "你现在的努力，可能正在套牢你" |
| 「怕被拒绝？因为你把自己当推销员，没当'讲故事的人'」 | [[KC-认知-002-虚构故事的力量]] | 认知颠覆型 | "你不需要卖产品，你只需要讲个故事" |

### 差异化价值

🔥 你的优势：别人讲"怎么做"，你可以讲**"为什么你会这样做"**——这不是鸡汤，是人类学+认知科学。

```
自媒体教练的内容：卡在哪里 → 用什么方法解决（what + how）
你的内容：为什么会卡在这里 → 你的大脑怎么决定的 → 怎么绕过这个认知陷阱（why → how）
```

### 风格适配

这组选题全部走[[脚本模板#模板四-认知颠覆型]]（45-60秒），配合[[脚本风格系统]]的"嘴替"开场法：
- 开头先替观众说出他们心里最困惑的那句话
- 中间用一个颠覆性的认知（来自《人类简史》）翻转他的视角
- 结尾回到他的创业场景，告诉他"所以你可以这样用"

#### 每条视频的通用结构（认知框架型专属）

```
【0:00-0:05】 替观众说心里话（嘴替法）
   "你是不是也[常见困境/困惑]？"

【0:05-0:10】 抛出认知反差
   "我今天告诉你一个真相，可能跟你想的不一样"

【0:10-0:35】 用《人类简史》概念拆解
   "人类学家发现……/赫拉利在《人类简史》里说……"
   "简单来说就是……"
   "放到你的创业里，这意味着什么？"

【0:35-0:45】 回到他的场景，给行动建议
   "所以你应该怎么做？第一……第二……第三……"

【0:45-0:50】 一句话收尾 + CTA
   "你现在卡在哪一步？评论区告诉我，我帮你看看"
```

> ⚠️ **注意**：不要把视频做成"讲书"。目标是**用书里的认知框架帮观众理解他们自己的困境**——书只是一个引子，核心是"你的事"。

### 维度二：创业 / 个人成长

#### 概念① 承认无知 → 创业第一性原理

**对你的意义：** 创业最大的坑不是"做错了"，是"不知道自己不知道"。

**实操三个动作：**

| 什么时候用 | 具体做法 |
|-----------|---------|
| **进入新领域** | 先承认自己是小白 → 花一周做调研（看评论区、搜搜索量、问有结果的人）→ 再动手 |
| **遇到瓶颈** | 停下来问："我卡在这里，是因为我不知道什么？" → 去找答案，而不是更努力 |
| **做内容前** | 先承认"我不知道观众真正想要什么" → 去看对标评论区 → 再写脚本 |

> **自检口诀**：你最需要的能力不是"知道"，是"快速承认不知道+快速去弄懂"。

#### 概念② 虚构故事 → 品牌和信任

**对你做安利/大健康的启发：**

```
你以为你在 → 卖产品、卖方案
实际上你在 → 讲一个"这个人可信"的故事
客户买的不是 → 营养品/创业机会
客户买的是 → "跟着这个人走不会错"的相信

所以你的工作不是"把产品讲清楚"
而是"让观众觉得你值得信任"
```

**信任建立的4个层次**（从弱到强）：
1. **专业**：你懂行 → 用数据、案例证明
2. **真实**：你不装 → 分享自己的踩坑经历
3. **一致**：你前后一致 → 说的和做的对得上
4. **利他**：你真心帮人 → 不急于成交，先解决问题

#### 概念③ 奢侈陷阱 → 审视你的"努力"

**对你的意义：** 做安利/创业最危险的不是不努力，是**努力错了方向还不自知**。

**自检问题**（每周一次）：
```
① 我这一周的"努力"，有多少是真正在推动生意？
② 有多少是在"感觉很忙但没产出"？
③ 我有没有在做一件"开始觉得是好事，现在感觉被套住"的事？
   （囤货？买系统？跟某个项目？）
```

#### 概念④ 消费合谋 → 区分"投资"和"消费"

**对你最有用的判断标准：**

```
买课/买系统/买工具前，问自己：
  
投资（✅ 可以买）：买了之后我会认真执行
消费（⚠️ 谨慎）：买完之后大概率放着吃灰
  
真相：
买课不能让你改变
执行才能让你改变
课程只是"改变的工具"，不是"改变本身"
```

### 维度三：带团队 / 团队管理

#### 概念 邓巴数 → 团队不同阶段的管法

| 团队规模 | 管理方式 | 重点做什么 | 危险信号 |
|:--------:|---------|-----------|---------|
| **1-5人** | 自己干+带 | 自己做榜样，带着他们干 | 光说不练 |
| **5-15人** | 人情+带教 | 手把手教，多沟通多吃饭 | 顾不上每个人 |
| **15-50人** | 规则+分工 | 写清楚流程，明确谁负责什么 | 开始出现沟通偏差 |
| **50-150人** | 制度+文化 | 建立制度，但也要反复讲"我们是谁" | 人情和制度打架 |
| **>150人** | 文化+故事 | 靠使命愿景凝聚，制度变成基础设施 | 文化稀释，各自为政 |

> **对你的参考：** 做安利带团队，大多数人卡在15-50人这个阶段——从"靠人情"到"靠制度"的转型最痛苦。

**核心建议：**
1. 团队>10人 → 开始写标准流程（SOP）
2. 团队>30人 → 找一个"二把手"，别自己管所有人
3. 团队>50人 → 定期讲"我们的故事"（成功的案例、团队的历史）

#### 概念 虚构故事 → 团队凝聚力

**带团队的本质是**：让大家共同相信一个"我们能成"的故事。

**实操做法：**

```
每天做的事：
- 多讲成功案例（"你看小王也做到了"）
- 多讲"为什么做"（而不是"怎么做"）
- 让伙伴感觉到"我是这个群体的一员"

避免做的事：
- 只讲制度不讲愿景（伙伴会觉得自己是工具）
- 只说问题不说希望（伙伴会想放弃）
- 只批评不鼓励（伙伴会失去信心）

核心：反复讲"我们是谁、我们去哪、为什么能到"
```

#### 概念 主体间现实 → 团队中的"默认共识"

每个团队都有自己**默认相信的东西**（主体间现实），这个东西决定了团队的文化：
- "我们团队是互帮互助的" → 大家相信这个 → 真的会互相帮忙
- "领导人不管我们的" → 大家信了这个 → 真的没人管了

**建设团队文化的本质**：在团队中建立一个有利于成长的主体间现实。

## 🔑 核心打法策略

### 1. 「先共情，后给方案」
起步期的每条视频，前5秒必须让用户觉得：
> "天哪，说的就是我！"
→ 再用干货把他留住

### 2. 「每条视频都是试用装」
不追求单条爆款，追求每一条都能吸引精准用户来私信你
→ 每条视频结尾都要有明确的行动引导

### 3. 「初期重搜索，轻推送」
因为你是新号，平台不会给你推送流量
→ 重点做搜索优化：标题+文案中融入真实搜索词
→ 坚持发，3个月后搜索流量会积累起来

### 4. 「不做爆款，做资产」
博主的原话：推送流量半个月就没了，搜索流量一直有一直有
→ 不追热点，专注做有长期搜索价值的痛点内容

| 继续还是换方向？ | 对照 [[四阶自检地图]] 决策矩阵 |

### 止损决策标准

```
拍了3条数据都差 → 判断：
  ① 别人讲同主题能火吗？
     能 → 是我的表达/呈现问题 → 练表达，不换方向
     不能 → 是这个主题没流量 → 换方向

拍了10条数据都差 → 判断：
  ① 是不是选错了平台？→ 试试其他平台
  ② 是不是目标人群不对？→ 换人群定位
  ③ 真的不行了 → 止损换赛道
```

> **止损不是不拍了，是换方向、换选题、换形式，继续试。**

## 🆕 新增认知框架：审美×AI — AI时代创业者的终极护城河（2026-07-15）

> 来源：《审美之外无他物：AI时代生存圣经》刘克亮 → 新增5个痛点(#51-#55) + 3张知识卡片

### 核心论点（一句话说清）

**AI消除了执行门槛后，人类唯一的竞争力 = 审美（反思性判断力）。** 这不是艺术鉴赏力，而是在没有标准的地方建立标准、在无数选项中做判断的能力。

### 为什么这个框架对你做AI+大健康短视频特别有用？

| 你的优势 | 这本书怎么放大这个优势 |
|---------|---------------------|
| 你懂大健康专业知识 | AI把执行变成公共基础设施，你的专业知识第一次有了杠杆——不需要学代码，你的判断力就是最稀缺的资源 |
| 你正在做短视频 | 双飞轮模型：AI负责内容生成的执行（快×多×省），你负责判断什么是好的内容（方向×标准×品味） |
| 你的观众在焦虑AI | 他们怕被AI替代——你告诉他们AI替代的是执行力，不是判断力。懂健康的人反而更值钱了 |

### 你的差异化定位升级

已有定位：认知拆解师（"为什么你会卡在这里"）

新增维度：**AI时代的价值判断师**（"为什么你不需要害怕AI，以及什么才是你现在最该练的能力"）

| 定位 | 区别 |
|------|------|
| 自媒体教练 → 实操教练（怎么做安利） |
| 付后坚 → 系统架构师（体系怎么搭） |
| 林博士 → 心态×方法三层派（怎么成长） |
| **你 → 认知拆解师 + AI时代价值判断师**（为什么你的大脑决定了成败 + 为什么AI时代懂健康的更值钱） |

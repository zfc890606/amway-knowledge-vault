## 💡 博主方法论 · 一句话记忆

```
本质    →  创造稀缺 + 放大情绪（两个核心原理）
开头    →  九大人性钩子（视觉/赚钱/恐惧/愤怒/八卦/猎奇/高价值/炫富/热点）
人设    →  反差人设（让观众猜不透你）
选题    →  高频痛点（评论区就是选题库）
结构    →  痛点 → 方案 → [原因过滤] → 一句话总结
语言    →  不高大上，人们怎么说你就怎么说（芒果TV不是CCTV）
流量    →  去用户多的地方"偷听"（不是当专家，是当自己人）
迭代    →  数据说话 → 每月复盘 → 自我进化
心法    →  先让自己成为答案，别人自然来找你
系统    →  知识卡片四要素 = 短视频四段结构（卡片即脚本）
```

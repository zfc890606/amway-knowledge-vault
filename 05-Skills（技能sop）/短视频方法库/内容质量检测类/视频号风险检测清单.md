# 🔴 视频号内容风险检测清单

> **适用范围：** 安利 × 大健康 × 互联网创业类短视频内容
> **依据：** 2026年微信视频号最新规则（金融行业公约/保健食品管理规则/功效造假治理细则等）
> **每次发视频前，过一遍这个清单。有一条不过就别发。**

---

## 一、健康功效类（⚠️ 最高风险区）

视频号2026年重点打击的就是**虚假宣传功效**，这是红线中的红线。

### ❌ 绝对不能说

| 违规类型 | 典型违规词 | 风险等级 |
|---------|-----------|:-------:|
| **宣称医疗功效** | 治病、消炎、降压、降糖、治疗、根治、痊愈、预防疾病、代替药品 | 🔴 封号 |
| **宣称保健功效（超出批准范围）** | 减肥、瘦身、排毒、抗癌、疏通血管、修复细胞、改善睡眠（普通食品说这些不行） | 🔴 封号 |
| **绝对化功效承诺** | 100%有效、奇效、一吃见效、三天见效、无效退款 | 🟠 限流/下架 |
| **制造健康焦虑** | "你的细胞在饿""隐性饥饿""悄悄毁掉你的健康"——这类话术需谨慎，可能被判定为制造焦虑 | 🟠 限流 |
| **代替药物暗示** | "吃了就不用吃药了""比吃药安全" | 🔴 封号 |

### ✅ 安全说法

| ❌ 危险 | ✅ 安全 |
|--------|--------|
| "降血糖降血压" | "日常饮食搭配参考" |
| "排毒减肥" | "体重管理、均衡膳食" |
| "调理身体"（普通食品慎用） | "日常保养、营养补充" |
| "治愈了我的XX病" | "我自己吃了之后感觉精力好了很多"（只讲个人感受，不承诺疗效） |
| "蛋白质能修复细胞"（可能被判定为伪医学） | "蛋白质是人体必需的营养素之一"（讲通用营养学常识） |

### 安全区指南

```
✅ 可以说：
  - 通用营养学常识（"蛋白质是人体必需的七大营养素之一"）
  - 个人使用感受（"我吃了之后感觉精力好了"）
  - 产品成分介绍（但不与疾病关联）

❌ 不能说：
  - 任何与"治疗疾病"挂钩的表述
  - 暗示产品可以代替药物
  - 制造健康焦虑后立即推荐产品（会被判定为违规营销）
```

## 二、绝对化用语类（容易忽略但违规）

| ❌ 违规 | ✅ 合规替换 |
|--------|-----------|
| 最、第一、全网第一、No.1、Top1 | 优质、精选、热门、口碑款 |
| 顶级、极致、完美、万能 | 高品质、受欢迎、实力推荐 |
| 唯一、首选、冠军、天花板 | 人气款、热门推荐 |
| 绝对、百分百、无敌 | 非常、很、相当 |

## 三、收入承诺类（直销/创业类内容的重灾区）

你的内容涉及"创业""副业""收入"，这是视频号金融类规则的敏感区。

### ❌ 绝对不能说

| 违规类型 | 典型违规词 | 风险 |
|---------|-----------|:----:|
| **收入承诺** | 月入过万、月入五万、年入百万、轻松月入 | 🟠 限流/下架 |
| **稳赚不赔** | 稳赚、躺赚、保本、零风险、只赚不赔 | 🔴 封号 |
| **暴富诱导** | 暴富、一夜暴富、财富自由、快速致富 | 🟠 限流 |
| **诱导加盟** | 跟我干月入XX、招代理、拉人头 | 🔴 封号 |

### ✅ 安全说法

| ❌ 危险 | ✅ 安全 |
|--------|--------|
| "我做安利月入三万" | "这是我个人的经历分享"（加声明） |
| "跟我做，月入过万不是梦" | "我把我的方法分享给你，你可以参考" |
| "稳赚不赔的副业" | "一个值得了解的创业机会" |
| "年入百万"（具体数字） | "收入上了一个新台阶"（模糊表述） |

### 安全话术指南

```
❌ "我团队里那些月入5万的人" → 可能被判定为诱导收入承诺
✅ "我团队里那些做得不错的人，他们的方法有一个共同点"

❌ "做安利不用囤货不用刷屏，一个月多赚5000" → 具体收入数字
✅ "做安利不用囤货不用刷屏，做好了自己能看得到回报"

核心原则：不讲具体收入数字，不讲"保证你能赚多少"
```

## 四、引流导流类（视频号严查）

| ❌ 违规 | 说明 |
|--------|------|
| 视频中展示个人微信/QQ/电话 | 可在主页留"视频号小店"入口，不能直接留微信号 |
| 评论区引导"加我微信" | 会被判定为违规导流 |
| 用"领资料"诱导加私域 | 变相引流也会被处罚 |
| 私信群发广告 | 频繁被举报会导致功能受限 |

### ✅ 合规做法

私信引导要留**在视频号生态内**：
- ✅ "私信我，我发链接给你"（指向视频号小店）
- ✅ "关注我，下条告诉你"（留在平台内）
- ✅ 主页可以挂"视频号小店"入口

## 五、内容创作红线（2026年新规）

| 规则 | 说明 |
|:----:|------|
| 🤖 **AI生成需标注** | 用AI生成的文案/画面，需要在显著位置标注"本内容由AI生成/辅助生成" |
| 🎬 **演绎需注明** | 如果是摆拍/情景演绎，需标注"剧情演绎，仅供参考" |
| 📋 **同质化降权** | 7天内发布超过3条结构/脚本高度相似的内容，不推流 |
| 📦 **搬运零容忍** | 搬运他人原创内容，多次违规永久封号 |
| 🔄 **二次审核** | 发布时通过≠永久安全，历史内容可能被重新审核下架 |

## 六、安利/直销类特殊注意事项

结合你的内容方向（安利 × 大健康 × 互联网创业），有几条特殊的红线：

| 场景 | 风险 | 建议 |
|:----:|:----:|:-----|
| **讲"做安利赚钱"** | 触达"收入承诺"和"诱导加盟"两条红线 | 不讲具体数字，讲方法讲逻辑 |
| **讲"产品效果"** | 触达"健康功效违规"红线 | 只讲个人感受，不讲治疗功效 |
| **引导"加我了解"** | 触达"违规引流"红线 | 引导到视频号小店或留在评论区 |
| **展示团队收入** | 触达"多层分销"敏感区 | 慎用，最好模糊化 |
| **讲"直销vs传销"** | 本身不违规，但措辞不当会被判定为争议内容 | 用客观事实，不要激烈辩论 |

## 七、发布前自检清单（3分钟过一遍）

### 🔴 高风险区（有一条不过就别发）

- [ ] 有没有出现"治病、降糖、降压、治疗"等医疗功效词？
- [ ] 有没有出现"无效退款、100%有效、一吃见效"等绝对化功效承诺？
- [ ] 有没有出现"月入过万、年入百万、稳赚、躺赚"等收入承诺词？
- [ ] 有没有引导用户加个人微信/QQ/电话？

### 🟠 中风险区（能改就改，改不了就标注免责声明）

- [ ] 有没有"最、第一、顶级、完美、唯一"等极限词？
- [ ] 有没有制造健康焦虑后立即推荐产品？（容易被判违规营销）
- [ ] 有没有"调理身体、排毒、改善睡眠"等功效暗示？（普通食品不能说）
- [ ] 有没有具体收入数字？（最好模糊化）

### ✅ 通过标准

- [ ] 功效类用语已替换为合规说法
- [ ] 收入部分已替换为"个人经历分享"等模糊表述
- [ ] 无引导私域引流的内容
- [ ] 如有AI生成内容，已标注
- [ ] 如为演绎内容，已标注

## 八、常用合规替换表（速查）

| 场景 | ❌ 危险说法 | ✅ 安全说法 |
|------|-----------|-----------|
| 讲产品营养 | "这个能改善你的睡眠" | "我自己吃了之后睡眠质量有提升" |
| 讲创业收入 | "跟我做月入三万" | "我把我的方法分享给你参考" |
| 讲健康 | "脂肪肝这样吃就能好" | "脂肪肝的日常饮食可以参考这个搭配" |
| 讲产品 | "这个产品是最好的" | "这个产品我自己用下来觉得不错" |
| 讲机会 | "稳赚不赔的创业机会" | "一个值得了解的创业方向" |
| 讲效果 | "吃了三天就见效" | "我坚持吃了三个月，感觉精力好了很多" |

---

> **一句话原则：** 讲个人经历可以，讲普遍结论不行；讲方法可以，讲承诺不行；留在平台内可以，引流到私域不行。

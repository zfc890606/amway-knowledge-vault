# 📕 小红书内容风险检测清单

> **适用范围：** 安利 × 大健康 × 互联网创业类内容（图文笔记/视频）
> **依据：** 小红书2026社区公约2.0 + 2026违禁词全面更新版
> **每次发小红书前，过一遍这个清单。有一条不过就别发。**

---

## 一、绝对化极限词（最高频违规）

小红书2026年AI审核语义识别大幅升级，谐音字、符号替换、空格拆词全都能识别。

| ❌ 违规 | ✅ 合规替换 |
|--------|-----------|
| 最、最佳、最好、最高级、最低价 | 优质、精选、热门、口碑款、很、相当 |
| 第一、唯一、首个、首家、全网首发 | 热门、人气、推荐、优选 |
| 顶级、顶尖、极致、极品、终极 | 高品质、受欢迎、实力推荐 |
| 国家级、世界级、全球级、质量免检 | 直接去掉，或用具体事实替代 |
| 永久、万能、100%、绝对、超强 | 长期、很多、非常、持续 |
| 王牌、冠军、领袖品牌、天花板 | 口碑好、人气高、很多人选 |
| 史无前例、绝无仅有 | 很少见、不容易、不多 |

### 安利内容特别提醒

| ❌ 危险 | ✅ 安全 |
|--------|--------|
| "最好的创业机会" | "一个值得了解的创业方向" |
| "最有效的营养方案" | "我自己用下来觉得不错的方案" |
| "全网第一的XX" | "口碑很好的一款产品" |
| "独一无二的模式" | "一种比较特别的模式" |

## 三、医疗健康类敏感词（重点！）

2026年小红书对健康类内容监管升级，不仅仅是药品，**营养品、保健品、食品的功效描述**也严查。

### ❌ 绝对不能说

| 违规类型 | 典型违规词 | 风险等级 |
|---------|-----------|:-------:|
| **宣称医疗功效** | 治疗、治愈、消炎、杀菌、抗菌、排毒、解毒、处方 | 🔴 封号 |
| **疾病名称关联** | 痤疮、皮炎、脚气、湿疹、荨麻疹（直接说产品能治） | 🔴 封号 |
| **减肥瘦身类** | 减肥、溶脂、吸脂、燃烧脂肪、瘦身、瘦脸、减脂 | 🟠 限流/下架 |
| **医美类** | 祛疤、丰胸、去皱纹、修复弹性纤维 | 🟠 限流 |
| **激素/药物类** | 激素、抗生素、临床有效、临床试验 | 🔴 封号 |

### ✅ 安全说法

| ❌ 危险 | ✅ 安全 |
|--------|--------|
| "降血糖降血压" | "日常饮食搭配参考" |
| "排毒减肥" | "体重管理、均衡膳食、身材管理" |
| "治愈了我的XX病" | "我自己坚持调整饮食后，体检指标有改善" |
| "调理身体"（普通食品慎用） | "日常保养、营养补充" |
| "细胞修复" | "营养均衡对身体有好处" |

### 安全原则

```
✅ 可以说：
  - 通用营养学常识（"蛋白质是人体必需的七大营养素之一"）
  - 个人真实感受（"我吃了之后感觉精力好了很多"）
  - 产品成分介绍（但不与疾病关联）

❌ 不能说：
  - 任何与"治疗疾病"挂钩的表述
  - 暗示产品可以代替药物
  - 制造健康焦虑后立即推荐产品（违规营销）
  - 用疾病名称+产品功效绑定描述
```

## 四、金融与收入敏感词（创业/副业类高危）

你的内容方向（安利创业）涉及收入话题，小红书对此类内容审核极严。

### ❌ 绝对不能说

| 违规类型 | 典型违规词 | 风险等级 |
|---------|-----------|:-------:|
| **具体收入数字** | 月入过万、月入X万、年入百万、日入XXX | 🟠 限流/下架 |
| **稳赚不赔** | 稳赚、躺赚、保本、零风险、只赚不赔 | 🔴 封号 |
| **暴富诱导** | 暴富、一夜暴富、财富自由、快速致富、钱多事少 | 🔴 封号 |
| **诱导加盟** | 招代理、拉人头、发展下线、团队裂变 | 🔴 封号 |
| **金融理财** | 收益率、回报率、保本理财、荐股、内幕消息 | 🔴 封号 |
| **副业承诺** | 副业月入过万、轻松搞钱、在家就能赚 | 🟠 限流 |

### ✅ 安全说法

| ❌ 危险 | ✅ 安全 |
|--------|--------|
| "我做安利月入三万" | "这是我个人的经历分享"（加声明） |
| "跟我做，月入过万不是梦" | "我把我的方法分享给你参考" |
| "稳赚不赔的副业" | "一个值得了解的创业机会" |
| "年入百万"（具体数字） | "收入上了一个新台阶"（模糊表述） |
| "招代理，月入X万" | "有感兴趣的朋友可以一起交流" |

### 安全话术核心原则

```
❌ 不讲具体收入数字
❌ 不讲"保证你能赚多少"
❌ 不给"稳赚"承诺
✅ 讲方法、讲逻辑、讲个人经历
✅ 用"分享"替代"招人"
✅ 用"参考"替代"保证"
```

## 五、导流/引流类（最易封号）

小红书对私域引流是**零容忍**，是封号率最高的违规类型。

### ❌ 绝对不能说

| 违规内容 | 说明 |
|---------|------|
| **微信/VX/WeChat** | 任何变体都不行（威信、V、微❤️等都会被识别） |
| **手机号/QQ/邮箱** | 直接展示联系方式 |
| **二维码/小程序码** | 图片里出现也不行 |
| **私信领取/加我进群** | 变相引流 |
| **其他平台名称** | 淘宝/猫、京东/狗东、抖音/某音、拼多多/多多等代称也会被识别 |
| **小红书谐音引流** | 小清单、小书店（特定场景可用，但慎用） |

### ✅ 合规做法

```
✅ "在评论区告诉我你的想法"（留在平台内）
✅ "收藏这篇笔记，方便以后找"（平台内互动）
✅ 主页可以挂小红书店铺/商品链接
✅ "私信我"（小红书私信功能是合规的）
✅ "关注我，下期分享XX"（留在平台内）
```

## 八、笔记特殊注意事项

小红书是**图文种草平台**，除了内容合规，还有格式层面的规则：

### 标题规范
```
✅ 15-20字，含关键词，引发好奇心
❌ 标题中出现绝对化词、敏感词（触发风控比正文更严）
✅ 示例：「失败是成功之母，那成功之父呢？我被导师问住了」
```

### 正文排版
```
✅ 每段3-5行，段落间空行
✅ 多用Emoji分段（📌 💡 ✨ ✅ ❌）
✅ 关键句单独成行，增加可读性
❌ 大段文字无分段（移动端阅读体验差）
```

### 话题标签
```
✅ 每篇3-5个标签，精准命中搜索流量
✅ 标签层级：大领域+细分+长尾
✅ 示例：#创业感悟 #个人成长 #人生道理
```

### 封面图
```
✅ 封面是点击率的关键
✅ 文字直接点出主题/金句
✅ 示例：封面大字"成功之父是谁？"
```

- [ ] 有没有出现"100%有效、无效退款、彻底根治"等虚假承诺？
- [ ] 有没有出现"月入过万、年入百万、稳赚、躺赚"等收入/暴富词？
- [ ] 有没有出现微信/手机号/二维码等导流信息？
- [ ] 有没有出现"招代理、拉人头、发展下线"等词？
- [ ] 有没有"最、第一、顶级、完美、唯一"等极限词？

### 🟠 中风险区（能改就改）

- [ ] 标题是否有敏感词？（标题触发风控比正文更严）
- [ ] 是否制造健康焦虑后立即推荐产品？

- [ ] 功效类用语已替换为个人感受/通用常识

- [ ] 无任何导流信息（微信/手机号/其他平台）
- [ ] 无极限词、绝对化词
- [ ] 笔记结构适合小红书图文阅读（分段+Emoji+标签）
- [ ] 如有AI生成/辅助内容，已标注

## 十、常用合规替换表（速查）

| 场景 | ❌ 危险说法 | ✅ 安全说法 |
|------|-----------|-----------|
| 讲产品营养 | "这个能改善睡眠" | "我吃了之后感觉睡眠质量有提升" |
| 讲创业收入 | "跟我做月入三万" | "我把我的方法分享给你参考" |
| 讲健康 | "脂肪肝这样吃就能好" | "脂肪肝的日常饮食可以参考这个搭配" |
| 讲产品 | "这个产品是最好的" | "这个产品我自己用下来觉得不错" |
| 讲机会 | "稳赚不赔的创业机会" | "一个值得了解的创业方向" |
| 讲效果 | "吃了三天就见效" | "我坚持了三个月，感觉精力好了很多" |
| 招人 | "招代理，跟我干" | "有感兴趣的朋友可以一起交流" |
| 引流 | "加我微信领资料" | "收藏这篇，需要的时候好找" |

---

> **一句话原则：** 个人感受可以说，普遍结论不能说；讲方法可以，讲承诺不行；留在平台内可以，引流到私域不行；AI生成要标注，不标就是违规。
>
> **小红书 vs 视频号差异：** 小红书对导流零容忍（封号风险最高）、对收入数字更敏感、对同质化检测更严。但健康功效类相对视频号略宽松（仍不可宣称治疗）。

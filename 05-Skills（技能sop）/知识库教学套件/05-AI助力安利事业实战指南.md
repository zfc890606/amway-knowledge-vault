## 快速开始：你现在就能用的 3 个场景

> 以下 3 个场景不需要安装任何工具，打开 Claude Code（或 DeepSeek）直接对话即可。

### 场景 1：写一篇朋友圈

```
帮我写一条朋友圈文案，主题是：
我今天帮一个客户做了营养早餐搭配，她坚持了 3 周，今天说精力好多了。

要求：
- 不要像广告
- 用真实对话的感觉写
- 100 字以内
```

### 场景 2：回答客户的一个营养问题

```
客户问我："我有脂肪肝，能吃蛋白粉吗？"
请你帮我写一段回答：
- 先说结论（能不能吃）
- 再说原因（为什么）
- 最后给建议（怎么吃）
- 用大白话，不用说太多专业术语
```

### 场景 3：做一次周复盘

```
帮我做一次本周复盘。
我这周做了这些事：
1. 发了 2 条视频
2. 跟进了 5 个客户
3. 参加了一次团队会议

帮我分析：
- 本周我做得好的 3 点
- 下周可以改进的 2 点
- 下周最该聚焦的一件事
```

> 试一下，看看 AI 的回答。不满意就继续追问，像跟一个助手聊天一样。

## 场景一：写短视频脚本

这是 AI 最擅长的场景之一。你不需要从零开始写稿。

### 用法 1：给选题，AI 出骨架

```
帮我写一条短视频脚本，主题是：
安利新手总是被拒绝，怎么办？

要求：
- 60 秒口播
- 开头上来就扎心，不要铺垫
- 结构：扎心 → 我经历过 → 换个角度看 → 给方法
- 结尾加一句互动的话
```

### 用法 2：给参考素材，AI 写完整稿

```
根据下面这段经验和知识，帮我写一条口播脚本：

[粘贴你认为有价值的素材——比如一段听课笔记、一篇公众号文章]

请把这变成一条 60 秒的口播稿。用口语，不要书面语。
开头三秒就要让人停下来。
```

### 用法 3：优化你的初稿

```
帮我改一下这段脚本，让它更像人话：

[粘贴你的初稿]

具体：
1. 去掉书面表达，改成口语
2. 去掉"首先/其次/最后"这类连接词
3. 每句话不超过 15 个字
4. 加一些"你"——让观众觉得在对他说话
```

> **关键心法**：AI 产出初稿，你加入真实经历修改后才是成品。

## 场景二：回答客户营养问题

客户的问题往往大同小异。用 AI 帮你准备标准答案，你只需要根据每个人微调。

### 常见问题模板

把客户问你的问题收集起来，让 AI 帮你准备回答：

```
我经常被问到这个问题：
"蛋白粉和药物一起吃会不会冲突？"

请帮我准备一个标准回答：
- 先说结论
- 再给简单的科学解释
- 最后给建议
- 200 字以内
- 不用专业术语，大白话
```

### 营养方案初稿

```
客户情况：
- 男，45 岁
- 脂肪肝（轻度）
- 平时工作忙，三餐不规律
- 晚餐经常喝酒
- 想让客户先改变，再来约谈

请你帮我写一段"建立认知"的对话：
先让他意识到严重性，再给他一个简单的调整方案（不改习惯先，只是增加点东西）
不要吓人，要让他觉得"这个我能做"
```

> ⚠️ **注意**：AI 不是医生，不能给出医疗建议。营养方案初稿仅供你参考，最终建议要结合你的专业知识判断。

## 场景三：整理会议记录/听课笔记

开会时没来得及记？或者记了但太乱？让 AI 整理。

### 录音/笔记整理

```
帮我整理这段听课笔记，提炼出 3 个我马上能用的方法：

[粘贴你的笔记内容]

输出格式：
1. 方法一：一句话 + 怎么做
2. 方法二：一句话 + 怎么做
3. 方法三：一句话 + 怎么做
```

### 金句提取

```
从这段内容中，提取 5 句最值得记住的话（金句）：

[粘贴内容]

要求：
- 每句一句话
- 原汁原味，不要改
- 适合发朋友圈或做视频标题
```

## 场景四：写朋友圈/社群文案

### 朋友圈文案

```
帮我写一条朋友圈文案：
今天跟客户聊天，她说用了蛋白粉一个月，睡眠好多了，早上起来不累了。

氛围：真实、温暖、不像广告
长度：80–120 字
附上：一个生活化的结尾，让人觉得你是在分享而不是推销
```

### 社群早安/晚安问候

```
帮我写 3 条早安问候文案，给营养健康社群的：
1. 轻松一点的（不要教条）
2. 带一个健康小知识
3. 每条 50–80 字
```

## 场景五：分析你的客户画像

了解你的目标客户，AI 可以帮你快速分析。

```
帮我分析一下"安利营养品客户"的常见画像：

请从这些维度分析：
1. 他们最关心什么问题（痛点）
2. 他们的年龄/职业/生活方式
3. 他们最容易在哪一步犹豫
4. 什么样的沟通方式最容易让他们接受
```

> 你可以把 AI 的分析结果存到你知识库的 `03-Resources/` 下，以后写文案、写脚本都能参考。

## 场景六：培训新人话术

帮你的伙伴准备应对各种场景的话术。

### 拒绝应对话术

```
帮我准备 5 个常见拒绝场景的回应话术：

1. "我没钱买"
2. "我考虑考虑"
3. "我不需要"
4. "我老婆不让"
5. "我朋友做过，没赚到钱"

每个回应要求：
- 先说表示理解的话
- 再说一个简单的反问/引导
- 不要说太长，30 秒以内
- 不要反驳客户，不要让他觉得你在说服他
```

### 新人第一周话术

```
帮我写一个新人的"第一周工作指南"话术版：
- 第一天：怎么跟身边人开口介绍自己在做安利
- 第三天：怎么邀请朋友了解产品
- 第七天：怎么跟进和确认意向

每个话术 100 字以内，口语化，像是一个有经验的人在教新人。
```

## 场景九：复盘你的事业进展

每周花 5 分钟让 AI 帮你复盘，保持方向不偏。

```
这是我本周的情况：
- 新增客户数：X
- 跟进次数：X
- 成交数：X
- 短视频发了 X 条
- 团队开会 X 次

帮我做复盘：
1. 本周做得最好的 3 件事
2. 本周最浪费时间的 2 件事
3. 下周应该聚焦在哪个方向
4. 对这个月目标的完成度预估（基于本周数据）
```

## 场景十：帮伙伴也用上 AI

当你自己用顺了，可以把 AI 的经验复制给团队成员。

### 给新人准备的 5 个 AI 场景

你可以把下面这段直接发给你团队的新人：

```
> 伙伴你好，AI 不是高科技，就是你的助手。
> 我现在每天都让 AI 帮我做这些事，你也可以试试：

1️⃣ **写朋友圈**：输入"帮我写一条今天帮客户做了营养搭配的朋友圈"
2️⃣ **准备客户回答**：输入"客户问我XXX，该怎么回"
3️⃣ **整理听课笔记**：听完课把笔记粘贴进去，让它提炼要点
4️⃣ **写脚本**：输入"帮我写一条关于XXX的短视频脚本"
5️⃣ **复盘**：输入"帮我复盘这周的工作"

> 秘诀就一句话——想到什么就直接问 AI，像跟一个聪明同事聊天一样。
> 不满意就继续追问，问到满意为止。
```

### 团队 AI 使用约定

如果你要把 AI 分享给团队，建议先定几条规则：

1. **不要让 AI 替你做决策** — AI 是参谋，不是司令
2. **营养健康问题 AI 只是参考** — 最终建议靠专业知识和医生意见
3. **不要输入客户隐私** — 比如全名、身份证、详细病历
4. **发给客户的东西先自己看一遍** — AI 可能会写错，你要把关

## 常见问题

### Q: AI 回答的不准确怎么办？
正常。AI 不是百科全书，它也会有错误。解决办法：
- **追问**："这个结论的依据是什么？"让它给来源
- **纠正**："不对，你重新想，应该是..."告诉它正确信息
- **补充**："再加一个角度..."不断缩小它的回答范围
- 最终：**你是决策者，AI 是参考**

### Q: 我怕 AI 会越来越笨，或者回答不对？
AI 的质量取决于你问的质量。问得越具体，回答越好。试对比：

> ❌ 差的提问："帮我写个文案"
> ✅ 好的提问："帮我写一条朋友圈文案，关于帮客户做了营养搭配获得好评，80字，真实感，不要像广告"

### Q: 用 AI 写的东西会不会没有"人味"？
会的——如果你只复制粘贴不改的话。正确用法：**AI 出骨架，你加血肉**。你加入真实的经历、真实的感受、真实的故事——这些 AI 永远给不了，因为它是 AI，你是人。

### Q: 我只装 DeepSeek 够不够？
日常文案、翻译、总结 DeepSeek 完全够用。但如果你要做深度的内容创作（脚本、分析、知识卡片），Claude Code 更强。建议两都装，按场景分工。

### Q: 怎么教我的伙伴也用 AI？
先教 3 个最简单的场景（写朋友圈、回答客户问题、整理笔记），让他们先用起来。用了觉得好，自然会想学更多。参考场景十的内容直接发给他们。

### Q: AI 工具需要梯子才能用吗？
- Claude Code → 需要（参考 [[04-网络配置教程（梯子）]]）
- DeepSeek → 不需要，国内直连
- ChatGPT → 需要（非必备，Claude Code 替代即可）
- 建议至少有一个国内可用的（DeepSeek）和一个国际的（Claude Code）

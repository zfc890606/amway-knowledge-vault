# 📚 伙伴教学套件 · 总览

> **目标**：帮你的伙伴们从零搭建一套"AI 增强的第二大脑 + 短视频创作系统"，用科技杠杆做好安利事业。
>
> 本套件由 6 个独立模块组成，每个模块可单独阅读，也可以按顺序学习。

| 模块 | 内容 | 适合谁 | 预计耗时 |

## 🗺️ 三条学习路径

根据你的角色和目标，选择适合的路线：

### 路径 A：我想快速上手 AI 做安利（推荐新人）

```
05-AI助力安利事业实战指南  ← 先从这里开始，立竿见影
        ↓
03-工具组装流程教程          ← 需要工具支撑时再装
        ↓
04-网络配置教程（梯子）      ← 卡在网络问题时再配置
```

> **适合**：只想先用 AI 提效，不想一上来折腾知识库和短视频的伙伴。

### 路径 B：我想搭建完整的知识管理系统

```
03-工具组装流程教程          ← 先装好工具
        ↓
04-网络配置教程（梯子）      ← 确保网络通畅
        ↓
01-知识库搭建教程            ← 搭建知识库骨架
        ↓
05-AI助力安利事业实战指南    ← 用 AI 喂知识库
```

> **适合**：想做个人知识管理，积累自己的认知资产的伙伴。

### 路径 C：我想做短视频 + 内容创业

```
03-工具组装流程教程          ← 先装好工具
        ↓
01-知识库搭建教程            ← 知识库是内容的基础
        ↓
02-短视频创作系统搭建教程    ← 搭建生产管线
        ↓
05-AI助力安利事业实战指南    ← AI 贯穿全流程提效
```

> **适合**：想做短视频获客、打造个人 IP 的伙伴。

## 📖 学习建议

1. **别一次性看完**：每个模块按你的节奏来，今天装工具，明天搭知识库，后天写脚本
2. **边看边做**：教程里每一步都有操作指令，跟着做比光看有效 10 倍
3. **卡住很正常**：每个模块末尾都有「常见问题」章节，先自查；还搞不定就问你团队的 AI 顾问
4. **复制模板，不从头造**：教程里提供了可直接复制的配置命令和模板，不需要自己发明
5. **先完成，再完美**：第一天能跑通就行，优化是后面的事

## 工具全景：三个工具各干什么

| 工具 | 主要功能 | 谁需要 |
|------|---------|--------|
| **Obsidian** | 知识管理软件（写笔记、建知识库） | 所有人 |
| **Claude Code** | AI 主力（写脚本、分析问题、做内容） | 需要 AI 写内容/分析 |
| **DeepSeek v4 Pro** | AI 备用（翻译、总结、文字处理） | 需要国内可用 AI 备选 |

### 它们怎么配合

```
你的日常工作
    │
    ├── 用 Obsidian 管理知识和素材
    ├── 用 Claude Code 写稿/分析/回答客户
    └── 用 DeepSeek 做翻译/总结/日常文字处理
         │
         └── 三者共享知识库 → AI 帮你调用知识库内容
```

## 第一步：安装 Obsidian

### Windows 用户

1. 打开浏览器，访问 `https://obsidian.md`
2. 点击 **Download**，选择 Windows 版本
3. 下载安装包（.exe），双击安装
4. 安装完成后打开 Obsidian
5. 选择"新建库"，命名为"知识库"，存在 `C:\Users\你的用户名\Documents`

### Mac 用户

1. 打开浏览器，访问 `https://obsidian.md`
2. 点击 **Download**，选择 macOS 版本
3. 下载 .dmg 文件，打开后拖入 Applications 文件夹
4. 安装完成后打开 Obsidian
5. 选择"新建库"，命名为"知识库"，存在 `~/Documents`

### 验证 Obsidian 安装成功

打开后你会看到 Obsidian 界面。新建一个笔记试试看——按 `Cmd+N`（Mac）或 `Ctrl+N`（Windows）新建一个空白笔记，输入 "Hello World"。

## 第二步：安装 Node.js（Claude Code 的前置条件）

Claude Code 需要 Node.js 环境来运行。

### 方法一：官网下载（推荐新手）

1. 访问 `https://nodejs.org`
2. 下载 **LTS 版本**（左侧写着"长期支持"的版本，不要下载右侧的"最新版"）
3. 双击安装包，一路点"下一步"完成安装

### 方法二：命令行安装

```bash
# Mac（使用 Homebrew）
brew install node

# 或者 Windows（使用 winget）
winget install OpenJS.NodeJS.LTS
```

### 验证 Node.js 安装成功

打开终端（Mac）或命令提示符（Windows），输入：

```bash
node --version
npm --version
```

如果看到版本号（例如 `v18.17.0` 和 `10.8.1`），说明安装成功。

## 第三步：安装 Claude Code（Claudian 插件）

Claude Code 有两种使用方式：作为 Obsidian 插件（推荐）和作为命令行工具。

### 方式一：Obsidian 插件（推荐大多数人）

1. 打开 Obsidian → 设置（左下角齿轮图标）
2. 点击"第三方插件" → "社区插件市场"
3. 搜索 **Claudian**（插件 ID: `realclaudian`）
4. 点击"安装"，安装后启用
5. 进入 Claudian 设置页，配置你的 API Key

### 方式二：命令行工具（推荐技术人员）

如果 Obsidian 插件方式不合适，也可以直接用命令行安装：

```bash
npm install -g @anthropic-ai/claude-code
```

安装后，在终端输入 `claude` 即可进入交互界面。

### 获取 API Key

Claude Code 需要 API Key 才能工作：

1. 访问 `https://console.anthropic.com/`
2. 注册账号（需要邮箱 + 手机号验证）
3. 登录后进入 API Keys 页面
4. 点击 **Create Key**，复制生成的 Key
5. 粘贴到 Claudian 设置页的 API Key 输入框

> **注意**：API Key 是敏感信息，不要发给别人，不要上传到公开网络。

### 验证 Claudian 安装成功

在 Obsidian 中，你应该能在右侧或底部看到 Claudian 的面板。输入一句：

```
你好，我的知识库在哪？请列出根目录的文件。
```

如果 AI 能回应你，说明安装成功。

## 第四步：配置 DeepSeek v4 Pro

DeepSeek 作为国内可用的 AI 备选，在你需要中文场景下的文字处理（翻译、总结、润色）时非常有用。

### 1. 注册 DeepSeek

1. 访问 `https://chat.deepseek.com/`
2. 注册账号（手机号即可，国内可用）
3. 登录后在设置中找到 API Keys 页面
4. 生成一个 API Key 并复制保存

### 2. 在 Obsidian 中使用 DeepSeek（通过 Copilot 插件）

1. Obsidian 设置 → 社区插件市场 → 搜索并安装 **Copilot** 插件
2. 在 Copilot 设置中配置 DeepSeek 的 API：
   - API 地址：`https://api.deepseek.com`
   - API Key：粘贴你刚复制的 Key
   - 模型：选择 `deepseek-chat` 或 `deepseek-v4-pro`（根据你的订阅版本）

### 3. 验证 DeepSeek 可用

在 Obsidian 的 Copilot 面板中提问：

```
帮我总结一下这篇文章的核心观点
```

如果能正常回复，说明配置成功。

### DeepSeek vs Claude Code：分工建议

| 任务 | 用哪个 |
|------|--------|
| 写脚本、创作内容 | Claude Code（更擅长创造） |
| 翻译外文资料 | DeepSeek（更便宜、中文好） |
| 分析营养数据 | Claude Code（更深度） |
| 整理会议记录 | DeepSeek（更快速） |
| 回答客户营养问题 | Claude Code + 知识库配合 |
| 日常润色、改写 | 两都可，看哪个响应快 |

## 第五步：初始化 CLI 环境（Mac 用户）

如果你使用 Mac，建议安装 **Homebrew**——它是 Mac 上的软件管家，后续安装工具会非常方便。

### 安装 Homebrew

打开终端（按 `Cmd+空格`，搜索"终端"打开），粘贴以下命令：

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

过程中可能会让你输入密码（你电脑的开机密码），输入时屏幕不会显示字符，正常输入后按回车即可。

### 验证 Homebrew

```bash
brew --version
```

如果看到版本号，说明安装成功。

## 常见问题

### Q: 安装 Node.js 时报错怎么办？
- **Mac 用户**：如果提示 `command not found: brew`，先安装 Homebrew（看第五步）
- **Windows 用户**：如果下载安装包时报"无法验证发布者"，点击"仍要运行"继续安装
- **权限问题**：Mac 用户如果提示权限不足，在命令前加 `sudo`（如 `sudo npm install ...`）

### Q: Claude Code 的 API Key 怎么收费？
- Claude Code 使用 Anthropic 的 API，按 Token 计费
- 日常使用（写脚本、分析内容）每月大概几十元人民币
- 建议首次充值 $10–$20（约 70–140 元）试用一个月
- DeepSeek 的价格更便宜，适合批量文字处理

### Q: 我是 Windows 用户，有什么不同？
本教程对 Windows 用户基本通用，只有以下差异：
- **终端**：用"命令提示符"或"PowerShell"代替"终端"
- **路径**：Windows 用 `C:\Users\xxx\` 代替 `~/`（Mac）
- **Homebrew**：Windows 不需要安装 Homebrew，用 `winget` 代替
- 其他步骤完全一致

### Q: 装完了一堆东西，好复杂，用不上怎么办？
先从最简单的开始：只用 Obsidian 写笔记 + DeepSeek 辅助。等习惯了再加 Claude Code。工具是为你服务的，不需要一次性全学会。

### Q: 我担心 AI 会泄露我的隐私？
- 所有 API 调用都是加密传输，Anthropic 和 DeepSeek 都有隐私保护政策
- 不要在 AI 对话中输入身份证号、银行卡号等敏感信息
- 商业秘密信息建议脱敏后再问 AI

### Q: 安装过程中网络很慢/连不上？
大概率是网络问题。先去 [[04-网络配置教程（梯子）]] 配置好网络，再回来继续。特别是安装 Node.js 和 API Key 配置需要联网。

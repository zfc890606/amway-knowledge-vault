# 🚀 四步部署流程

> 从零到一套 AI + 知识库 + 短视频系统，只需要这 4 步。
> 按顺序走，每步做完打勾。

──────────────────────────────────────

## Step 1 · 梯子（网络配置）

**目标**：让电脑能访问国际网络（装 Claude Code、下东西用）

**你要做的事**：

① 下载一个 Clash 客户端（找你推荐人拿链接，或搜"Clash 客户端下载"）
② 安装后导入"订阅链接"（你的服务商会给你）
③ 开启"系统代理"开关
④ 打开终端（Mac 叫"终端"，Windows 叫"命令提示符"），执行：
   ```
   export https_proxy=http://127.0.0.1:7890
   export http_proxy=http://127.0.0.1:7890
   ```
   （如果端口是 7897 就改成 7897）

⑤ 验证：浏览器打开 github.com，能打开 → ✅

> 以后每次新开终端，先执行第④步的命令。
> 如果觉得麻烦，把这三行加到 ~/.zshrc 里就一劳永逸了。

──────────────────────────────────────

## Step 2 · 装工具（Obsidian + Claude Code + DeepSeek + 充值）

**目标**：装好这 3 个工具

### ① 装 Obsidian
- 访问 obsidian.md → 下载 → 安装
- 新建一个库，名字随便取，存在电脑上
- ✅ 装好后能新建笔记

### ② 装 Node.js（Claude Code 需要它）
- 访问 nodejs.org → 下载 LTS 版本 → 安装
- ✅ 终端执行 node --version 能看到版本号

### ③ 装 Claude Code（通过 Obsidian 插件）
- 打开 Obsidian → 设置 → 第三方插件 → 社区插件市场
- 搜索 "Claudian"（插件 ID: realclaudian）→ 安装并启用
- ✅ 能看到 Claudian 面板

### ④ 充值 + 配置 API Key
- 访问 console.anthropic.com → 注册账号
- 进入 API Keys → 点 Create Key → 复制密钥
- 粘贴到 Obsidian → Claudian 设置 → API Key
- 充值 $10–$20（首次够用一个月）
- ✅ 在 Claudian 问一句"你好"能回复

### ⑤ 装 DeepSeek（国内 AI 备选，不需要梯子）
- 访问 chat.deepseek.com → 注册 → 进设置生成 API Key
- Obsidian → 社区插件市场 → 装 Copilot 插件
- 配置：API 地址填 `https://api.deepseek.com`，贴入 Key
- ✅ 问一句"帮我翻译"能回复

> 到这一步，你的工具就装全了。

──────────────────────────────────────

## Step 3 · 知识库搭建（给 Claude Code 的提示词）

**目标**：让 Claude Code 自动帮你建好知识库骨架

**你要做的事**：

① 在 Obsidian 中新建一个空白库（或者用刚才建的那个）
② 打开 Claudian 面板
③ 复制下面整段提示词，发给 Claude Code：

──────────────────────────────────────
> 请在我的 Obsidian 知识库根目录下，按照姜胡说的知识管理方法，创建完整的知识库目录结构。
>
> 核心方法论是五步闭环：读 → 想 → 写 → 做 → 盘
> 核心理念：知识不是用来囤的，是用来"运行"的。
>
> 请创建以下目录结构（用 [[wiki-link]] 格式做索引页）：
>
> ```
> 00-灵感库/
> ├── index.md          ← 入盒规则（写清楚什么该收、什么不该收）
> ├── 创作流程.md       ← 7刃筛选 + 挖矿8条 + 4K成卡
> └── 知识卡片/
>     └── 卡片模板.md   ← 4K格式模板
>
> 01-Projects/
> 02-Areas/
> ├── 01-营养解决方案/
> ├── 02-创业认知/
> ├── 03-短视频系统/
> └── 04-带团队/
>
> 03-Resources/
> ├── wiki/
> │   ├── _index.md
> │   ├── 概念/
> │   └── 来源/
> ├── 01-营养/
> ├── 02-创业/
> ├── 03-短视频/
> └── 04-带团队/
>
> 04-Archive/
> 05-Skills/
> 06-输出/
> Attachments/
> 日记/
> README.md
> ```
>
> 具体要求：
> 1. 每个目录下创建对应的 index.md 导航页
> 2. 入盒规则包含：✅收（引发思考的/可行动的/打破认知的）❌不收（纯资讯/琐事/将来可能有用）
> 3. 创作流程包含：7刃筛选（真实/深度/可迁移/我想/有行动/有变现/时机）× 挖矿8条（概念/因果/机制/数据/案例/反例/关联/行动）× 4K格式
> 4. 卡片模板包含：一句话要点 + 背后的机制 + 执行步骤 + 边界条件
> 5. README.md 作为知识库首页，能通过 wikilink 访问所有主要目录
> 6. 知识卡片的命名规则：KC-营养-* / KC-创业-* / KC-视频-* / KC-团队-* / KC-认知-*
──────────────────────────────────────

④ 等 AI 跑完，检查目录结构对不对
⑤ ✅ 目录建好，READNE.md 能点开

──────────────────────────────────────

## Step 4 · 短视频创作系统（给 Claude Code 的提示词）

**目标**：让 Claude Code 自动帮你搭好短视频创作系统

**你要做的事**：

① 打开 Claudian 面板
② 复制下面整段提示词，发给 Claude Code：

──────────────────────────────────────
> 请在我的知识库中搭建一套短视频创作系统。
>
> 我已经有了知识库骨架，现在需要短视频创作相关的所有文件。
>
> 请在 02-Areas/03-短视频系统/ 下创建以下文件：
>
> ### 1. 痛点排行榜.md
> 创建一个痛点排行榜表格，包含：排名、痛点描述、来源、频率（⭐评级）、适合的视频类型。
> 先写入至少 10 个安利大健康领域的痛点，例如：
> - 安利做不起来/总是失败
> - 新手第一步做什么
> - 长客会怎么推广/续约难
> - 互联网上怎么做安利
> - 安利是不是传销
> - 没有人脉怎么做安利
>
> ### 2. 内容策略.md
> 当前阶段的打法策略，包含：目标人群定位、内容方向、发布节奏（每周几条）、选题方向。
>
> ### 3. 关键词库.md
> 建立搜索关键词表，分组：安利类、创业类、大健康类。每个关键词标注优先级。
>
> ### 4. 脚本风格系统.md
> 包含：
> - 四大支柱：嘴替感 + 大奕式自信 + 姜胡说情绪 + 张医生式笃定
> - 12条铁律：开头直接扎心/每30秒一个"嗯？"/用具体场景代替抽象/三个观点递进/多用对话少讲道理/结尾笃定等
> - 六种视频结构：A痛点共鸣型/B方法教学型/C认知颠覆型/D营养科普型/E案例见证型/F答疑解惑型
> - 九大人性钩子：视觉/赚钱/恐惧/愤怒/八卦/猎奇/高价值/炫富/热点
> - 风格自检5关
>
> ### 5. 脚本模板.md
> 六种视频结构各给一个模板，包含：时长分配、情绪曲线、每段功能说明。
>
> ### 6. 数据追踪.md
> 数据追踪表模板：视频编号、发布时间、播放量、点赞、评论、转发、粉丝增长、备注。
>
> ### 7. 总控台仪表板（02-Areas/03-短视频系统/_index.md）
> 一个总览页面，汇总所有模块状态、当前 TOP 痛点、快捷指令、本周计划。
>
> 所有文件用 Obsidian wikilink 互相引用。内容用中文，口语化。
──────────────────────────────────────

③ 等 AI 跑完
④ 打开 02-Areas/03-短视频系统/ 检查文件都有了

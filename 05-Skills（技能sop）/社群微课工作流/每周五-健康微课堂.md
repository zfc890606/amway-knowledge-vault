## 📢 推广文案模板

每次周五微课，**第一条输出**就是推广文案，按此格式生成：

```
@所有人  
每周五，健康共修微课堂
《[主题]》[期号]

[一个震撼的事实/数据/金句 — 吸引眼球]
[一段价值描述 — 说明为什么值得听、有什么用、谁适合听]
[系列名称]
《[主题]》

今晚19：30点，社群邀请到本群主理人[姓名]，将会在本群为大家分享[一句话内容预告]

大家敬请期待[表情][表情][表情]
```

**填入规则：**
- 期号格式：`26-XX期`（延续上期数字）
- 事实/数据行：从参考文件或搜索结果中提取最有力的一个数字或结论
- 价值描述：2-3句话，说清谁需要听、听了有什么收获
- 主讲人：统一为"本群主理人张医生"
- 时间：统一为"今晚19：30点"
- 表情：统一用 `[鼓掌][鼓掌][鼓掌]`

**示例（糖尿病主题）：**

```
@所有人  
每周五，健康共修微课堂
《糖尿病逆转方案控糖饮食要点》26-11期

86%的糖尿病是可以通过科学管理完全可逆的
更新知识，科学管理，全营养逆转糖尿病的整体解决方案已经历经无数案例的验证，大部分的糖友实现了去药物，摆脱终生用药的宿命
糖尿病逆转系列课程
《糖尿病逆转之控糖饮食要点》 

今晚19：30点，社群邀请到本群主理人张医生，将会在本群为大家分享控糖饮食的要点

大家敬请期待[鼓掌][鼓掌][鼓掌]
```

## 四段式讲稿框架

### 第一部分：欢迎与社群主理人问候（约2分钟）

**目的：** 暖场、建立连接、预告价值

```
结构：
- 问候语："亲爱的社群伙伴们，大家晚上好！"
- 自我身份提醒："我是你的健康管理师XXX"
- 互动破冰："今天要聊的话题，跟每个人都有关系——[主题]"
- 价值预告："听完你会知道…"
- 互动引导："有类似情况的扣1"
```

### 第二部分：主题名词解释（约4分钟）

**目的：** 让小白听懂，建立专业信任

```
结构：
- 大白话解释核心概念（像给初中生讲课）
- 用比喻帮助理解（如：身体像一台车、肝像化工厂）
- 关键数据/事实支撑
- 一句话总结："所以XXX说白了就是…"
```

### 第三部分：知识分解 + 现实痛点（约6分钟）

**目的：** 拆知识 → 对号入座 → 制造"说的就是我"的共鸣

```
结构：
- 把主题拆成2-3个关键知识点
- 每个知识点对应一个真实生活场景
- 灵魂拷问："你是不是也…"
- 指出常见误区
- 过渡句："那这个问题到底怎么解决呢？"
```

### 第四部分：纽崔莱产品解决方案（约4分钟）

**目的：** 自然过渡到产品，提供可落地执行方案

```
结构：
- 解决思路总原则
- 推荐具体纽崔莱/汉本萃产品组合
  → 产品名 + 核心成分 + 为什么管用
- 使用建议（怎么吃、吃多久、搭配什么）
- 预期效果
- 行动号召："想了解具体方案的私信我"
```

## PPT大纲格式

每页包含「标题」「核心要点」「配图建议」，用 `---` 分隔页：

```markdown
第1页：封面
  标题：[微课主题]
  副标题：[一句话钩子]
  主讲人：[社群主理人]
  配图建议：[相关健康主题背景图]

---

第2页：目录
  ① 认识[主题概念]
  ② [主题]与我们的日常生活
  ③ 这些坑你踩过吗？
  ④ 营养解决方案

---

第3页：什么是[主题概念]
  核心要点：
  · 要点1（一句话）
  · 要点2（一句话）
  · 要点3（一句话）
  配图建议：[示意图/对比图/数据图表]

---

...（以此类推）

---

最后页：行动号召
  核心信息一句话总结
  联系方式/私信引导
```

## 语言风格指南

| ❌ 不要用（学术/销售腔） | ✅ 要用（社群口语化） |
|-----------------------|-------------------|
| 从分子生物学角度分析 | 简单来说就是… |
| 膳食营养补充剂干预方案 | 怎么吃能改善 |
| 线粒体功能障碍 | 细胞没力气干活了 |
| 临床研究表明 | 有研究说 |
| oxidative stress | 身体生锈了 |
| 建议您坚持使用 | 你坚持吃一段时间试试 |

## 完整工作流

```
你提供：
  ① 分享主题
  ② 图片素材（每个阶段的照片）
  ③ 如是邀请嘉宾，还需提供嘉宾称呼+分享要点

AI 同步输出两样：

  ① 📢 推广文案 → 你直接复制发社群预热
  ② 📝 六段式图文讲稿 → 每段文字标注配图位置

你拿到：
  [1] 📢 推广文案 → 提前发群
  [2] 📝 图文讲稿 → 周二晚按段发文字+配图
```

## 六段式图文讲稿框架

**整体格式：** 文字段落 + `[插入图片]` 标记，每段文字读完配一张图

### 第一段：接触营养的过程 + 改变图片

**目的：** 交代背景，引发"我也是"的共鸣

```
📝 文字写什么：
- 最初的状态（健康问题/困惑/不满意的生活状态）
- 怎么接触到营养/纽崔莱的（朋友介绍/偶然机会/自己搜索）
- 当时的心理活动（怀疑/好奇/试试看）
- 做的最初的改变是什么

🖼️ 配图位置：[建议插入：改变前的照片/刚接触营养时的记录]
```

---

### 第二段：做KOC的分享 + KOC改变图片

```
🖼️ 配图位置：[建议插入：做KOC期间的照片/改变对比图]
```

## 推广文案模板

**第二条输出**就是推广文案，按此格式生成：

```
@所有人 
《我的健康我做主》
我的健康故事专栏

今天晚上将会请到我们本群优秀[嘉宾称呼]，与大家分享[主题内容]

1 [要点1]
2 [要点2]
3 [要点3]
4 [要点4]
………………
大家今晚20：00本群，敬请期待[嘉宾称呼]的分享哦[鼓掌][鼓掌][鼓掌]
```

**填入规则：**
- 嘉宾称呼：如"红利老师"、"李老师"，如果自己分享则写"本群主理人[你的名字]"
- 主题内容：一句话概括分享主题
- 要点：4-6个提纲，从分享内容中提取最吸引人的亮点
- 时间：统一为"今晚20：00"
- 表情：统一用 `[鼓掌][鼓掌][鼓掌]`

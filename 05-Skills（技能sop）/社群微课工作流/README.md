# 🎙️ 社群微课工作流

> **一句话：** 你给我主题和参考文件，我给你讲稿 + PPT 大纲，你丢进 NotebookLM 生成 PPT。
>
> **适用场景：** 大健康社群日常运营，每周两场固定微课

## 🔄 通用工作流

```
你提供：
  ① 今天星期几 → 自动匹配对应工作流
  ② 主题内容
  ③ 参考文件（可选）

AI 执行：
  ① 读取指定/相关本地文件
  ② 必要时联网搜索补充
  ③ 按对应框架生成讲稿
  ④ 生成 PPT 大纲（可丢 NotebookLM）
```

# #第二板块深度解密：官方标准数据背后的“吸收率统计学本”

官方膳食指南在制定标准时，科学家们并不是按照“细胞用多少，你就吃多少”来定的。他们是用**逆向乘法**推导出来的：

**细胞的生理净需求（真正在体内消耗的）：** 一个成人每天通过消耗、汗液和消耗的皮屑，真正消耗并需要细胞补充的“净钙量”其实只有**$200 \sim 300\text{ mg}$左右**。

**科学家的公式（乘以吸收率）：** 科学家知道混合膳食和普通补充剂的平均吸收率仅约$30\%$。

$$\text{每日摄入标准} = \frac{\text{生理净流失量 } 300\text{ mg}}{\text{平均吸收率 } 30\%} = \mathbf{1000\text{ mg}}$$

也就是说，官方推荐你每天吃$1000\text{ mg}$（不管是依靠食物还是补充剂），他的预期就是**让你只吸收其中的$300\text{ mg}$左右**，剩余的$700\text{ mg}$本来就是计划好要随大便汇报去的。

因此，我们前面根据修复标准算的全天“倍立健+2至3片钙镁片=标称出来吞入”$880 \sim 1100\text{ mg}$”的物理模拟，已经完美匹配了官方的配额预算，不需要为了吸收率再去翻倍。

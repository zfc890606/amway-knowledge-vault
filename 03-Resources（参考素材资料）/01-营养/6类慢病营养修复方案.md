## ① 三高 / 心脑血管

### 目标人群画像
- 40-65岁，体检报告亮红灯
- 长期应酬/久坐/外卖族
- 知道自己"该注意"但不知道"具体怎么做"

### 通俗病因（短视频可讲版）
> **血管像水管，用久了里面会有水垢。血脂高就是水垢太多，血压高就是水压太大，血管壁变脆了就容易破——这就是心梗脑梗的剧本。**

### 营养修复逻辑

| 问题 | 修复逻辑 | 关键营养素 |
|------|---------|-----------|
| **血脂高（甘油三酯）** | 减少合成原料 + 促进代谢排出 | Omega-3（鱼油）、膳食纤维、茶多酚 |
| **胆固醇高** | 阻断肠道重吸收 + 促进肝脏代谢 | 植物甾醇（茶族）、Omega-3 |
| **血压高** | 放松血管平滑肌 + 保护血管内皮 | 钙、镁、维生素C、大蒜素 |
| **动脉硬化** | 抗氧化防血管内皮损伤 | 维生素C、维生素E、类胡萝卜素 |

### 可执行方案

| 餐次 | 原则 | 具体建议 |
|------|------|---------|
| **早餐** | 高蛋白启动代谢 | 蛋白粉饮 + 燕麦 + 少量坚果，不要包子油条 |
| **午餐** | 控油控盐 | 先吃蔬菜→再吃蛋白→最后主食（粗粮） |
| **晚餐** | 轻断食窗口 | 睡前4小时吃完，蔬菜为主+少量优质蛋白 |
| **每日补充** | 基础+专项 | 蛋白粉 + 维C + 鱼油 + 钙镁片 |

### 🎬 可用的短视频选题

| 选题 | 钩子 |
|------|------|
| 「血脂高不是吃油多，是你吃错了油」 | 颠覆"少吃油就能降血脂"的认知 |
| 「血压高的人，早上千万别做这件事」 | 制造紧迫感（起床太猛导致血压飙升） |
| 「血管堵了70%都没感觉？」 | 恐惧驱动，让人想了解怎么预防 |
| 「每天一杯这个，血管年轻10岁」 | 正面吸引，给具体方案 |

## ② 糖尿病 / 代谢综合征

### 目标人群画像
- 血糖偏高/确诊II型糖尿病
- 腹部肥胖（"苹果型身材"）
- 有家族史但还没到吃药阶段
- 想吃药又怕一辈子离不开药

### 通俗病因（短视频可讲版）
> **胰岛素是钥匙，细胞是锁。血糖高不是因为钥匙不够（胰岛素不足），而是锁生锈了（胰岛素抵抗）。II型糖尿病核心不是缺胰岛素，是细胞不听话了。**

### 营养修复逻辑

| 问题 | 修复逻辑 | 关键营养素 |
|------|---------|-----------|
| **胰岛素抵抗** | 减少糖摄入 + 增强细胞对胰岛素的敏感度 | 铬、镁、膳食纤维、大蒜素 |
| **血糖波动** | 减缓碳水吸收速度 + 控制餐后血糖 | 膳食纤维随餐、蛋白质前置 |
| **并发症预防** | 保护微血管 + 神经营养 | 维生素B族、维C、维E、类胡萝卜素、越橘 |
| **胰腺修复** | 抗氧化损伤 + 促进胰岛细胞恢复 | 越橘提取物、锌、抗氧化剂 |

### 可执行方案

| 餐次 | 原则 | 具体建议 |
|------|------|---------|
| **进食顺序** | 定时定量 | 先吃膳食纤维 → 蛋白质 → 最后碳水 |
| **碳水选择** | 低GI替换 | 白米饭换糙米/燕麦/藜麦，每餐主食不超一个拳头 |
| **加餐技巧** | 防低血糖 | 上午/下午各一小把坚果+蛋白饮 |
| **每日补充** | 基础+专项 | 蛋白粉 + 维B族 + 钙镁片 + 大蒜片 + 越橘 |

### ⚠️ 特别注意

- 需要配合医生指导，不可直接停药
- 营养补充是"辅助修复"，不是"替代治疗"
- 建议客户先做体检拿到数据，3个月后复查对比

### 🎬 可用的短视频选题

| 选题 | 钩子 |
|------|------|
| 「糖尿病不是糖吃多了，是这个器官出问题了」 | 认知颠覆，解释胰岛素抵抗 |
| 「血糖高的人，早餐千万别吃这3样」 | 给具体禁忌（粥/白面包/果汁） |
| 「II型糖尿病可以逆转吗？答案是——」 | 希望驱动，给正面案例 |
| 「1个动作测你有没有糖尿病风险」 | 互动型，引发自查→关注 |

## ③ 脂肪肝 / 肝胆问题

### 目标人群画像
- 30-55岁，体检报告"轻度脂肪肝"
- 不胖的人也有（瘦人脂肪肝）
- 医生说"没事，注意饮食"但没给具体方案
- 转氨酶偏高但不知道怎么办

### 通俗病因（短视频可讲版）
> **肝脏是身体的化工厂。你每天吃的碳水、酒精、药物都要经过它处理。活儿太多、原料不够，肝细胞被脂肪撑到"罢工"——这就是脂肪肝。问题不是油吃多了，是肝累了。**

### 营养修复逻辑

| 问题 | 修复逻辑 | 关键营养素 |
|------|---------|-----------|
| **脂肪堆积** | 促进脂肪代谢出肝 | 维生素B族、胆碱、卵磷脂 |
| **肝细胞损伤** | 修复受损肝细胞膜 | 奶蓟/水飞蓟（护肝片核心成分） |
| **解毒负担重** | 增强肝脏解毒通路 | 谷胱甘肽前体（蛋白粉+维C+硒） |
| **炎症反应** | 降低肝脏炎症水平 | Omega-3（鱼油）、维生素E |

### 可执行方案

| 餐次 | 原则 | 具体建议 |
|------|------|---------|
| **全天** | 拒绝精制糖+果糖 | 奶茶/水果汁/甜点是脂肪肝元凶 — 果糖直接走肝脏代谢 |
| **早餐** | 高蛋白促脂肪代谢 | 蛋白粉饮 + 全麦 + 少量好脂肪 |
| **午餐** | 护肝组合 | 深色蔬菜(十字花科) + 优质蛋白 + 好油(橄榄油) |
| **晚餐** | 轻断肝 | 睡前4小时禁食，给肝脏夜间修复时间 |
| **每日补充** | 护肝组合 | 蛋白粉 + B族 + 护肝片 + 鱼油 |

### 🎬 可用的短视频选题

| 选题 | 钩子 |
|------|------|
| 「不喝酒也会得脂肪肝？这3个习惯才是元凶」 | 打破瘦人/不喝酒的盲区 |
| 「脂肪肝不是吃油吃多了，是吃糖吃多了」 | 认知颠覆（果糖→肝脏代谢→脂肪堆积路径） |
| 「转氨酶高怎么办？不是降，是要修复」 | 科普方向转变 |
| 「每天一杯这个，帮你把肝里的脂肪代谢掉」 | 给具体解决方案 |

## ④ 肠胃 / 消化系统

### 目标人群画像
- 长期外卖族、熬夜族
- 胃胀/反酸/便秘/拉肚子交替出现
- 吃一点就胀、或者吃完不消化
- 做了胃镜也没查出大问题，但就是不舒服

### 通俗病因（短视频可讲版）
> **肠道是人的"第二大脑"。它的健康决定了你能吸收多少营养。肠道菌群像一个小花园——好菌是花，坏菌是杂草。你吃外卖、熬夜、乱吃药，就是在往花园里倒除草剂。**

### 营养修复逻辑

| 问题 | 修复逻辑 | 关键营养素 |
|------|---------|-----------|
| **肠道菌群失衡** | 补充好菌 + 喂养好菌 | 益生菌 + 膳食纤维（益生元） |
| **胃黏膜损伤** | 修复胃黏膜屏障 | 蛋白质、维生素A、锌 |
| **消化酶不足** | 提供消化辅酶 | B族维生素、木瓜酵素/菠萝酵素 |
| **肠道炎症** | 抗炎修复肠道黏膜 | Omega-3（鱼油）、谷氨酰胺（蛋白粉） |
| **便秘** | 增加粪便体积+润滑 | 膳食纤维 + 水 + 好油脂 |

### 可执行方案

| 阶段 | 方案 | 周期 |
|------|------|------|
| **1-2周（排毒期）** | 蛋白粉 + 膳食纤维 + 益生菌 + 大量水 | 先清理肠道环境 |
| **3-4周（修复期）** | + 葆蔚饮品（黏膜修复）+ 维B族 | 修复胃黏膜和肠壁 |
| **5-8周（巩固期）** | 逐步加回健康饮食多样性 | 建立新饮食习惯 |

### 🎬 可用的短视频选题

| 选题 | 钩子 |
|------|------|
| 「你的肚子不是饿了，是你的菌在饿」 | 把"饥饿感"重新归因到菌群 |
| 「长期吃外卖的人，肠道里到底长什么样」 | 具象化刺激，制造认知冲击 |
| 「便秘不是喝水不够，是这个机制出问题了」 | 颠覆"多喝水就能通"的常识 |
| 「胃镜做了没问题，为什么还是不舒服？」 | 共情大量"查不出问题但难受"的人群 |

## ⑤ 关节 / 骨骼

### 目标人群画像
- 45+岁，膝盖/腰椎开始出问题
- 久坐族（肩颈腰背僵硬）
- 运动爱好者（跑步膝、网球肘）
- 更年期女性（骨密度下降）

### 通俗病因（短视频可讲版）
> **关节像机器的轴承，用久了磨损是正常的。但很多人不是"磨损快"，而是"润滑油不够、原料不够"。骨头不是石头，它是有生命的组织——它需要原料来修自己。**

### 营养修复逻辑

| 问题 | 修复逻辑 | 关键营养素 |
|------|---------|-----------|
| **骨密度下降** | 增加钙吸收 + 促进骨胶原合成 | 钙 + 镁 + 维生素D + 维生素K |
| **关节软骨磨损** | 补充软骨基质原料 | 葡萄糖胺 + 软骨素 + 胶原蛋白 |
| **关节炎/炎症** | 降低关节炎症水平 | Omega-3（鱼油）+ 维C + 姜黄素 |
| **肌肉流失（肌少症）** | 维持肌肉量保护关节 | 优质蛋白质（蛋白粉）+ 维D |

### 可执行方案

| 项 | 建议 |
|----|------|
| **每日基础** | 钙镁片 + 维D + 蛋白粉（胶原蛋白合成原料） |
| **专项补充** | 葡萄糖胺 + 软骨素（50岁以上推荐） |
| **抗炎支持** | 鱼油（Omega-3）+ 维C |
| **饮食注意** | 减少高糖/精制碳水（炎症食物），增加深色蔬菜 |

### 🎬 可用的短视频选题

| 选题 | 钩子 |
|------|------|
| 「膝盖痛不是老了，是少了这个原材料」 | 归因到营养缺乏而非年龄 |
| 「喝骨头汤补钙？你被骗了多少年」 | 打假/常识颠覆 |
| 「肩颈僵硬不是坐太久，是你缺了这个」 | 重新归因 |
| 「50岁后最该吃的营养素，很多人不知道」 | 年龄精准人群+信息差 |

## ⑥ 免疫 / 亚健康

### 目标人群画像
- 容易感冒/疲劳/乏力
- 睡眠质量差、精力不足
- 总觉得"哪里都不舒服但查不出问题"
- 工作压力大、焦虑感强

### 通俗病因（短视频可讲版）
> **你的免疫系统是一支军队。每天睡得好、吃得好，军队就训练有素。熬夜、压力、垃圾食品就是在给军队断粮。身体不是突然生病的——是你把军队饿了好几年。**

### 营养修复逻辑

| 问题 | 修复逻辑 | 关键营养素 |
|------|---------|-----------|
| **免疫力低下** | 增加免疫细胞原料+活性 | 蛋白粉（免疫球蛋白基础）、维C、锌、大蒜素 |
| **疲劳/精力差** | 改善线粒体能量合成 | B族维生素、辅酶Q10、铁 |
| **睡眠质量差** | 调节神经递质 + 压力激素 | 钙、镁、B族、色氨酸（蛋白粉中） |
| **氧化应激** | 清除自由基 + 抗炎 | 维C、维E、类胡萝卜素、花青素 |

### 可执行方案

| 项 | 建议 |
|----|------|
| **每日基础** | 蛋白粉 + 维C + 倍利健（多维矿物质基础包） |
| **免疫升级** | + 大蒜片 + 类胡萝卜素（冬季/流感季加强） |
| **抗疲劳** | + 铁 + B族（精力差的人优先补充） |
| **助眠** | 睡前钙镁片 + 蛋白饮（色氨酸促眠） |

### 🎬 可用的短视频选题

| 选题 | 钩子 |
|------|------|
| 「为什么你总是很累？不是缺觉，是缺这个」 | 把疲劳归因到营养而非懒惰 |
| 「免疫力不是越强越好，是平衡才好」 | 高级认知，建立专业感 |
| 「3种食物悄悄摧毁你的免疫力」 | 给具体禁忌 |
| 「为什么你换季必感冒？问题出在这里」 | 季节性话题，精准人群 |

## 使用说明

1. **每条视频只讲一个病种的一个具体问题**（如"脂肪肝和碳水的关系"），不要面面俱到
2. **方案要具体到可执行**（"早餐吃这个不吃的那个"而非"注意饮食"）
3. **出方案前先科普病因**（先说"为什么会得"，再说"怎么修复"）
4. **结尾引导到私信**（"想知道适合你的具体方案，评论区告诉我你的体检数据"）
5. **免责话术固定**（"这个方案是营养辅助修复，不是替代治疗，用药请遵医嘱"）

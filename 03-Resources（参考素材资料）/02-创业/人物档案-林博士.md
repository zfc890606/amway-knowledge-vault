## 🏛️ 成冠八字方针（培养领导人体系）

林博士传承并落地的成冠核心框架，以**培养领导人**为方向：

```
          ╱ ① 态度（树根）╲
         ╱ ② 能力（树干）  ╲   ← 培养领导人
        ╱ ③ 架构（树枝）    ╲
       ╱ ④ 业绩（果实）      ╲
```

### ① 态度（树根）

态度是一切的基础，如同大树的树根：

- **「安利是我自己的事业」** — 创业者心态，不是打工者心态
- **感恩与跟随** — 对上手领导人的感恩、臣服与推崇
- **主动咨询沟通** — 主动和领导人/教练沟通，不被动等待
- **面对困难的积极态度** — 面对困难、挫折、挑战时的坚韧心态

### ② 能力（树干）

能力是支撑事业的树干：

- **迅速掌握20+1+1** — 掌握产品知识和事业讲解基本功
- **独立操办家庭聚会** — 能独立组织、主讲家庭聚会
- **主动担任中心工作组** — 在团队中承担组织工作
- **掌握ABC法则** — 借力系统运作的核心能力

### ③ 架构（树枝）

架构决定事业能长多大：

- **20×20×20模型**：建立20个优质顾客 + 20个活跃代表
- **每个点持续跟进沟通20次**，一定令事情发生
- **4×4架构**：培养自己先成为领导人 → 培养四个「五星领导人」
- **4×4共有20个种子领导人** — 安利是复制的事业

### ④ 业绩（果实）

业绩是以上三点的自然结果：

- **五大目标、四小目标** — 目标体系化管理
- **逆向推演**：定位自己为领导人 → 快速成长改变 → 培养为五星级种子领导人（稳健12%）→ 复制4个五星领导人 → 达成稳健21% → 4×4翡翠

## 🧠 核心思维体系（完整版）

### 一、心灵成长优先于财富追求

林老师最核心的关切：**先让心灵强大起来，再追求成功。**

> **"我很担心你们今天在心态没有成长之前就追求财富，急于追求成功的这种心态会让你的人生失去了平衡。"**
> **"我们整个的社会都处于非常浮躁的状态。"**
> **"无论潮流怎么变，不论环境怎么变，其实这个世界上一切的问题都可以归纳为心的问题。"**
> **"我们很多的人用眼睛去观察世界，用耳朵去聆听世界，用手脚去追赶世界，却没有用大脑去考虑自己的内心，整理自己的内心，修补自己的内心。结果他们在困顿的时候找不到方向，更找不到方法，他们在成功的时候却失去了自我。"**

**核心逻辑**：
```
心态成长 → 追求成功 → 完善人格 → 全方位的成功人生
   ↑                                      ↓
    ← ← ← ← ← 傅先生八字方针 ← ← ← ← ←
```

### 二、「穷且益坚，不改青云之志」— 志气思维

> **"老狄维士曾经说过一句话：我们都曾经经历过贫穷，但是我们只是选择了不再呆在那个地方，选择了追求成功而已。"**
> **"我想告诉你们，我也经历过贫穷，所以我对这句话是有深切体会的。"**
> **"穷且益坚，不改青云之志，这应该是每个人应有的一点志气。"**
> **"追求自己的成功中完善自己的人格，这才是傅先生说的——追求全方位的成功人生。"**

### 三、目标 × 勇气 = 尊重

> **"我不在乎你们每个月达到多少的目标，我在乎你每个月有没有去定下目标，并且全力以赴的勇气。"**
> **"从大里说，就是每个改变自己人生和命运的勇气。这才是值得我尊重的。"**

**目标不是数字，是勇气的证明。**

### 四、「定能生慧」— 心定思维

> **"心定方能定天下，只有专注才能做出有灵感。"**
> **"定能生慧，勤能补拙。"**

**核心认知**：任何时代都会有新情况新变化，但只要人心定下来，就一定能找到方法。

> **"给我们带来压力的往往是由于对自己未来的不确定性……在这个多变的世界里，如果我们无法把握周围的各种因素，至少我们可以做自己心灵的主人。不计得失，不算易难，不畏人言，由自己的良知去寻找自己的人生价值观，去选择自己前进的方向。"**
> **"不论周围多么黑暗，内心光明的人一定可以找到光明，一定可以给自己可以预计的未来。"**

### 五、「目标是我唯一的导师」— 目标思维

> **"安利是做不完的，只有做目标才会做得完。"**
> **"如果你做的事情和目标无关，那就是错的。"**
> **"做安利没有结果，做目标才有结果。"**
> **"只有目标，没有问题！凡事发生必有利于我。"**

### 六、「人才是事业的根本」— 人才思维

> **"我们不要被业绩遮蔽了，人才是事业的根本！"**
> **"重视人，重视亮点。"**

**四种人，三种工作**：

| 类型 | 工作方法 |
|------|---------|
| 不想做、不会做 | **做沟通** |
| 想做、不会做 | **做辅导** |
| 会做、不想做 | **做激励** |
| 想做又会做 | 放手发挥 |

### 七、「个人服从系统」— 系统思维

> **"个人不要跟团队作对，团队不要跟系统作对。"**
> **"生存靠个人，发展靠团队，保障靠系统。"**
> **"有系统的打败没有系统的。"**

**伦理层级**：个人 < 团队 < 中心 < 工委 < 系统 < 公司

### 八、「没有爱就没有教育」— 团队即家人

> **"嫌人很难成为领导人。嫌你就不说你，说你就不嫌你，又嫌你又说你，人际关系就搞砸了。"**
> **"领导人必须懂得去欣赏、去关爱一个人。钱多少不重要，心胸宽广，有容人之量。"**
> **"法刚好与这个相违背的：不是用技术去带他，而是用心去带他。"**

**团队凝聚力公式**：
```
爱 + 感恩 + 跟随 = 凝聚力
      ↓
用心带人 → 不以技术带人
```

> **"困顿的时候，回到中心来，回到团队来，回到你的领导人的身边来，回到阳光俱乐部来，不论你多么的失意，我们一如既往的支持着你。"**

### 九、「妨碍你成功的只有三个缺点」— 自我突破思维

> **"其实每个人妨碍你成功的东西不会超过三个。如果你把这三个妨碍你的问题解决了，你也就成功了。"**
> **"找出妨碍你成功的那2到3个缺点，你一定会成功。"**

**典型妨碍**：花在微信上的时间太多、不爱学习、信心动摇、懒惰、无知、坏毛病……

> **"这个世界上，能够阻挡你的，唯有你自己。"**
> **"如果你不能控制你自己，谁能让你成功？"**

### 十、「你行你不行都是对的」— 选择即命运

> **"你说你行你就迟早会行，你说你不行你就立马就不行——这取决于你自己。"**
> **"这本身不是个对错的问题，是个选择的问题。你想选择成与败，取决于你自己。"**

## 🏛️ 华宫精神（四根支柱）

林炜老师亲自撰写的华宫团队精神内核（2016年启航培训结业赠语）：

### ① 绝不逃避

> **"不逃避挑战，不逃避问题，不逃避犯错误的代价，不逃避纷争。"**

**核心认知**：任何一个人，如果愿意勇敢地面对生活，面对各种问题——工作难题、生活挑战、情感波折——如果你采取不逃避的办法，不找任何借口，坦诚而直接地面对，你将真正成为生命的主人。

> **"你不能够想要成长，却拒绝成长应有的挑战；你不能够想要成为卓越，却原谅自己，鼓励自己成为懦夫。"**
> **"是挑战与挫折，培养了一个真正的领导者、成功者！挑战和挫折，才是我们真正的导师。"**

### ② 勤奋

> **"这种勤奋，源于生命的热爱，和对人生每一刻的珍惜！这种勤奋更是对于自己的自律、高要求，和负责任。"**

**涵盖**：自律、负责、珍惜、热忱。

> **"我们从不抱任何侥幸心理，也从不奢望上天对我们额外照顾。如果上天真的是恩宠我们，那我们唯有更加珍惜和勤奋，来回馈上天的呵护！"**

### ③ 全局观

> **"你必须深刻明了，只有全局的进步，才能够保证你的成果。只有全局的美好，才能照顾你想要的美好。"**

> **"正是因为国家进步了，我们才顺便获得富有；正是因为企业成长了，才可以帮助员工们过得更好。"**

### ④ 真诚

> **"真诚，指的首先是对于自己的真诚，真实地面对自己。无论是自私的自己、渺小的自己、胆怯的自己、虚伪的自己，还是正面的自己。"**

> **"我们同意和理解人性并不完美，每一个人都可能有缺点。但是，既然我们自己也有，我们就愿意承认，也愿意去保护那些因为有缺点而害怕的人。"**

> **"一切的阴暗面，不会伤害我们，而是我们生活的一部分，被我们承认，并合理地安置。"**

### 2️⃣ 人生曲线三阶段

通过读书提升认知→改变观念→带动事业。推荐书目包括《贫穷的本质》《管道的故事》等。

| 0-2000分 | 服务顾客 | 养成长客会习惯，20个100分顾客 |

### 6️⃣ 四大基本功与三大领导力

**四大基本功**：
1. 沟通 — 随时打电话，每天20个以上
2. 跟进激励 — 从各种角度分析安利的价值
3. 辅导能力 — 复制就是辅导
4. 家庭聚会 — 基础作战单元

**三大领导力**：
```
沟通 → 跟进激励 → 辅导
   ↓        ↓         ↓
良好的沟通是跟进激励的前提
跟进激励是把安利对人生的影响阐述清楚
辅导就是复制的过程
```

> **"没有强势的领导力，要有很好的沟通能力。领导人最怕是手中没有真理，身边又没有群众。"**

### 7️⃣ 团队生态与伦理

**伦理层级**：个人 < 团队 < 中心 < 工委 < 系统 < 公司

**团队礼仪20条**（节选）：端茶递水拎包闭嘴、聆听时注视领导人、吃饭让座停筷、推崇领导人话语……

> **"推崇不是能力，是一种态度，是对领导人的欣赏。"**

**协作铁律**：
- 合作就要吃亏，跟随就要收起个性
- 所有意见不同当面说，不要背后说
- 两大崩盘原因：信心不足、缺乏全局观

### 8️⃣ 安利中国 · 爱国论述

林老师对"做安利是否爱国"的系统回答：

**爱国的定义**：爱国 = 爱人民 + 爱土地。恨别的国家不是爱国主义，是种族主义。

**安利中国的身份**：
- 安利中国遵守中国法律，向中国政府纳税
- 安利中国雇用中国员工，提供中国就业岗位
- 安利中国使用中国原材料

> **"我做了这么多年安利中国，给中国贡献多少税收我没计算过，但是我确定从来没有给美国贡献过一分钱税收。"**

**关于外汇**：安利中国在中国赚的钱能否寄回美国，由我国外汇管理局依法管理。

### 9️⃣ OKR工作法

林老师在团队群组中推行OKR方式：
> **"这个群对我们来说就相当于是一个目标看板。大家每天都把自己的进展说到这个群里边来，同时互相给对方提供意见和合作配合的机会。"**

# 安利奖励制度深度解析

> **底层逻辑**：安利不是卖产品赚钱，而是建立资产（管道）。奖励制度设计的核心是 —— **鼓励分享、奖励复制、保障世袭**。

---

## 前言：为什么是大健康？

### 大健康——后疫情时代的最大风口

2016年《"健康中国2030"规划纲要》提出：2030年我国大健康行业规模将达到**16万亿**。到2023年，这个数据已被刷新——中国健康管理协会会长在健博会产业高峰论坛上表示，到2030年，我国大健康产业规模预计达到**29.1万亿**。

> 大健康产业发展已经成为国民经济增长的新引擎。

### 六大驱动因素

| 驱动因素 | 具体表现 |
|:---|:---|
| **人口结构** | "二胎"人口增长，人口"老龄化"进程加剧，健康需求激增 |
| **技术环境** | 信息化技术发展，智能硬件应用成熟，推进产业升级 |
| **经济环境** | 居民可支配收入增加，医疗保健支出比重明显增加 |
| **社会环境** | 全球环境污染加剧，工作生存压力大，亚健康范围扩大 |
| **意识环境** | 健康意识增强，食品安全深入人心，运动健身、养生日益受关注 |
| **政策环境** | 国务院印发《"健康中国2030"规划纲要》，医疗改革核心推动 |

### 触目惊心的健康数据

| 数据 | 人数 | 占比 |
|:---|:---:|:---:|
| 脂肪肝患者 | **1.2亿** | — |

> **全国每天1.08万人确诊癌症，每分钟约8人确诊患癌！**

### 医院能解决慢病吗？

**临床医学的客观认识：**
- 对慢性非传染性疾病只能实施**症状控制**，无法让慢性病彻底好转
- 医生无法控制我们的生活方式，只能建议
- **如果把慢性病交给医生管理，那无疑是灾难**

**健康管理的客观认识：**
- 慢病来源于生活方式和三餐饮食
- 健康管理**一定是靠自己管理**

### 疾病谱的改变

```
从过去的：急性传染感染性疾病
    转向：慢性非传染性疾病
          ↑
    主要由生活方式、三餐饮食造成
```

### 安利在大健康中的角色

**产品基础：**
- 纽崔莱——全球唯一在自有有机农场种植、采收、加工植物原料的维生素及膳食补充剂品牌
- 两度参与国家"十三五"重点科研项目
- 三度独家支持DRIs（膳食营养素参考摄入量）标准修订
- 产品覆盖人类全生命周期，守护百万家庭健康

**系统支撑：**
- 医生营养师社群：来自全国临床医学、营养学、健康管理学、心理健康学等专业人才参与交流
- 双系统助力：华宫医生营养师（专业赋能、培训体系）+ 大A公司（产品支持、平台实力）

**三步走路径：**
```
第一步：让自己健康（成为健康幸福的呈现者）
```

## 二、三大奖金制度详解

### 1️⃣ 个人销售佣金（6%~27%含叠加）

这是最基础的奖金，根据个人及小组的净营业额按月结算：

| 净营业额 | 佣金比例 | 叠加产品券 |
|:---:|:---:|:---:|
| 2,500 | 3% | 6% |
| 7,500 | 6% | 6% |
| 12,500 | 9% | 6% |
| 25,000 | 12% | 6% |
| 50,000 | 15% | 6% |
| 87,500 | 18% | 6% |
| 125,000 | 21% | 6% |

> **安利是自己的事业** — 你的收入只取决于自己的业绩和开多少"分店"，不受上面推荐人影响。你是自己的老板。

---

### 2️⃣ 市场开拓奖金（市场推广服务费）

这是安利制度最核心的设计 —— **鼓励你帮助别人成功**。

**基本逻辑：**
```
你分享4个人（A/B/C/D），每人做6,000元
                ↓
你小组总业绩 = 6,000 × 5 = 30,000元 → 按18%计算
你的收入 = 30,000×18% - 4×(6,000×9%) = 5,400 - 2,160 = 3,240元
加上微购佣金和加码奖金（30%）：3,240 + 600 + 972 = 4,812元
而A/B/C/D每人收入 = 6,000×9% + 600 = 1,140元
```

**关键认知：**
- **你帮4个人各赚1,140元 → 你赚4,812元**
- 你付出的努力是4份，收入是他们的4倍 — **多劳多得**
- 新加入的人只要努力，收入可以超越领导人 — **可超越**
- 你不从任何伙伴身上赚钱，而是通过帮助大家做大市场获得回报 — **公平**

> **比尔盖茨评价**：安利的制度简直无懈可击！

---

### 3️⃣ 稳健市场奖金（6%领导奖）— 核心资产

这是安利制度**最有保障力**的奖金，决定了这份事业的"管道属性"。

**触发条件**：当你培养出一个独立市场，该市场月业绩稳定达到12.5万元。

**计算方法**：该市场业绩 × 6%（公司额外发放，不从该市场扣）
```
你培养出A市场达到12.5万
→ 你获得：125,000 × 6% = 7,500元/月（额外收入）

你培养出B市场也达到12.5万
→ 再获得：7,500元/月
```

**为什么说这是"资产"？**
```
资产 = 不做也有 + 可以继承
     ↓
即使你个人业绩为0，只要A市场稳定运营
你每月依然获得7,500元领导奖
并且这份收入可以世袭给子女
```

**年收入推演**：
| 市场数 | 月收入 | 年收入 |
|:---:|:---:|:---:|
| 1个DD市场 | 7,500 | 约9万 |
| 2个DD市场 | 15,000 | 约18万 |
| 3个DD市场 | 22,500 | 约27万 |
| 6个DD市场（钻石） | 45,000+ | 约60万+ |

## 四、五大核心价值（与奖金制度的关系）

### 价值一：安利是自己的事业

**制度体现**：你的收入完全取决于你的业绩和分店数量，不受推荐人影响。

```
你（收入2,880元） ← 推荐人赚不到你的钱
↓         ↑
A-D（收入180元/人） ← 你帮他们赚
```

> 你不是给安利打工，也不是给推荐人打工 — 你是**自己当老板**。

### 价值二：公平 · 多劳多得 · 可超越

| 维度 | 制度体现 |
|:---|:---|
| **公平** | 领导人不从你身上赚1分钱，不是"利用"人的关系 |
| **多劳多得** | 你付出4份努力，帮助4个人 → 收入是1个人的4倍 |
| **可超越** | 加入早晚不影响，谁努力谁收入高 → 后加入可超越领导人 |

### 价值三：助人自助

这是安利成功最核心的秘密。

```
不帮助别人的A/B/C/D → 收入各180元
帮助4个人的你       → 收入2,880元（是他们的16倍）
```

> **你帮的人越多，你赚得越多。** 这不是零和博弈，而是正和游戏。

### 价值四：保障与世袭

**传统工作的困境**：不干归零（人生抛物线，归零→重来）

**安利的保障**：
```
你
（个人业绩：0）
    ↓
A市场（≥12.5万）→ 你依然领取6%领导奖（7,500元/月）
    ↓
            可以世袭传承给后代
```

> ▸ 资产的特点：① 不做也有 ② 可以继承
> ▸ 这是安利区别于一切传统工作的核心 — **建立管道，而非提水桶**

### 价值五：安利是大事业（无上限）

```
你
 ├─ A → DD
 ├─ B → DD
 ├─ C → DD
 ├─ D → DD
 ├─ E → DD
 ├─ F → DD
 └─ ... 无上限
```

> 麦当劳开1家店没优势，开3万家就很厉害。安利一样 — **你可以无限复制**。

| 阶段 | 传统人生 | 安利人生 |
|:---|:---|:---|
| 23~25岁 | 参加工作，提升能力 | 刚做安利，积累经验 |

### 三种创业模式

| 模式 | 含义 | 风险 |
|:---|:---|:---|
| **造船出海** | 自己创建品牌/生意 | 高投入、高风险、需照顾一辈子 |
| **买船出海** | 加盟品牌店 | 资金门槛高 |
| **借船出海** | 上安利的船，不下船就行 | **低门槛、有系统支持** |

> **安利是"借船出海"** — 船已经造好了（公司+系统），你只需要上船，不下船。

## 六、一句话总结

```
安利的奖金制度 = 个人销售佣金（自己卖）
                + 市场开拓奖金（帮人卖）
                + 6%领导奖（拥有资产，不做也有）
                + 年度分红（股东权益）
                + 世袭保障（可传承）
```

**核心秘密就四个字：助人自助。**
你帮的人越多、越成功，你的收入越高、越稳定、越有保障。

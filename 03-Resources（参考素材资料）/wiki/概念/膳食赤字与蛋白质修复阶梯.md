# 膳食赤字与蛋白质修复阶梯

## 膳食赤字概念

即使摄入足量食物，细胞实际获得的营养也存在巨大**折损**。以蛋白质为例，2026年工业化农业背景下：

### 蛋白质折损账本（以60kg上班族为例）

| 折损环节 | 折损幅度 | 原因 |
|----------|---------|------|
| 原料折损 | -20% | 速生与水培模式 → 肉质高水分、低营养密度 |
| 高温烹饪折损 | -30% | 外卖重油煎炸 → 不可吸收的AGEs"死结" |
| 消化酶亏空折损 | -20% | 胃酸活性低下 → 大颗粒蛋白无法有效离子化 |
| **实际到账** | **~25g** | 对比战时修复所需72-96g，存在巨大赤字 |

## 蛋白质修复三大阶梯

引用石汉平教授的口腔营养补充（ONS）模型：对于慢病患者，**营养不是在"支持"疾病，而是在"治疗"疾病**。

| 阶梯 | 剂量 | 适用场景 |
|------|------|---------|
| **阶梯一：基础维持量** | 1.0-1.2 g/kg/d | 维持氮平衡，防止肌肉消瘦 |
| **阶梯二：组织修复量** | 1.2-1.5 g/kg/d | 慢性低度炎症、器官上皮更新的"战时爆发期" |
| **阶梯三：深度逆转量** | 1.5-2.0 g/kg/d | 重度消耗、恶病质及合成抵抗人群（黄金起始线1.6g） |

### 动态决策逻辑

1. **肾功能筛查**：CKD 3期及以上 → 严格限制0.6-0.8g/kg/d；正常 → 进入下一步
2. **人群特质评估**：
   - 普通慢病/素食 → 1.2-1.5g/kg/d（膳食 + 3勺蛋白粉）
   - 中重度初老/合成抵抗 → 1.6-2.0g/kg/d（膳食 + 5勺蛋白粉 + 基源欣活）

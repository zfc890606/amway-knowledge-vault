# 肝胆净化与护肝计划

## 肝脏的角色

肝脏是人体最大的消化腺，承担：
- **代谢功能**：糖、脂肪、蛋白质代谢中枢
- **解毒功能**：药物、酒精、代谢废物的处理工厂
- **合成功能**：蛋白质合成（白蛋白、凝血因子等）

胆囊储存肝脏分泌的胆汁，按需释放辅助脂肪消化。

## 肝损伤链条

现代生活方式下的典型肝损伤路径：

```
高脂高糖饮食 → 脂肪堆积（脂肪肝）
     ↓
氧化应激 → 炎症（NASH）
     ↓
肝细胞损伤 → 纤维化
     ↓
肝硬化 → 肝功能衰竭
```

## 护肝计划核心要素

### 营养干预靶点

| 营养素 | 作用机制 | 来源 |
|--------|---------|------|
| **优质蛋白质** | 肝细胞修复原料，提供谷胱甘肽合成前体 | 蛋白粉、基源欣活 |
| **卵磷脂** | 乳化脂肪，促进肝脏脂肪转运 | 卵磷脂胶囊 |
| **高纯Omega-3** | 抗炎、降低肝脏脂肪堆积 | 鱼油 |
| **B族维生素** | 能量代谢辅酶，支持肝脏解毒酶系统 | B族复合 |
| **抗氧化剂** | 对抗氧化应激，保护肝细胞膜 | 维生素C、E、硒 |

### 肝脏修复周期

肝脏细胞的更新周期为**150-300天（5-6个月）**：
- 脂肪肝的营养干预需**持续5-6个月**才能在B超或生化指标上看到明显恢复
- 承诺"3天治好脂肪肝"在生物学上是绝对不科学的

## 胆汁分泌与排毒

- 肝脏每天分泌约500-1000ml胆汁
- 胆汁不仅是消化液，也是肝脏排出脂溶性毒素和代谢废物的主要通道
- 胆管堵塞会影响胆汁流动，进而影响肝脏解毒效率

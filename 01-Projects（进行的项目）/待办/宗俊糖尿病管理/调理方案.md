# 🥗 宗俊 · 血糖调理方案

> 方案制定日期：2026-06-21 · 来源：飞书妙记

---

## 📋 情况分析

**诊断判断**：胰岛β细胞营养原料匮乏、矿物质、抗氧化营养素长期缺失，糖代谢酶合成不足，胰岛受损、胰岛素受体敏感度下降，C肽反应不敏感

**当前用药（截至2026-07）**：

| 类型 | 药物 | 剂量 | 时间 |
|:----:|:----|:----:|:----|
| 🔵 速效胰岛素 | 赖浦（餐前） | 4-6单位 | 每餐前 |
| 🔵 长效胰岛素 | 甘精（睡前） | 15-18单位 | 晚10-11点 |
| 💊 口服药 | 二甲双胍片 | 1粒（按需） | 无法打胰岛素时备用 |

**身体状况**：
- 体重：约55kg
- ⚠️ 车祸恢复中（左手扭伤、右手骨折、腰部受伤），运动受限
- 有动态血糖仪（雅培），指导老师可远程查看

## 🥣 三餐膳食 + 营养素方案

### 🌅 早餐（营养早餐）

| 项目 | 内容 |
|:----|:------|
| **营养饮** | 沛源蛋白2勺 + 姜黄粉0.5g + 肉桂粉0.5g + 亚麻籽油5ml → 摇摇杯冲温水 |
| **餐食** | 水煮蛋/蒸蛋2个 + 绿叶蔬菜一碟 |
| **营养素** | 钙镁片2片 + 小麦胚芽油5粒（随餐吞服） |
| **益生菌** | 健活益生菌1条（直接撕开吃） |
| **主食建议** | 尽量不吃主食，若吃选杂豆饭30g |

### ☀️ 午餐

| 项目 | 内容 |
|:----|:------|
| **餐食** | 瘦鱼肉/鸡胸肉 + 绿叶蔬菜 + 小份杂粮饭（50g左右） |
| **营养素** | 钙镁片2片 + 小麦胚芽油5粒（随餐吞服） |
| **备注** | 单位食堂吃 → 选绿叶菜 + 自带牛肉，避开红烧/勾芡/甜味菜；主食可用杂豆饭替代白米饭 |

### 🌆 晚餐

| 项目 | 内容 |
|:----|:------|
| **营养饮** | 沛源蛋白2勺 + 姜黄粉0.5g + 肉桂粉0.5g + 亚麻籽油5ml → 摇摇杯冲温水 |
| **餐食** | 豆腐/鲜虾 + 瓜茄类蔬菜，少量杂粮主食（能不吃可不吃） |
| **营养素** | 钙镁片2片 + 小麦胚芽油5粒（随餐吞服） |
| **益生菌** | 健活益生菌1条 |
| **睡前** | 长效胰岛素（甘精）+ 缓慢饮水 |

> ⚠️ **每日饮水量**：2500ml以上（至少2000ml）

## 🚫 饮食宜忌

### ✅ 推荐
绿叶菜、冬瓜、黄瓜、杂豆类、深海鱼、鸡蛋、豆制品、亚麻籽、原味坚果、青稞米、藜麦、荞麦、山药、脆藕

### ❌ 忌口
- 过量精制米面（白米饭、白面条）
- 含糖饮料、糕点蜜饯、油炸食品
- 高糖水果（荔枝、龙眼、芒果等）
- 浓汤（鱼汤等乳白色汤）
- 外卖/食堂中的隐形糖（红烧、勾芡、凉拌汁）

### 🍳 烹饪要点
- 干饭模式，加水偏少，不熬煮软烂（避免糊化）
- 杂粮饭/藜麦/杂豆 **煮熟后放凉** → 产生RS3抗性淀粉，GI进一步下降
- 干湿分离（不要汤拌饭），喝水提前或餐后喝
- 外出吃饭可备一碗白开水涮菜去油盐糖

## 🏃 运动建议

| 类型 | 频率 | 说明 |
|:----|:----|:------|
| 🧘 **八段锦** | 早晨有空做 | 居家或户外 |
| 🏃 **超慢跑** | 餐后1小时，40分钟 | 中餐餐后 |

| 🦶 **比目鱼肌训练** | 每天可做 | 踮脚，躺着/坐着都可做，不伤腰 |
| 🚶 **餐后散步** | 每次30分钟+ | 目前主要运动方式 |

> ⚠️ 目前车祸恢复期（腰部受伤），优先做 **比目鱼肌训练** 和 **超慢跑**，不做大幅度动作。等恢复后再加抗阻力训练。

## 📝 日常养护要点

1. **每餐七分饱**，主食粗细搭配（优先吃营养餐确保量）
2. **每日规律步行/运动**，杜绝熬夜、暴饮暴食
3. **营养素看耐受情况**，随时反馈调量
4. **定期监测**空腹、餐后血糖，动态调整饮食
5. **主食摄入量建议**：每日优质碳水总量120-170g，每餐30-50g
6. **进餐顺序**：先吃蔬菜 → 蛋白质 → 最后主食（如果需要）
7. **避免频繁加餐**，给胰岛休息时间
8. **低血糖处理**：低于3.9及时处理，但不要靠夜间加餐应对，应减长效胰岛素剂量

## 💡 核心控糖逻辑

```
少吃主食 → 减少胰岛素用量 → 打破"多吃-多打"循环
     ↓
营养修复胰岛（沛源蛋白+钙+Omega-3）
     ↓
增肌训练 → 提高胰岛素敏感性
     ↓
最终目标：减少药物依赖，实现血糖自主稳定
```

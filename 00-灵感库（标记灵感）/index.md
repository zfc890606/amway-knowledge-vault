## ✅ 入盒标准

以下内容可直接放进灵感库：

| 类型 | 例子 | 存放方式 |
|------|------|---------|
| 💡 **灵感闪念** | "也许可以做一个XX视频" | 直接写灵感笔记 |
| 📝 **知识卡片** | 已整理好的4K卡片 | → [[知识卡片/]] |
| 📄 **素材片段** | 一段有启发的引文/数据 | 标注来源后存入 |
| ❓ **待解问题** | "XX的底层机制是什么？" | 标记 #待挖矿 |
| 🎙️ **飞书妙记** | 讲课/学习/对话录音转写 | → [[飞书妙记/]]（自动同步） |

## ❌ 不入盒

- 完整的文章/书稿 → 去 [[../03-Resources]] 或 [[../Clippings]]
- 项目任务 → 去 [[../01-Projects]]
- 已完成输出 → 去 [[../5-输出]]

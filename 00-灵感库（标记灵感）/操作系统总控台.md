## ════════ 流水线A：知识加工 ════════

### 📥 输入 → 入盒 → 成卡 → 反哺

```
你随时丢内容过来（链接/文章/思考/问题/录音）
       │
       ▼
┌─ ① 入盒记录 ──────────────────────────────────┐
│  写入 00-灵感库/index.md 「最新灵感」+ 标注日期   │
└────────────────────┬──────────────────────────┘
                     ▼
┌─ ② 查重 ──────────────────────────────────────┐
│  ⚠️ 先看当下有没有相同或类似的？                   │
│  ├─ 有 → 跳过（不进卡片/洞见卡/选题）            │
│  └─ 没有 / 有必要加 → 继续往下                   │
└────────────────────┬──────────────────────────┘
                     ▼
┌─ ③ 判断分流 ───────────────────────────────────┐
│  ├─ 有深度认知 → 知识卡片（KC-板块-编号）         │
│  ├─ 有金句洞见 → 洞见卡（DJ-板块-编号）           │
│  ├─ 有选题价值 → 更新 选题库                     │
│  └─ 归入板块   → 更新对应 index.md              │
└────────────────────┬──────────────────────────┘
                     ▼
┌─ ④ 并行反哺 ───────────────────────────────────┐
│  路① 知识沉淀 → 归类4大板块 + 更新索引            │
│  路② 选题金句 → 选题库 + 洞见卡 → 触发流水线B    │
│  路③ 项目反哺 → 更新待办文件 → 流入流水线C       │
└────────────────────────────────────────────────┘
```

### 📂 三大原材料库 + 两个输出库（分清输入/输出）

```
流水线A自动产出（输入侧）：
│
├─ ① 查重 ────────────────────────────────────────────
│  ⚠️ 加之前先看：当下有没有相同或类似的？
│  ├─ 有 → 跳过，不再重复建
│  └─ 没有 / 有必要加 → 继续往下
│
├─ ② 产出 ────────────────────────────────────────────
│  ├── 知识卡片/   ← 📇 KC-板块-编号-主题 格式
│  │                 完整的认知框架（因果/原理/方法/场景）
│  ├── 洞见卡/     ← 💎 DJ-板块-编号-DJ-主题 格式
│  │                 一句话洞见+可用方法（写脚本时取金句）
│  │                 📌 已整合原金句素材库，现共 86 张
│  └── 选题库.md   ← 🔥 按痛点频率排榜的选题方向
│                    写脚本前来这里看拍什么
│

流水线B产出（输出侧，需你指令）：
├── 脚本库/         ← 🎬 你说"写脚本"后存入
└── 公众号草稿/     ← 📱 你说"改公众号"后存入
```

> **自动执行规则**：任何内容进来，不需问用户，直接跑完整流程。
> 飞书录音：用户说"拉飞书的录音"才触发，不后台自动跑。

## ════════ 流水线B：短视频创作 ════════

```
触发方式（三种）：
  ① 你说"帮我找一下选题" → 双通道扫痛点 → 更新选题库
  ② 你主动丢内容进来 → 流水线A成卡 → 有选题价值就入选题库
  ③ 你说"把第X号痛点拍成脚本" → 直接写，不调研不更新榜
                                          │
                                          ▼
┌─ Phase 1：选题（你说"找一下选题"才跑）───────────────┐
│  双通道：                                             │
│  通道A → 扫渠道表（抖/红/知/视频号/公/社群/喜马拉雅）  │
│  通道B → 你丢内容→A加工→评估选题价值                   │
│  → 更新痛点排行榜（加前查重） → 你过目                │
└──────────────────────┬─────────────────────────────┘
                       ▼
┌─ Phase 2：脚本（你说"拍脚本"才跑）───────────────────┐
│  读素材：选题库 + 洞见卡（取金句）+ 知识卡片（取观点） │
│                                                       │
│  选模板（看选题类型，我来定）：                         │
│  ├─ 案例见证型 → 案例模板                             │
│  ├─ 干货方法型 → 极简结构法 / 方法教学型              │
│  ├─ 心态情绪型 → 痛点共鸣型（情绪曲线拉满）            │
│  ├─ 认知颠覆型 → 认知颠覆模板                         │
│  ├─ 营养科普型 → 营养科普模板                         │
│  └─ 其他类型 → 我看着定                               │
│                                                       │
│  风格写作（嘴替×自信×情绪）→ 金句嵌入 → 质检8道关    │
│  → 存 脚本库/script-编号-主题.md                      │
└──────────────────────┬─────────────────────────────┘
                       ▼
┌─ Phase 3：公众号 ──────────────────────────────────┐
│  你说"把这条脚本改成公众号"                          │
│  → 口播体→阅读体改写 → 公众号排版 → 质检            │
│  → 存 公众号草稿/article-编号-主题.md               │
└──────────────────────────────────────────────────┘
```

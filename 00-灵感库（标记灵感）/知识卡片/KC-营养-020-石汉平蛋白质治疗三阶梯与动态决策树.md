# 《石汉平蛋白质治疗三阶梯与动态决策树》

> **一句话本质**：
> 慢病患者按日常RNI吃蛋白质，永远不够修复——必须用临床ONS（口服营养补充）阶梯，按"1.0维持→1.2-1.5修复→1.6-2.0深度逆转"三档加量，但前提是**先查肾功能**。

---

## 🧩 因果 / 原理

**它解决了什么问题？**
很多人认为"我每天吃肉就够了"——但2026年现代饮食经过三重折损（空心化-20%、AGEs毒化-30%、胃酸不足-20%），实际到账只有标称量的~50%。慢病修复需要1.2-1.5g/kg/天的蛋白质，普通人靠吃根本达不到。石汉平教授提出ONS阶梯模型，用**提纯蛋白粉绕过消化障碍**，精准补齐缺口。

**为什么有效？**

### 石汉平教授ONS三阶梯模型

石汉平教授，中华医学会肠外肠内营养学分会前任主任委员，主导提出：**慢性病患者的营养不是在"支持"疾病，而是在"治疗"疾病。**

| 阶梯 | 蛋白质量 (g/kg/d) | 60kg成人日需 | 适用人群 |
|------|-------------------|-------------|---------|
| **维持量** | 1.0~1.2 | 60~72g | 健康人群、慢病稳定期 |
| **修复量** | **1.2~1.5** | **72~90g** | **慢病/低度炎症/组织修复（核心阶梯）** |
| **深度逆转** | 1.6~2.0 | 96~120g | 重度消耗/恶病质/合成抵抗/少肌症 |

> **黄金起始线**：1.2~1.6 g/(kg·d) 是慢病修复的"细胞修复黄金交叉带"

### 为什么日常膳食蛋白不够？

**2026年现代饮食三重折损（60kg上班族典型日）：**

```
全天膳食标称55g蛋白质
    │
    ├─ 原料折损 -20%：速生肉高水分低密度
    ├─ 烹饪折损 -30%：外卖AGEs变性死结
    └─ 消化折损 -20%：胃酸不足、大颗粒流失
            │
            ▼
细胞实际到账仅 25~27g
```

**对比修复需求72~90g → 每日隐性赤字45~70g！**

这就是为什么必须用**提纯植物蛋白粉**（PDCAAS=1，零草酸截胡，免消化秒吸收）来强行补缺。

## 🛠 可执行方法

### 动态蛋白质决策树

```
[开始：慢病/亚健康营养干预]
    │
    ▼
检查肾功能与胃肠消化力
    │
    ├─ CKD 3期及以上 → 严格限制蛋白 0.6~0.8 g/(kg·d)
    │                       （限优质植物蛋白）
    │
    └─ 肾功能正常
            │
            ▼
        评估人群类别
            │
    ├─ 基础慢病/日常素食/中国胃
    │       └─ 1.2~1.5 g/(kg·d) → 膳食盘+3勺蛋白粉
    │
    └─ 中重度初老/顽固平台期/少肌症
            └─ 1.6~2.0 g/(kg·d) → 膳食盘+5~6勺蛋白粉+基源欣活
```

### 纽崔莱蛋白粉换算

| 蛋白粉量 | 纯蛋白 | 适用阶梯 |
|---------|-------|---------|
| 3平勺 | 24g | 基础维持（1.0g/kg） |
| 5平勺 | 40g | 修复量（1.2~1.5g/kg） |
| 6平勺 | 48g | 深度逆转（1.6~2.0g/kg） |

**实际配比示例（60kg慢病修复）**：
- 膳食实际到账：~25g
- 蛋白粉：5~6勺 = 40~48g
- 总到账：65~73g → 约1.2 g/(kg·d) **精准命中修复带**

### 服用要点
- **空腹/两餐间冲泡**：免消化秒吸收，绕过胃酸不足
- **16:00 ONS截击**：下午空腹一勺，拦截肌肉流失
- **特殊场景**：
  - 胃酸不足：冲得更稀，小口慢饮
  - 肠漏症：先从蛋白粉+益生菌开始修黏膜，2周后再加量

## 📍 场景与边界

### 适用场景
- 慢病方案设计（糖尿病/脂肪肝/高血压等）
- 客户说"我吃很多肉了不需要蛋白粉"
- 术后/产后/病后组织修复
- 少肌症/衰老性肌肉流失

### 边界条件
- **CKD 3期及以上**：严格限蛋白0.6~0.8g/kg，必须在医生监督下
- **婴幼儿/孕妇**：按特定指南，不可套用此阶梯
- **蛋白质过敏/苯丙酮尿症**：禁忌
- **单肾/肾移植者**：遵医嘱
- **不是说吃肉不好**：是在现代三重折损+消化能力下降的背景下，**提纯蛋白粉是最效率的补蛋白方式**
- **仍需膳食基础盘**：蛋白粉是补充不是替代，三餐仍需优质食物来源

### 常见失败原因
1. **用量不足**：每天1-2勺想修复慢病——掩耳盗铃
2. **忽略肾功能**：CKD患者冲高蛋白 → 加速肾衰竭
3. **与水冲不对**：必须用温水（<40°C），高温破坏蛋白质结构
4. **只用蛋白粉不管其他营养**：蛋白质需要B族和锌来代谢利用

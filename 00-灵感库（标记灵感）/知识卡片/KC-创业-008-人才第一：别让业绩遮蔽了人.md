# 《人才第一：别让业绩遮蔽了人》

> **一句话本质**：业绩是树上的果子，人才是树根。你天天盯着果子摘，树总有一天会枯。把精力放在浇灌树根上——培养人，果子自然来。

---

## 🧩 因果 / 原理

**它解决了什么问题？**
很多团队长做久了就变成"业绩导向"——每天看报表、盯分数、催出单。但你越催业绩，人越烦；人越烦，业绩越差。这是个恶性循环。

**为什么有效？**
- **林博士核心观点**："我们不要被业绩遮蔽了，人才是事业的根本！"
- **付后坚核心观点**："推荐的人就要帮助他，帮助他最好的方法就是帮他赚钱；留人最好的方法，也是帮他赚到钱。"
- **底层逻辑**：安利是复制的事业。你把一个人培养好了，他帮你复制出更多的人。业绩只是培养人的自然结果

**关键认知转换**：

```
❌ 盯着业绩 → 催人出分 → 人累心累 → 业绩更差

✅ 盯着人才 → 培养能力 → 能力变业绩 → 人才再复制 → 业绩持续
```

**林老师的两个检测工具**：

**工具一：四种人**（详见 [[KC-创业-007-四种人三种工作：团队管理的万能框架|KC-创业-007]]）
- 不想做不会做 → 做沟通
- 想做不会做 → **重点培养**
- 会做不想做 → 做激励
- 想做又会做 → 给空间

**工具二：三个问题**（带团队时的灵魂拷问）
1. 你想小伙伴的成功，还是小伙伴的业绩？
2. 你为他的成功做了什么？
3. 你的团队有没有因为你而变得更好？

---

## 🛠 可执行方法

### 最小行动（今天就能做）
找出一位你最近在催业绩的伙伴，今天不谈业绩——只问他："你最近有什么需要我帮忙的吗？"

### 完整执行步骤

**第一步：做领导人成功的三个动作**
1. **重视人，重视亮点**：每个人身上都有亮点，先找到它、夸它
2. **帮助人赚到钱**：留人最好的方法不是画饼，是让他赚到第一笔钱
3. **辅导人成长**：从"帮他做"变成"教他做"，从"教他做"变成"看他做"

**第二步：付后坚"人才四筛法"**
找种子领导人，用四个标准筛选：
1. **人品好** — 自私的人做不大，先看人品
2. **有梦想** — 有改变现状的渴望
3. **受教度** — 听懂了做，听不懂做了就知道了
4. **条件** — 有一定的基础

**第三步：避免"最大的浪费"**
> "安利最大的浪费是找错人。距离不是最大的成本，最大的成本是培养错了人。"

用人之前先筛选——不可培养的（人品差/不诚实/极度自私）不要浪费时间。

**第四步：林博士的提醒**
> "不要只想着小伙伴的业绩。如果你想小伙伴的成功，你为他的成功做了什么？你才会成功。"

## 📍 场景与边界

### 适用场景
- ✅ 每天盯着团队业绩但越盯越差
- ✅ 感觉自己成了"业绩警察"而不是领导人
- ✅ 团队有人才但不知道怎么培养
- ✅ 留不住人——培养了就跑了

### 边界条件
- ⚠️ "重视人才"不等于不管业绩——两者是因果关系不是二选一
- ⚠️ 培养人要花时间，短期内可能"慢"——但长期是"快"
- ⚠️ 不是所有人都值得培养——先筛选再投入

### 常见失败原因
- 只关注业绩不关注人——人跑了业绩归零
- 只关注人不关注业绩——团队养懒汉
- 培养错人——在不对的人身上花了太多时间

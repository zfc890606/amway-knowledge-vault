# 膳食赤字与蛋白质修复三阶梯

> **一句话本质**：
> 即使你"吃够了"，细胞实际到账的营养也存在巨大折损——农业空心化(-20%)、高温烹饪变性(-30%)、胃肠吸收障碍(-20%)三重剥削后，标称摄入仅50-70%真正到达细胞。慢病患者的蛋白质需求远高于常人，必须用**口腔营养补充（ONS）治疗量**而非RDA维持量才能满足修复需求。

---

## 🧩 因果 / 原理

### 蛋白质的"三重剥削"账本（以60kg上班族为例）

即使每天摄入足量食物，细胞实际获得的蛋白质也存在巨大折损：

| 折损环节 | 折损幅度 | 原因 |
|----------|---------|------|
| **① 原料折损** | -20% | 速生与水培模式 → 肉质高水分、低营养密度；土壤贫瘠→作物蛋白质缩水 |
| **② 高温烹饪折损** | -30% | 外卖重油煎炸 → 蛋白质变性，产生不可吸收的AGEs"死结" |
| **③ 消化酶亏空折损** | -20% | 胃酸活性低下（现代人通病） → 大颗粒蛋白无法有效离子化分解 |
| **实际到账** | **~25g**（仅为标称55g的45%） | 对比战时修复所需72-96g，存在巨大赤字 |

### 石汉平教授 ONS（口腔营养补充）三阶梯模型

> **对于慢病患者，营养不是在"支持"疾病，而是在"治疗"疾病。**
> ——石汉平教授

| 阶梯 | 剂量 | 适用场景 |
|------|------|---------|
| **阶梯一：基础维持量** | 1.0-1.2 g/kg/d | 维持氮平衡，防止肌肉消瘦——日常保健，非生病状态 |
| **阶梯二：组织修复量** | 1.2-1.5 g/kg/d | 慢性低度炎症、器官上皮更新的"战时爆发期" |
| **阶梯三：深度逆转量** | 1.5-2.0 g/kg/d | 重度消耗、恶病质及合成抵抗人群（黄金起始线1.6g） |

### 动态决策逻辑

```
1. 肾功能筛查
   ├─ CKD 3期及以上 → 严格限制0.6-0.8g/kg/d
   └─ 正常 → 进入下一步

2. 人群特质评估
   ├─ 普通慢病/素食 → 1.2-1.5g/kg/d（膳食 + 3勺蛋白粉）
   └─ 中重度初老/合成抵抗 → 1.6-2.0g/kg/d（膳食 + 5勺蛋白粉 + 基源欣活）
```

### 为什么RDA维持量不够？

RDA（推荐膳食供给量）0.8g/kg/d 是针对**健康人群**的"不生病"标准，而非"修复"标准。慢病状态下：
- 炎症消耗增加1.5-2倍蛋白质需求
- 肠道吸收效率下降30-50%
- 肝脏解毒需要大量氨基酸合成谷胱甘肽
- 组织修复需要额外蛋白质原材料

**结论**：慢病患者的蛋白质需求是健康人的**2倍**，必须用ONS治疗量补足。

## 🛠 可执行方法

### 最小行动（今天就能做）

早餐增加一杯蛋白粉饮——即使早餐吃过了，大多数人早餐的蛋白质摄入远低于修复需要。

### 完整执行步骤

**阶段一：评估（第1周）**
1. 计算体重（kg）× 阶梯系数 → 确定日蛋白质需求量
2. 记录3天膳食 → 计算每日蛋白质实际摄入量
3. 减去三重剥削折损（× 50-70%）→ 得到真实到账量
4. 计算差距 = 修复需求量 - 真实到账量

**阶段二：补足（持续）**
5. 就餐膳食提高蛋白质量（鸡蛋、鱼、瘦肉、豆制品）
6. 使用蛋白粉补足差额（每勺蛋白粉≈8g纯蛋白，吸收率95%）
7. 分次补充（每次10-20g，分3-5次，全天均匀摄入）

**阶段三：优化（必要时）**
8. 胃酸不足者随餐配维C（帮助蛋白质离子化吸收）
9. 肠漏者先2周修复肠道再提高蛋白量
10. 高龄+合成抵抗者考虑基源欣活（含多酚，辅助肌肉蛋白合成）

### 实用换算

| 体重 | 阶梯二（修复量） | 阶梯三（逆转量） |
|------|----------------|----------------|
| 50kg | 60-75g/天 | 75-100g/天 |
| 60kg | 72-90g/天 | 90-120g/天 |
| 70kg | 84-105g/天 | 105-140g/天 |
| 80kg | 96-120g/天 | 120-160g/天 |

## 📍 场景与边界

### 适用场景
- 慢病患者的营养方案设计（蛋白质核心）
- 向客户解释"我吃得很好了为什么要补蛋白粉"
- 术后/病后修复的营养支持
- 老年人肌少症/肌肉流失的营养干预
- 肿瘤患者的营养支持（恶病质预防）

### 边界条件
- **什么时候不适用？**
  - CKD 3期及以上（肾功能不全）——需严格限蛋白
  - 苯丙酮尿症（PKU）患者
  - 肝性脑病急性发作期
- **重要前提**：
  - 肾功能需正常或接近正常（测血肌酐+eGFR）
  - 蛋白质补足需要配合总热量达标——单纯高蛋白低热量不行
- **常见失败原因？**
  - 只算膳食纸上量，忽略三重剥削
  - 一次补太多（蛋白质单次吸收上限约30g），分次补充效果更好
  - 忽略碳水配合——蛋白质的利用需要胰岛素信号配合，低碳水≠零碳水

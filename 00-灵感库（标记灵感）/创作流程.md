# 创作流程：从灵感到成品

> 参考：灵感炼金炉（7刃筛选 + 挖矿8条）→ 知识卡片 KC（4K制式）→ 输出

---

## 第一阶段：收集（入盒）

**入口**：任何来源的灵感碎片

**动作**：标记"这是灵感" → 放进 [[知识库/00-灵感库（标记灵感）/index|00-灵感库]]

**产出**：未经加工的原始素材

---

## 第二阶段：炼金（7刃筛选）

从灵感库中取出一条素材，用以下 7 个问题筛选：

| # | 刀刃问题 | 目的 |
|---|---------|------|
| ① | 这真的是对的吗？ | 质疑第一直觉 |
| ② | 这个认知能帮我解决什么具体问题？ | 锁定实用性 |
| ③ | 有没有反例或边界条件？ | 寻找局限性 |
| ④ | 如何用一句话说清楚？ | 提炼核心 |
| ⑤ | 这个观点和其他知识冲突吗？ | 交叉验证 |
| ⑥ | 谁需要知道这个？ | 确定受众 |
| ⑦ | 现在能做什么？ | 导向行动 |

**通过 5/7 刃** → 进入挖矿阶段
**通过 < 5 刃** → 暂存，定期复盘

---

## 第三阶段：挖矿（8条路径）

对通过筛选的素材，从以下 8 个角度深挖：

| # | 矿脉 | 挖掘方向 |
|---|------|---------|
| ① | **溯源** | 这个观点最早来自哪里？原始出处可靠吗？ |
| ② | **解构** | 拆成最小单位：前提→推理→结论 |
| ③ | **类比** | 能用什么生活化的比喻讲清楚？ |
| ④ | **数据** | 有没有支撑的数据？数据来源是什么？ |
| ⑤ | **案例** | 实际发生的例子？成功/失败的？ |
| ⑥ | **反方** | 反对观点是什么？他们的论据是什么？ |
| ⑦ | **迁移** | 这个逻辑还能用在哪些领域？ |
| ⑧ | **行动** | 知道这个后，我应该做什么改变？ |

**产出**：挖矿笔记（围绕以上8条的思考记录）

---

## 第四阶段：成卡（4K制式）

将挖矿笔记整理为知识卡片，存放在 [[知识卡片/]]：

### 4K 卡片结构

```
┌──────────────────────────────┐
│ 🔑 Key Insight（核心洞察）    │ ← 一句话说透
├──────────────────────────────┤
│ 📖 Knowledge（知识体系）       │ ← 概念/原理/数据
├──────────────────────────────┤
│ 🔗 Context（上下文/边界）      │ ← 适用条件/局限性
├──────────────────────────────┤
│ ⚡ Kick（行动指引）            │ ← 知道了怎么做
└──────────────────────────────┘
```

**命名规范**：`KC-[领域]-[序号]-[标题].md`

示例：`KC-健康管理-001-慢病的细胞级根源.md`

---

## 第五阶段：输出

卡片积累充分后，组装为：

| 输出类型 | 用途 | 存放位置 |
|---------|------|---------|
| 🎤 **口播逐字稿** | 短视频出镜 | [[../01-Projects]] |
| 📄 **长文** | 公众号/文章 | [[../5-输出]] |
| 🎬 **短内容** | 图文/切片 | [[../01-Projects]] |
